**提示 1：** 考虑下，哪些 $b_i$ 是相关的？

**提示 2：** 什么情况下会发生相等？

先考虑下哪些 $b_i$ 是相关的。我们把可能相等的连起来，发现是：

$b_1\to a_2\to b_3\to a_4\to b_5\to\dots\to b_n$

每个 $b_i$ 要么等于前一个元素，要么等于后一个元素。

我们用 $0$ 表示等于后一个元素，用 $1$ 表示等于前一个元素，则 $b_1$ 对应的标记是 $0$ ， $b_n$ 对应的标记是 $1$ 。

我们只需找到标记中出现 $01$ 的位置，对应的 $b_i$ 就相等了。

而数组开头是 $0$ 结尾是 $1$ ，我们只需不断二分，保证区间左端点是 $0$ 右端点是 $1$ ，在区间长度变为 $2$ 的时候就找到了 $01$ 。（下面代码的逻辑跟这个略有不同）

查询次数是 $\mathcal{O}(\log n)$ 次的。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    def check(x):
        print(2, 2 * x - 1, flush=True)
        v1 = II()
        print(1, 2 * x, flush=True)
        v2 = II()
        return v1 == v2
    
    t = II()
    
    for _ in range(t):
        n = II()
        v = (n + 1) // 2
        
        l, r = 1, v - 1
        while l <= r:
            mid = (l + r) // 2
            
            if check(mid): l = mid + 1
            else: r = mid - 1
        
        print(3, 2 * r - 1, 2 * r + 1, flush=True)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

    int t;
    cin >> t;

    while (t --) {
        int n;
        cin >> n;

        int v = (n + 1) >> 1;
        int l = 1, r = v - 1;

        while (l <= r) {
            int mid = (l + r) >> 1;
            
            int v1, v2;
            cout << 2 << ' ' << 2 * mid - 1 << endl;
            cin >> v1;
            cout << 1 << ' ' << 2 * mid << endl;
            cin >> v2;

            if (v1 == v2) l = mid + 1;
            else r = mid - 1;
        }

        cout << 3 << ' ' << 2 * r - 1 << ' ' << 2 * r + 1 << endl;
    }

	return 0;
}
```
