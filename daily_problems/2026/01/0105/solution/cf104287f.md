**提示 2：** $d\bmod c$ 可能是哪些数值？

先求 $a,b$ 的最小公倍数 $V$，则 $d$ 一定是这个数的倍数，于是我们就可以得到 $kV$ 。

而一个数的倍数关于另一个数取模的结果，可能是如何的呢？

首先，根据裴蜀定理，可以找到一个系数，使得 $kV\bmod c=\mathrm{gcd}(V,c)$ ，而无论如何取模结果显然都是 $\mathrm{gcd}(V,c)$ 的倍数（ $x$ 的倍数关于另一个 $x$ 的倍数取模，结果是 $x$ 的倍数，否则有矛盾），所以可取的数值就是 $\mathrm{gcd}(V,c)$ 的若干倍。

因此最大结果就是 $(c/\mathrm{gcd}(V,c)-1)\times\mathrm{gcd}(V,c)=c-\mathrm{gcd}(V,c)$ 。

时间复杂度为 $\mathcal{O}(\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        a, b, c = MII()
        outs.append(c - math.gcd(c, math.lcm(a, b)))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int a, b, c;
		cin >> a >> b >> c;
		cout << c - gcd((long long)c, 1ll * a * b / gcd(a, b)) << '\n';
	}

	return 0;
}
```
