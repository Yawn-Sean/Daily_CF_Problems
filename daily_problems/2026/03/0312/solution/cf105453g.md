**提示 1：** 考虑以全亏为初始状态，基于此的最优结果如何？

先把所有成本都扣掉。这样完成一个航班的收益就是 $b-c$ 了。

于是就是选子集，最大化 $b-c$ 的和。

假设前 $i$ 项的最优结果是 $dp[i]$ ，则有两种可能：

- 第 $i$ 项不要，也就是 $dp[i-1]$ 。

- 要第 $i$ 项，此时可以发现前一轮被选的最大下标 $pt$ ，则是 $dp[pt]+b-c$ 。

因为随着 $i$ 增大， $pt$ 是单调递增的，所以整体只需要使用前缀最大值和单指针维护 $pt$ ，就能快速线性得到答案。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, T = MII()
    ts = []
    bs = []
    
    ans = 0
    
    for _ in range(n):
        t, b, c = MII()
        ans += c
        ts.append(t)
        bs.append(b - c)
    
    dp = [0] * (n + 1)
    pt = 0
    
    for i in range(n):
        while ts[i] - ts[pt] >= T and pt + 1 <= i:
            pt += 1
        dp[i + 1] = fmax(dp[i], dp[pt] + bs[i])
    
    print(dp[n] + ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, T;
	cin >> n >> T;

	vector<int> ts, bs;

	int ans = 0;
	for (int i = 0; i < n; i ++) {
		int t, b, c;
		cin >> t >> b >> c;
		ans += c;
		ts.emplace_back(t);
		bs.emplace_back(b - c);
	}

	vector<int> dp(n + 1, 0);
	int pt = 0;

	for (int i = 0; i < n; i ++) {
		while (ts[i] - ts[pt] >= T && pt + 1 <= i) pt ++;
		dp[i + 1] = max(dp[i], dp[pt] + bs[i]);
	}

	cout << dp[n] + ans;

	return 0;
}
```
