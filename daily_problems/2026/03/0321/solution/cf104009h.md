**提示 1：** $K$ 从 $x$ 变成 $x+1$ 应该怎么维护？

**提示 2：** 如何快速执行 $K$ 次？

如果我们有多一轮机会应该如何决策呢？

设 $x$ 轮时，我们的期望是 $V$ ，则我们当前轮次如果得到的结果大于 $V$ 我们就应该直接见好就收，否则我们应该取 $V$ 。

因此可以得到相邻两个轮次的递推关系，这样直接模拟 $K$ 轮，而对应的 $V$ 单调递增，所以可以直接用一个指针单向移动，找到对应的 $V$ 的分界位置，时间复杂度为 $\mathcal{O}(n+k)$ 。

但还可以解决 $K$ 更大的问题。因为如果我们的分界线保持不变，则相邻两项的递推关系保持不变，都是 $a_{i+1}=ka_i+b$ 的形式，所以就可以快速执行多次递推。

执行几次呢？我们可以二分找到分界线到达下一个区间的次数，这样我们最多进行 $n$ 次二分，就可以完整找到 $k$ 次是怎么分配的了。具体可见后面的代码。

时间复杂度为 $\mathcal{O}(n\log^2 k)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, k = MII()
    
    vals = []
    weights = []
    
    for _ in range(n):
        v, w = MII()
        vals.append(v)
        weights.append(w)
    
    st_range = sorted(range(n), key=lambda x: vals[x])
    
    acc_total = [0]
    acc_weights = [0]
    
    for i in st_range:
        v = vals[i]
        w = weights[i]
        
        acc_total.append(acc_total[-1] + v * w)
        acc_weights.append(acc_weights[-1] + w)
    
    ans = acc_total[n] / acc_weights[n]
    
    while k:
        l, r = 0, n - 1
        while l <= r:
            mid = (l + r) // 2
            if vals[st_range[mid]] > ans: r = mid - 1
            else: l = mid + 1
        
        cur = l
        prob = acc_weights[cur] / acc_weights[n]
        to_add = (acc_total[n] - acc_total[cur]) / acc_weights[n]
        
        v = to_add / (1 - prob)
        
        l, r = 1, k
        while l <= r:
            mid = (l + r) // 2
            nans = pow(prob, mid) * (ans - v) + v
            if nans > vals[st_range[cur]]: r = mid - 1
            else: l = mid + 1
        
        cur = fmin(l, k)
        k -= cur
        ans = pow(prob, cur) * (ans - v) + v
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);


	int n, k;
	cin >> n >> k;

	vector<pair<int, int>> lottery(n);
	for (auto &[v, w]: lottery) cin >> v >> w;

	sort(lottery.begin(), lottery.end());

	vector<long long> acc_total = {0}, acc_weights = {0};
	for (auto &[v, w]: lottery) {
		acc_total.emplace_back(acc_total.back() + 1ll * v * w);
		acc_weights.emplace_back(acc_weights.back() + w);
	}

	long double ans = (long double)acc_total.back() / acc_weights.back();

	while (k) {
		int l, r;

		l = 0, r = n - 1;
		while (l <= r) {
			int mid = (l + r) / 2;
			if (lottery[mid].first > ans) r = mid - 1;
			else l = mid + 1;
		}

		int cur = l;
		long double prob = (long double)acc_weights[cur] / acc_weights[n];
		long double to_add = (long double)(acc_total[n] - acc_total[cur]) / acc_weights[n];
		long double v = to_add / (1 - prob);

		l = 1, r = k;
		while (l <= r) {
			int mid = (l + r) / 2;
			long double nans = powl(prob, mid) * (ans - v) + v;
			if (nans > lottery[cur].first) r = mid - 1;
			else l = mid + 1;
		}

		cur = min(l, k);
		k -= cur;
		ans = powl(prob, cur) * (ans - v) + v;
	}

	cout << fixed << setprecision(15) << ans;

	return 0;
}
```
