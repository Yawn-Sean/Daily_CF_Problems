**提示 1：** 从头开始每一段越长越好。

**提示 2：** 如何判断新增边是否产生矛盾？如何快速恢复成出厂设置？

首先，每一段都是连续的，所以从开头一段开始，每一段都是取越长越好，这样就能给后面的段落留空间。

而判断是否二分图，同时新增一条边能快速维护，可以使用一个大小为 $2n$ 的并查集，用 $u$ 和 $v+n$ 相连 $v$ 和 $u+n$ 相连表示两者不同色，这样一旦 $u$ 和 $u+n$ 连通就产生矛盾了。 这样就可以用并查集快速实现。

而一旦不满足我们就需要新开一段，重置并查集。而我们只需记录并查集修改过的元素就可以快速回退之前的结果了。

时间复杂度为 $\mathcal{O}(n+m\alpha(n))$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, m = MII()
    dsu = UnionFind(n * 2)
    
    ans = 1
    cur = []
    
    for _ in range(m):
        u, v = GMI()
    
        if dsu.find(u) == dsu.find(v):
            for x in cur:
                dsu.back(x)
                dsu.back(x + n)
    
            ans += 1
            cur.clear()
    
        dsu.merge(u, v + n)
        dsu.merge(v, u + n)
    
        cur.append(u)
        cur.append(v)
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

    int n, m;
    cin >> n >> m;

    atcoder::dsu uf(n * 2);
    int ans = 1;
    vector<int> cur;

    while (m --) {
        int u, v;
        cin >> u >> v;
        u --, v --;

        if (uf.leader(u) == uf.leader(v)) {
            ans ++;
            for (auto &x: cur) {
                uf.reinit(x);
                uf.reinit(x + n);
            }
            cur.clear();
        }

        uf.merge(u, v + n);
        uf.merge(v, u + n);

        cur.emplace_back(u);
        cur.emplace_back(v);
    }

    cout << ans;

	return 0;
}
```
