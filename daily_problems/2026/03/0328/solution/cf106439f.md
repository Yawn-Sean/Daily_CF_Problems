**提示 1：** 考虑如何将 $n$ 的问题转化为规模更小的问题。

本题相当于求基环森林，每个点都不指向自己，且只有一个点入度为 $0$ 。

不妨枚举这个点。设 $n$ 个点的答案是 $f(n)$ 。

如果去掉这个点仍然满足上述条件，则有 $f(n-1)$ 种方案。而这个点本身有 $n$ 种取法，因此可以将 $nf(n-1)$ 转移到 $f(n)$ 中去。

否则，这个点去掉之后只剩下了一堆环，且没有自环。这就是 $n-1$ 个点的错排方案。这个可以自行搜索怎么算。

此时这个点有 $n$ 种选法，其指向的点有 $n-1$ 种选法，因此这里需要将 $n(n-1)$ 倍的 $n-1$ 的错排转移到 $f(n)$ 中。

综上完成 DP，时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    M = 10 ** 6 + 5
    mod = 998244353
    
    mis = [0] * M
    mis[0] = 1
    for i in range(2, M):
        mis[i] = (i - 1) * (mis[i - 1] + mis[i - 2]) % mod
    
    dp = [0] * M
    for i in range(3, M):
        dp[i] = (dp[i - 1] + (i - 1) * mis[i - 1]) % mod * i % mod
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        outs.append(dp[n])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 5, mod = 998244353;

	vector<int> mis(M, 0), dp(M, 0);

	mis[0] = 1;
	for (int i = 2; i < M; i ++)
		mis[i] = 1ll * (i - 1) * (mis[i - 1] + mis[i - 2]) % mod;

	for (int i = 3; i < M; i ++)
		dp[i] = (dp[i - 1] + 1ll * (i - 1) * mis[i - 1]) % mod * i % mod;

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;
		cout << dp[n] << '\n';
	}

	return 0;
}
```
