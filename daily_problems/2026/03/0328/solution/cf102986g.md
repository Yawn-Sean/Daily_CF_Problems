**提示 1：** 考虑用期望可加性。

**提示 2：** 中间部分怎么考虑。

考虑期望可加性。整体可以分为三个步骤：先到第一个要到的位置，中间走到最后要到的位置，再从最后要到的位置走到终点。

第一部分的期望等于第一个位置的期望减去 $0$ ，最后一部分的期望等于 $W$ 减去最后一个位置的期望。第一个位置的期望等于最后一个位置的期望，所以这部分加起来结果是 $1$ 。

中间部分，每一步是 $(i,j)$ 的概率都是相等的，所以可以计算所有 $(i,j)$ 距离的均值。总共又走了 $n-1$ 次，因此将这个结果乘 $n-1$ 加入答案即可。

时间复杂度为 $\mathcal{O}(n\log n)$ ，当然可以利用值域较小进一步优化。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, w = MII()
    nums = LII()
    nums.sort()
    
    ans = 0
    for i in range(n):
        ans += (2 * i - (n - 1)) * nums[i]
    
    print(ans * 2 / n + w)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, w;
	cin >> n >> w;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	sort(nums.begin(), nums.end());
	long long ans = 0;

	for (int i = 0; i < n; i ++)
		ans += 1ll * (2 * i - (n - 1)) * nums[i];

	cout << fixed << setprecision(15) << (long double)ans * 2 / n + w;

	return 0;
}
```
