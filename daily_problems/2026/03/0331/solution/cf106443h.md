**提示 1：** 哪些本来就不可能吃到？

**提示 2：** 考虑相邻的两个食物之间需要满足的条件。如何简化问题？

首先，如果有 $x\gt y$ ，显然直接往那边跑也来不及，直接去掉这些。

否则，如果相邻的两个被吃的位置是 $(x_1,y_1), (x_2,y_2)$ ，则 $|x_1-x_2|\leq y_2-y_1$ ，因此 $y_1-y_2\leq x_1-x_2\leq y_1+y_2$ ，也就是 $y-x,y+x$ 都满足小于等于的关系。

所以我们设两个维度是 $y-x,y+x$ ，则只需要关于一个维度排序，这样对另一个维度就只需求最长非递减子序列就行。注意第一个维度相同的情况下，第二个维度升序排列。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n = II()
    xs = []
    ys = []
    
    for _ in range(n):
        x, y = MII()
        if y - x >= 0:
            xs.append(y - x)
            ys.append(y + x)
    
    n = len(xs)
    
    st_range = list(range(n))
    
    st_range.sort(key=lambda x: ys[x])
    st_range.sort(key=lambda x: xs[x])
    
    lis = []
    
    for i in st_range:
        if len(lis) == 0 or lis[-1] <= ys[i]:
            lis.append(ys[i])
        else:
            lis[bisect.bisect_right(lis, ys[i])] = ys[i]
    
    print(len(lis))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<pair<int, int>> pts;

	for (int i = 0; i < n; i ++) {
		int x, y;
		cin >> x >> y;
		if (y - x >= 0)
			pts.emplace_back(y - x, y + x);
	}

	sort(pts.begin(), pts.end());

	vector<int> lis;
	for (auto &[x, y]: pts) {
		if (lis.empty() || lis.back() <= y) lis.emplace_back(y);
		else lis[upper_bound(lis.begin(), lis.end(), y) - lis.begin()] = y;
	}

	cout << lis.size();

	return 0;
}
```
