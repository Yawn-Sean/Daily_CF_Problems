**提示 1：** 只需考虑二元集合。

**提示 2：** 二元集合求的是什么？这件事怎么最小化？

首先肯定只需要考虑二元集合。因为这里某一位结果是 $1$ ，等价于选取的子集中的元素的这一位有 $0$ 也有 $1$ ，而集合越小这件事情越不容易达成。

二元集合的情况下，这里求的就是元素的异或。一个元素集的异或最小值，可以用 Trie ，也可以注意到最小异或一定是排序后某一组相邻元素的异或。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
        
        nums.sort()
        outs.append(min(nums[i] ^ nums[i - 1] for i in range(1, n)))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;

		sort(nums.begin(), nums.end());

		int ans = 2e9;
		for (int i = 1; i < n; i ++)
			ans = min(ans, nums[i] ^ nums[i - 1]);

		cout << ans << '\n';
	}

	return 0;
}
```
