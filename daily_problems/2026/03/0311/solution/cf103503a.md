**提示 1：** 添加的数字越小越好。

**提示 2：** 一旦你知道添加的数字的最大值，你就已经知道加了几个数了。

首先，我们应该不断挑选还不在数组中的最小元素进去。

这样到某个数停下来时，我们只要知道这个数是多少就能快速得到加了几个数（就是这个数减去原数组中比它小的数的个数）。

我们可以二分添加数字到 $x$ ，则我们可以计算添加到 $x$ 时，数组的和是多少（等于原数组的和，减去原数组中小于等于 $x$ 的数的和，加上 $1\sim x$ 的和），根据其和 $S$ 的大小关系得到最终的 $x$ ，也就得到了答案。

时间复杂度为 $\mathcal{O}(n\log n+\log M\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, s = MII()
    nums = LII()
    
    nums.sort()
    cur = sum(nums)
    
    acc = list(accumulate(nums, initial=0))
    
    l, r = 0, 2 * 10 ** 9
    
    while l <= r:
        mid = (l + r) // 2
        pos = bisect.bisect_left(nums, mid)
        if cur + (mid + 1) * mid // 2 - acc[pos] >= s:
            r = mid - 1
        else:
            l = mid + 1
    
    print(l - bisect.bisect_left(nums, l) + n)

```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	long long s;
	cin >> n >> s;

	vector<long long> nums(n);
	for (auto &v: nums) cin >> v;

	long long cur = 0;
	for (auto &v: nums) cur += v;

	sort(nums.begin(), nums.end());

	vector<long long> acc(n + 1, 0);
	for (int i = 0; i < n; i ++)
		acc[i + 1] = acc[i] + nums[i];

	long long l = 0, r = 2e9;
	while (l <= r) {
		long long mid = (l + r) / 2;
		int pos = lower_bound(nums.begin(), nums.end(), mid) - nums.begin();
		if (cur + (mid + 1) * mid / 2 - acc[pos] >= s) r = mid - 1;
		else l = mid + 1;
	}

	int pos = lower_bound(nums.begin(), nums.end(), l) - nums.begin();
	cout << l - pos + n;

	return 0;
}
```
