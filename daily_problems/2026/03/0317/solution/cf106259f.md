**提示 1：** 每个 $(i,j)$ 都对结果产生贡献。

**提示 2：** 每个 $(i,j)$ 对结果的贡献权重是一样的。

先看提示。每个 $(i,j)$ 对结果有贡献，且贡献是 $i,j$ 在某一次被同时操作的概率乘以 $|a_i-a_j|$ 。

因为任何一对 $(i,j)$ 被操作的概率是等价的，因此权重一致，而总权重是 $n-1$ ，因为操作这么多次，而总共有 $\frac{n(n-1)}{2}$ 对数，所以每个 $(i,j)$ 对的权重都是 $(n-1)/\frac{n(n-1)}{2}=\frac{2}{n}$ 。

所以最后我们要求的就是 $\frac{2}{n}\sum\limits_{1\leq i\lt j\leq n}|a_i-a_j|$ 。

后者可以考虑对数组进行排序，看中间每一段间隔的权重。每一段间隔的权重等于前面的点数乘后面的点数。

因为涉及到排序，所以时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    mod = 998244353
    
    for _ in range(t):
        n = II()
        nums = LII()
        nums.sort()
        
        ans = 0
        for i in range(1, n):
            ans = (ans + i * (n - i) * (nums[i] - nums[i - 1])) % mod
        
        outs.append(ans * 2 * pow(n, -1, mod) % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t, mod = 998244353;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> nums(n);
		for (auto &v: nums) cin >> v;

		sort(nums.begin(), nums.end());

		int ans = 0;
		for (int i = 1; i < n; i ++)
			ans = (ans + 1ll * i * (n - i) * (nums[i] - nums[i - 1])) % mod;

		cout << 2ll * ans * quickPow(n, mod - 2, mod) % mod << '\n';
	}

	return 0;
}
```
