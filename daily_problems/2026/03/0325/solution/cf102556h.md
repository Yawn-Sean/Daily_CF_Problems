**提示 1：** 对于一个 $N$ ，其 $M$ 如何计算？

**提示 2：** 应当枚举哪些 $N$ 呢？

对于一个 $N$ ，我们可以将其因子 $x$ 与 $N/x$ 进行两两配对，这样假设总共有 $cnt$ 个因子，则乘积应当等于 $N^{cnt/2}$ 。

所以 $M$ 一定是某个数的幂次，同时 $N$ 也是这个数的幂次。因此我们找到最小的使得 $M=p^x$ 的 $p$ ，再枚举 $N=p^v$ 即可。这里 $N^{cnt/2}$ 是否和 $M$ 相等也可以通过比较质因子次数得到。

时间复杂度为 $\mathcal{O}(\sqrt{M})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    m = II()
    
    if m == 1:
        exit(print(1))
    
    cnt = Counter()
    
    for x in range(2, 4 * 10 ** 7):
        while m % x == 0:
            cnt[x] += 1
            m //= x
    
    if m > 1:
        cnt[m] += 1
    
    g = 0
    for x in cnt.values():
        g = math.gcd(g, 2 * x)
    
    for i in range(1, g + 1):
        if g % i == 0:
            
            factor_cnt = 1
            for x in cnt:
                factor_cnt *= 1 + 2 * cnt[x] // g * i
            
            if i * factor_cnt == g:
                ans = 1
                for x in cnt:
                    ans *= pow(x, 2 * cnt[x] // g * i)
                print(ans)
                exit()
    
    print(-1)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	long long m;
	cin >> m;

	if (m == 1) return cout << 1, 0;

	unordered_map<long long, int> cnt;

	for (int x = 2; x < 40'000'000; x ++) {
		while (m % x == 0) {
			cnt[x] ++;
			m /= x;
		}
	}

	if (m > 1) cnt[m] ++;

	int g = 0;
	for (auto &[k, v]: cnt) g = gcd(g, 2 * v);

	for (int i = 1; i <= g; i ++) {
		if (g % i == 0) {
			int factor_cnt = 1;
			for (auto &[k, v]: cnt)
				factor_cnt *= 1 + 2 * v / g * i;
			if (i * factor_cnt == g) {
				long long ans = 1;
				for (auto &[k, v]: cnt) {
					for (int j = 0; j < 2 * v / g * i; j ++) {
						ans *= k;
					}
				}
				return cout << ans, 0;
			}
		}
	}

	cout << -1;

	return 0;
}
```
