**提示 1：** 尝试将一个数字移动到另一个位置。这件事要多少的步骤呢？

考虑将一个元素移动到另一个位置需要多少成本。

不妨考虑将某个元素移动到最后。如果直接利用原操作，可以用一个长度为 $2l$ 的序列将某个元素移动 $l$ 个格子。这样经过 $\log$ 量级的操作就可以完成这个任务，整体就是 $\mathcal{O}(n\log n)$ 次操作。能获得 80 分。

这个操作主要的坏处在于，最后明明没差几步了却还得这么走。有没有什么更好的想法呢？

我们不考虑从当前序列恢复到排序，而是利用当前操作的 “逆操作”，将一个排序的序列变成当前序列，这样我们最后反向操作就好了。

这样的效果是什么呢？如果当前该元素的位置是 $i$ ，则逆操作后可以变成 $2i$ ，且最后剩下的一小步可以直接到达。

这样，如果随机从一个位置到第 $n$ 个位置，需要的步数的期望就是 $\sum\log\frac{n}{i}/n$ 量级的，而这个数由 Stirling 公式可以发现是 $\mathcal{O}(1)$ 的。因此总共 $n$ 次这样的操作，总操作次数就是 $\mathcal{O}(n)$ 的了。

怎么让我们的操作次数更接近期望而非极端情况呢？可以使用随机化，一开始将序列打乱，这样就更不容易出现被精心设计的数据卡次数的情况了。

时间复杂度为 $\mathcal{O}(n^2)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n = II()
    perm = LGMI()
    
    ops = []
    help_perm = list(range(n))
    
    def op(l, r):
        ops.append(f'{l + 1} {r + 1}')
        half = (r - l + 1) // 2
        vals = []
        for i in range(half):
            vals.append(help_perm[l + i + half])
            vals.append(help_perm[l + i])
        for i in range(l, r + 1):
            help_perm[i] = vals[i - l]
    
    for _ in range(500):
        l = random.randint(0, n - 1)
        r = random.randint(0, n - 1)
        
        if (r - l) % 2:
            if l > r: l, r = r, l
            op(l, r)
    
    for i in range(n - 1, -1, -1):
        for j in range(i):
            if help_perm[j] == perm[i]:
                cur = j
                while cur != i:
                    v = fmin(cur + 1, i - cur)
                    op(cur - v + 1, cur + v)
                    cur += v
    
    ops.reverse()
    
    print(len(ops))
    print('\n'.join(ops))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> perm(n), help_perm(n);
	for (auto &x: perm) cin >> x;
	iota(help_perm.begin(), help_perm.end(), 1);

	vector<pair<int, int>> ops;

	auto op = [&] (int l, int r) -> void {
		ops.emplace_back(l + 1, r + 1);

		int half = (r - l + 1) / 2;
		vector<int> vals;

		for (int i = 0; i < half; i ++) {
			vals.emplace_back(help_perm[l + i + half]);
			vals.emplace_back(help_perm[l + i]);
		}

		for (int i = l; i <= r; i ++) help_perm[i] = vals[i - l];
	};

	for (int i = 0; i < 500; i ++) {
		int l = rng();
		int r = rng();

		l = abs(l) % n;
		r = abs(r) % n;

		if ((r - l) & 1)  {
			if (l > r) swap(l, r);
			op(l, r);
		}
	}

	for (int i = n - 1; i >= 0; i --) {
		for (int j = 0; j < i; j ++) {
			if (help_perm[j] == perm[i]) {
				int cur = j;
				while (cur != i) {
					int v = min(cur + 1, i - cur);
					op(cur - v + 1, cur + v);
					cur += v;
				}
			}
		}
	}

	reverse(ops.begin(), ops.end());

	cout << ops.size() << '\n';
	for (auto &[x, y]: ops) cout << x << ' ' << y << '\n';

	return 0;
}
```
