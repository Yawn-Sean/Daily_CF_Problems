**提示 2：** 接下来怎么贪心？

其实两个操作可以让任意数码在任意位置。

我们可以先调整所有第一个数字要用的各个数码到第一个数字中去。为此我们把第一个数字不要的数码移动的末尾，再把第二个数字中要拿给第一个数字的数码移动到末尾，再进行交换。交换若干次显然就能完成任务。

而越大的数码显然放在越高位越好，所以这么贪心的安排各个数码。

最后将两个安排好的数取模再求和即可。

时间复杂度为 $\mathcal{O}(\log A+\log B)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    sa = [int(c) for c in I()]
    sb = [int(c) for c in I()]
    
    cnt = [0] * 10
    for x in sa: cnt[x] += 1
    for x in sb: cnt[x] += 1
    
    l = max(len(sa), len(sb))
    weights = [0] * l
    
    for i in range(len(sa)):
        weights[i] += 1
    
    for i in range(len(sb)):
        weights[i] += 1
    
    ans = 0
    cur = 9
    
    mod = 998244353
    
    for i in range(l - 1, -1, -1):
        ans = ans * 10 % mod
        
        for _ in range(weights[i]):
            while cnt[cur] == 0:
                cur -= 1
            ans = (ans + cur) % mod
            cnt[cur] -= 1
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	string sa, sb;
	cin >> sa >> sb;

	vector<int> cnt(10, 0);
	for (auto &c: sa) cnt[c - '0'] ++;
	for (auto &c: sb) cnt[c - '0'] ++;

	int l = max(sa.size(), sb.size());
	vector<int> weights(l, 0);
	for (int i = 0; i < sa.size(); i ++) weights[i] ++;
	for (int i = 0; i < sb.size(); i ++) weights[i] ++;

	int ans = 0, cur = 9, mod = 998244353;
	for (int i = l - 1; i >= 0; i --) {
		ans = 10ll * ans % mod;
		while (weights[i] --) {
			while (!cnt[cur]) cur --;
			ans = (ans + cur) % mod;
			cnt[cur] --;
		}
	}

	cout << ans;

	return 0;
}
```
