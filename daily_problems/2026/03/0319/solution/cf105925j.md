**提示 1：** 粒子相位和过滤器是相对的。

**提示 2：** 认为粒子是不变的，则后面的过滤器每个应该怎么调整？

对于提示 2，我们可以将后面第一个过滤器的 $A_i$ 减去 $K$ ，紧接着的分别减去 $2K,3K,\dots$ 以此类推。这样我们只需找到后缀中第一个比当前的数小的数就行。

因为绕一圈回到自己的时候显然已经会被过滤掉，所以我们只需复制一遍数组，就可以模拟完整的过滤过程了。

而前面的减去 $K$ 这件事，可以直接把 $i$ 位置的数值减去 $i\times K$ ，那么后缀就相对于当前位置分别减去 $K,2K,\dots$ 了。

此时这么处理，就只需求后缀更小的元素，这用单调栈显然就够了。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, k = MII()
    nums = LII()
    
    for i in range(n):
        nums.append(nums[i])
    
    for i in range(2 * n):
        nums[i] -= i * k
    
    ans = [0] * n
    
    stk = []
    for i in range(2 * n - 1, -1, -1):
        while stk and nums[stk[-1]] >= nums[i]:
            stk.pop()
        if i < n: ans[i] = stk[-1] % n + 1
        stk.append(i)
    
    print(' '.join(map(str, ans)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, k;
	cin >> n >> k;

	vector<long long> nums(n);
	for (auto &v: nums) cin >> v;

	for (int i = 0; i < n; i ++) nums.emplace_back(nums[i]);

	for (int i = 0; i < 2 * n; i ++)
		nums[i] -= 1ll * i * k;

	vector<int> stk;
	vector<int> ans(n);
	for (int i = 2 * n - 1; i >= 0; i --) {
		while (!stk.empty() && nums[stk.back()] >= nums[i])
			stk.pop_back();
		if (i < n) ans[i] = stk.back() % n + 1;
		stk.emplace_back(i);
	}

	for (auto &x: ans) cout << x << ' ';

	return 0;
}
```
