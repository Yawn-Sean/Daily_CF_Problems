**提示 1：** 考虑总状态数和不合法的状态数。

首先总方案数是多少呢？考虑 $a,a+b$ 是从 $1,2,\dots,n-1$ 中选两个数，因此方案是 $\frac{(n-1)(n-2)}{2}$ 种。

不合法的方案数呢？也就是最大值大于等于 $n/2$ 的。这个元素有三种取法，设为 $k$ ，而剩余两个元素和是 $n-k$ ，因此方案数是 $n-k-1$ 。关于不同的 $k$ 求和又是个等差数列了。

接下来概率就用 $1$ 减去不合法概率就好。

时间复杂度为 $\mathcal{O}(\log mod)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    mod = 10 ** 9 + 7
    
    outs = []
    
    for _ in range(t):
        n = II()
        total = (n - 1) * (n - 2) // 2 % mod
        invalid = n // 2 * (n // 2 - 1) // 2 * 3 % mod
        outs.append((total - invalid) * pow(total, -1, mod) % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int mod = 1e9 + 7;

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		int total = 1ll * (n - 1) * (n - 2) / 2 % mod;
		int valid = 1ll * (n / 2) * (n / 2 - 1) / 2 % mod * 3 % mod;

		cout << 1ll * (mod + total - valid) % mod * quickPow(total, mod - 2, mod) % mod << '\n';
	}

	return 0;
}
```
