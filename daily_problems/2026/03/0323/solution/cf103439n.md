**提示 2：** 应该怎么配对呢？

权值相当于取其中一对的最大值减去另一对的最小值。（取其中更大的结果）

我们简单的认为是取其中一对的最大值减去另一对的最小值，这样最大化这个结果的时候也顺便解决了原问题（原问题不可能有更优的解）。

所以每一对我们都调整到 $a\leq b$ ，这样我们就是取一半的 $b$ 减去另一半的 $a$ 。

我们不妨以 $\sum b$ 为基准，在这个基础上，我们只需减去一半的 $a+b$ 就行。这显然可以取最小的一半。

时间复杂度为 $\mathcal{O}(n\log n)$ 。如果用快速选择可以线性解决。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n = II()
    
    xs = []
    ys = []
    
    for _ in range(n * 2):
        x, y = MII()
        if x > y: x, y = y, x
        xs.append(x)
        ys.append(y)
    
    st_range = sorted(range(2 * n), key=lambda i: xs[i] + ys[i])
    
    print(sum(ys[st_range[2 * n - 1 - i]] - xs[st_range[i]] for i in range(n)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<pair<int, int>> pairs(2 * n);
	for (auto &[x, y]: pairs) cin >> x >> y;

	sort(pairs.begin(), pairs.end(), [&] (pair<int, int> &x, pair<int, int> &y) {
		return x.first + x.second < y.first + y.second;
	});

	long long ans = 0;

	for (int i = 0; i < n; i ++) {
		auto [x1, y1] = pairs[i];
		auto [x2, y2] = pairs[2 * n - 1 - i];
		ans += max(x2, y2) - min(x1, y1);
	}

	cout << ans << '\n';

	return 0;
}
```
