**提示 1：** 将问题转化为一个好解决的形式。

因为我们要求均值在 $[a,b]$ 之间的子数组个数，我们可以理解为均值不超过 $b$ 的子数组个数减去均值严格小于 $a$ 的子数组个数。

不超过 $b$ 可以理解为严格小于 $b+\frac{1}{n}$ ，因此这样就完全只需解决均值不超过 $x$ 的子数组个数的问题。

我们将每个元素都减去 $x$ ，这里减去的数是有理数，可以同乘分母后再减，就毫无精度问题了。这样就是求均值小于 $0$ 的子数组个数。

这样的子数组在前缀和数组中对应一个逆序对，因此直接求逆序对个数即可。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, a, b = MII()
    nums = LII()
    
    ans = 0
    
    acc = [0] * (n + 1)
    
    for i in range(n):
        acc[i + 1] = acc[i] + (nums[i] * n - (b * n + 1))
    
    ans += reversePairs(acc)
    
    for i in range(n):
        acc[i + 1] = acc[i] + (nums[i] * n - a * n)
    
    ans -= reversePairs(acc)
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, a, b;
	cin >> n >> a >> b;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	long long ans = 0;

	vector<long long> acc(n + 1, 0);

	for (int i = 0; i < n; i ++) acc[i + 1] = acc[i] + 1ll * nums[i] * n - (1ll * b * n + 1);
	ans += reversePairs(acc);

	for (int i = 0; i < n; i ++) acc[i + 1] = acc[i] + 1ll * nums[i] * n - 1ll * a * n;
	ans -= reversePairs(acc);

	cout << ans;

	return 0;
}
```
