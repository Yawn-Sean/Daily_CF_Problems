**提示 1：** 很明显，最容易确定数值的是 $2^i$ 形式的数，查询两个 $2^i$ ，得到的结果就是 $f(2^i)$ 的两倍。

**提示 2：** 如何将其他数变成 $2^i$ 。

先看提示 1，那么剩下的数咋办呢？

我们只能用将求 $f(x)$ 转化为求另一个 $f(2^k-x)$ ，这样能到一个 $2^i$ 吗？

答案是肯定的，我们只需找到最小的满足 $2^k\gt x$ 的 $k$ ，这样 $2^k-x$ 二进制表示的位数一定更小，同时最低的非零位保持不变，因此迟早会只剩下最低的非零位。

一个数所需的查询数量是 $\mathcal{O}(\log M)$ 的。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    def query(a, b):
        print('?', a, b, flush=True)
        return int(LI()[1])
    
    def answer(x):
        print('!', x, flush=True)
        return I()
    
    def solve(x):
        if x & -x == x:
            return query(x, x) // 2
        v = 1 << x.bit_length()
        return query(v - x, x) - solve(v - x)
    
    n = II()
    nums = LII()
    
    for x in nums:
        answer(solve(x))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	auto query = [&] (long long x, long long y) -> long long {
		cout << "? " << x << ' ' << y << endl;
		char c; long long v;
		cin >> c >> v;
		return v;
	};

	auto answer = [&] (long long x) -> void {
		cout << "! " << x << endl;
		char c; cin >> c;
	};

	auto solve = [&] (auto &self, long long x) -> long long {
		if ((x & -x) == x) return query(x, x) / 2;
		long long v = 1ll << 64 - __builtin_clzll(x);
		return query(v - x, x) - self(self, v - x);
	};

	int n;
	cin >> n;

	vector<long long> vals(n);
	for (auto &v: vals) cin >> v;
	for (auto &v: vals) answer(solve(solve, v));

	return 0;
}
```
