**提示 1：** 整棵树的深度其实可以转化为更简单的形式。

**提示 2：** 考虑枚举边。

整棵树深度最深的点一定是 $n$ 。（可以想下为啥）

于是考虑枚举 $n$ 连的边。

如果连到 $n-1$ ，则对应的原树答案是 $dp[n-1]$ ，新增高度和等于 $1$ 乘以方案数，即 $2^{n-2}$ ，因此将 $dp[n-1]+2^{n-2}$ 转移过去。

如果连到 $n-2$ ，则对应原树答案是 $dp[n-2]$ ，且因为 $n-1$ 可以任意选择连到哪个节点，所以这个原树答案需要乘以 $2$ 再转移。新增高度和的逻辑跟之前类似，因此 $dp[n]=dp[n-1]+2dp[n-2]+2^{n-1}$ 。完成 DP 即可。

时间复杂度为 $\mathcal{O}(M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    M = 2 * 10 ** 5 + 5
    mod = 10 ** 9 + 7
    
    pw2 = [1] * M
    dp = [0] * M
    
    for i in range(1, M):
        pw2[i] = pw2[i - 1] * 2 % mod
    
    dp[1] = 1
    
    for i in range(2, M):
        dp[i] = (dp[i - 1] + 2 * dp[i - 2] + pw2[i - 1]) % mod
    
    t = II()
    outs = []
    
    for _ in range(t):
        outs.append(dp[II()])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 2e5 + 5, mod = 1e9 + 7;

	vector<int> pw2(M, 1), dp(M, 0);
	for (int i = 1; i < M; i ++) pw2[i] = 2 * pw2[i - 1] % mod;
	
	dp[1] = 1;
	for (int i = 2; i < M; i ++)
		dp[i] = (dp[i - 1] + 2ll * dp[i - 2] + pw2[i - 1]) % mod;

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;
		cout << dp[n] << '\n';
	}

	return 0;
}
```
