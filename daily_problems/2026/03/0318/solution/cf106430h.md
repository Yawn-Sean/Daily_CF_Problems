**提示 1：** 如果我们可以预处理 $x+y=V$ 的结果，这样将这些结果用前缀和预处理就能得到答案了。同时也可以每次查询 $l=r=V$ ，所以求出每一个结果是必要的。

**提示 2：** 同一组 $V$ 怎么算答案？

首先看提示。

接下来求解 $x+y=V$ 的情况下的答案。我们可以枚举 $V$ 的质因子 $p$ ，让 $\mathrm{gcd}(x,y)=p$ ，这样，等式两边同除以 $p$ 后，我们就只需求将 $V/p$ 拆分为两个正的互质数的和的方案数。这显然就是 $\phi$ 函数。

$\phi(x)$ 等于 $x$ 乘以 $(1-1/p_1)(1-1/p_2)\dots$ ， $p_1,p_2,\dots$ 是 $x$ 的各个不同的质因子。

$\phi(x)$ 可以在素数筛的过程中快速计算。同时后面的 $\sum\phi(V/p)$ 也可以在遍历每个 $p$ 的时候，在后面 $p,2p,\dots$ 的位置进行更新。因此两轮类似埃氏筛的过程可以解决这个问题。

时间复杂度为 $\mathcal{O}(M\log\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    M = 2 * 10 ** 5
    
    phi = list(range(M + 1))
    
    phi[1] = 0
    for i in range(2, M + 1):
        if phi[i] == i:
            for j in range(i, M + 1, i):
                phi[j] = phi[j] // i * (i - 1)
    
    acc = [0] * (M + 1)
    
    for i in range(2, M + 1):
        if phi[i] == i - 1:
            for j in range(i, M + 1, i):
                acc[j] += phi[j // i]
    
    for i in range(1, M + 1):
        acc[i] += acc[i - 1]
    
    t = II()
    outs = []
    
    for _ in range(t):
        l, r = MII()
        outs.append(acc[r] - acc[l - 1])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 2e5;

	vector<int> phi(M + 1);
	iota(phi.begin(), phi.end(), 0);
	phi[1] = 0;

	for (int i = 2; i <= M; i ++) {
		if (phi[i] == i) {
			for (int j = i; j <= M; j += i) {
				phi[j] = phi[j] / i * (i - 1);
			}
		}
	}

	vector<long long> acc(M + 1, 0);

	for (int i = 2; i <= M; i ++) {
		if (phi[i] == i - 1) {
			for (int j = i; j <= M; j += i) {
				acc[j] += phi[j / i];
			}
		}
	}

	for (int i = 1; i <= M; i ++) acc[i] += acc[i - 1];

	int t;
	cin >> t;

	while (t --) {
		int l, r;
		cin >> l >> r;
		cout << acc[r] - acc[l - 1] << '\n';
	}

	return 0;
}
```
