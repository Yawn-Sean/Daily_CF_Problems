**提示 1：** 注意你只需要求期望。有可加性。

**提示 2：** 考虑每个最大值的路径有几条。

表面上求极差期望，其实只需求最大值和最小值的期望再相减。这两者本质一致，我们只说最大值。

我们从小到达遍历结点，并将遍历过的结点之间连边。这样我们可以在遍历 $i$ 结点时，得到总的路径条数（每个大小为 $x$ 的连通块路径有 $C_x^2$ 条）。与前一轮相减就是新增的路径数量，这个数量乘以 $i$ 的权值就是 $i$ 的贡献。将这些贡献相加即可。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    nums = LII()
    
    for _ in range(8):
        nnums = []
        for i in range(26):
            nnums.append(nums[i])
            nnums.append(nums[i + 26])
        nums = nnums
        
        flg = True
        for i in range(26, 52):
            if nums[i] == 1:
                flg = False
        
        if flg: exit(print('YES'))
    
    print('NO')
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	vector<int> nums(52);
	for (auto &v: nums) cin >> v;

	for (int i = 0; i < 8; i ++) {
		vector<int> nnums;
		for (int i = 0; i < 26; i ++) {
			nnums.emplace_back(nums[i]);
			nnums.emplace_back(nums[i + 26]);
		}
		nums.swap(nnums);

		bool flg = true;
		for (int i = 26; i < 52; i ++) {
			if (nums[i] == 1) {
				flg = false;
			}
		}

		if (flg) return cout << "YES", 0;
	}

	cout << "NO";

	return 0;
}
```
