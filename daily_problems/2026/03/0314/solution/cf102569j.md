**提示 1：** 找到一个简单一点的形式。

考虑两个数组长度都为 $3$ 。此时暴力一点的方式可以直接枚举所有组合看是否满足要求。我们这里说下手动构造的可能方法。

先假设两个是 $a,a,b$ 和 $c,c,d$ 的形式。同时 $a\lt b$ ， $c\lt d$ 。

这样有 $4$ 个自由度，组成三个不等式蛮可能是可以有解的。

如果 $c$ 比 $a,b$ 都小，第一个条件自动满足。

如果 $a+b\gt c+d$ ，那么第二个条件也满足。

如果 $2a+b\gt 2c+d$ ，第三个条件也满足。

这个凑解就容易多了。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    print(3)
    print(3, 3, 4)
    print(3)
    print(1, 1, 7)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	cout << "3\n3 3 4\n3\n1 1 7";

	return 0;
}
```
