**提示 1：** 考虑每一组能整除的元素对答案的贡献。

看提示。

对于每一组满足 $i$ 能被 $j$ 整除的元素对 $(i,j)$ ，则如果 $i,j\leq N$ ，那么长度为 $N$ 的排列中，恰好有一半使得 $i$ 在 $j$ 后面。即其贡献为 $\frac{N!}{2}$ 。

所以我们只需计算不超过 $N$ 的元素对中出现的整除关系的数量，乘 $\frac{N!}{2}$ 即答案。

也就是 $1\sim N$ 中每个元素的非自身的因子个数之和。于是，预处理 $1\sim N$ 的因子个数，求前缀和即可。

前者可以通过枚举较小元素 $x$ ，在 $2x,3x,\dots$ 加入计数，因为 $x$ 是这些元素的因子，这样就可以用 $\mathcal{O}(M\log M)$ 复杂度预处理。

预处理时间复杂度为 $\mathcal{O}(M\log M)$ ，单次查询复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 10 ** 5 + 5
    cnt = [0] * M
    
    for i in range(1, M):
        for j in range(i * 2, M, i):
            cnt[j] += 1
    
    for i in range(1, M):
        cnt[i] += cnt[i - 1]
    
    outs = []
    mod = 10 ** 9 + 7
    rev2 = (mod + 1) // 2
    
    f = [1] * M
    
    for i in range(1, M):
        f[i] = i * f[i - 1] % mod
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        outs.append(cnt[n] * rev2 % mod * f[n] % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e5 + 5;
	vector<int> cnt(M, 0);

	for (int i = 1; i < M; i ++) {
		for (int j = i * 2; j < M; j += i) {
			cnt[j] ++;
		}
	}

	for (int i = 1; i < M; i ++) cnt[i] += cnt[i - 1];

	int mod = 1e9 + 7, rev2 = (mod + 1) / 2;
	vector<int> f(M, 1);
	for (int i = 1; i < M; i ++) f[i] = 1ll * i * f[i - 1] % mod;

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;
		cout << 1ll * cnt[n] * rev2 % mod * f[n] % mod << '\n';
	}

	return 0;
}
```
