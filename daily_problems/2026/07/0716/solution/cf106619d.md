**提示 1：** 先找到两个元素的关系，再确认其中一个元素。

首先可以每个位都查一次。这样可以知道两个元素的异或和。

接下来只要找到其中一个元素即可。

我们选取异或和中非 $0$ 的一位 $x$ ，考虑任何一个其它的位，如果 $x+2^v$ 的结果为 $1$ ，则有 $x$ 的那一个数一定有 $2^v$ ，否则一定没有。于是查 $59$ 次可以确定这个数。

然后用异或和推另一个数即可。最后需要调整到 $a\lt b$ 。

时间复杂度为 $\mathcal{O}(\log M)$ ，查询次数也是这个量级。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    def query(m, v):
        print('?', m, v, flush=True)
        return II()
    
    def answer(x, y):
        print('!', x, y, flush=True)
    
    t = II()
    outs = []
    
    for _ in range(t):
        xor_val = 0
        
        for i in range(60):
            if query(1 << i, 1 << i):
                xor_val |= 1 << i
    
        v = xor_val & -xor_val
        
        for i in range(60):
            if v >> i & 1: continue
            nv = v | (1 << i)
            if query(nv, nv):
                v = nv
        
        x = v
        y = x ^ xor_val
        
        if x > y: x, y = y, x
        
        answer(x, y)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	auto query = [&] (long long m, long long v) -> int {
		cout << "? " << m << ' ' << v << endl;
		int x; cin >> x; return x;
	};

	auto answer = [&] (long long a, long long b) -> void {
		cout << "! " << a << ' ' << b << endl;
	};

	while (t --) {
		long long xor_val = 0;

		for (int i = 0; i < 60; i ++) {
			if (query(1ll << i, 1ll << i)) {
				xor_val |= 1ll << i;
			}
		}

		long long tmp = xor_val & -xor_val;

		for (int i = 0; i < 60; i ++) {
			if (tmp >> i & 1) continue;
			long long ntmp = tmp | (1ll << i);
			if (query(ntmp, ntmp)) tmp = ntmp;
		}

		long long a = tmp, b = tmp ^ xor_val;
		if (a > b) swap(a, b);
		answer(a, b);
	}

	return 0;
}
```
