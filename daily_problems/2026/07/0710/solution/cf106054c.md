**提示 1：** 考虑 $i,j$ 能成功配对的条件。

要使得 $P[P[i]]=i$ ，则令 $P[i]=x$ 有 $P[x]=i$ ，即 $i,x$ 有这样的配对关系。

而如果数值 $x,y$ 这样配对配上了，那么它们移动后的位置是 $y,x$ 。而设它们刚开始的位置是 $p_x,p_y$ ，那么两个元素的移动次数应该一致，也就是 $(p_x-y)\bmod n=(p_y-x)\bmod n$ 。

于是 $(p_x+x)\bmod n=(p_y+y)\bmod n$ 。

因此只要满足这个条件 $x,y$ 就能配对。因此只需分类统计 $(p_x+x)\bmod n$ ，就可以知道有多少元素对满足这个条件相同。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    p = LGMI()
    
    d = [0] * n
    ans = 0
    
    for i in range(n):
        x = (i + p[i]) % n
        d[x] += 1
        ans += d[x]
    
    print(2 * ans - n)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	long long ans = 0;
	vector<int> cnt(n, 0);

	for (int i = 0; i < n; i ++) {
		int x;
		cin >> x;
		x = (x + i) % n;
		cnt[x] ++;
		ans += cnt[x];
	}

	cout << 2 * ans - n;

	return 0;
}
```
