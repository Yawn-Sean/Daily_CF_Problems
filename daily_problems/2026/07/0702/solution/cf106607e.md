**提示 1：** 两条线段的点有啥关系？所以相交的条件是？

首先 $(x,y)$ 关于 $y=x$ 对称后得到 $(x,y)$ ，所以整条线段相当于做了关于 $y=x$ 的对称。

于是，如果开始的线段在 $y=x$ 的同一边，则两条线段无交点。否则，原先在 $y=x$ 上的那个点对称后还在 $y=x$ 上，因此有交点。

所以容易一点的写法是用总点对数减去在同一边的点对数（用 $x,y$ 大小关系判断）。注意可能有点刚好在 $y=x$ 上，直接忽略就行，因为它们形成的线段已经有一点在 $y=x$ 上了，一定相交。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        ans = n * (n - 1) // 2
        
        c1 = 0
        c2 = 0
        
        for _ in range(n):
            x, y = MII()
            if x < y:
                ans -= c1
                c1 += 1
            elif x > y:
                ans -= c2
                c2 += 1
        
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, c0 = 0, c1 = 0;
		cin >> n;

		long long ans = 1ll * n * (n - 1) / 2;

		while (n --) {
			int x, y;
			cin >> x >> y;

			if (x > y) {
				ans -= c0;
				c0 ++;
			}
			else if (x < y) {
				ans -= c1;
				c1 ++;
			}
		}

		cout << ans << '\n';
	}

	return 0;
}
```
