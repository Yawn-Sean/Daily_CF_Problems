**提示 1：** 两条线段的点有啥关系？所以相交的条件是？

首先 $(x,y)$ 关于 $y=x$ 对称后得到 $(x,y)$ ，所以整条线段相当于做了关于 $y=x$ 的对称。

于是，如果开始的线段在 $y=x$ 的同一边，则两条线段无交点。否则，原先在 $y=x$ 上的那个点对称后还在 $y=x$ 上，因此有交点。

所以容易一点的写法是用总点对数减去在同一边的点对数（用 $x,y$ 大小关系判断）。注意可能有点刚好在 $y=x$ 上，直接忽略就行，因为它们形成的线段已经有一点在 $y=x$ 上了，一定相交。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 10 ** 6 + 5
    isPrime = [1] * M
    
    primes = []
    for i in range(2, M):
        if isPrime[i]:
            primes.append(i)
            for j in range(i, M, i):
                isPrime[j] = 0
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        
        if n == 1:
            outs.append(1)
            continue
        
        pr = []
        
        cur = n
        for i in primes:
            if i * i > cur: break
            if cur % i == 0:
            pr.append(i)
            while cur % i == 0:
                cur //= i
        
        if cur > 1: pr.append(cur)
        
        total = 0
        if len(pr) == 1: total = (pr[0] - 1) * pr[0] // 2
        else:
            for i in range(1, pr[0]):
                res = 1
                for p in pr:
                    res *= p - i
                total += res
        
        ans = n
        for p in pr:
            ans //= p
        
        outs.append(ans * total % 998244353)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, c0 = 0, c1 = 0;
		cin >> n;

		long long ans = 1ll * n * (n - 1) / 2;

		while (n --) {
			int x, y;
			cin >> x >> y;

			if (x > y) {
				ans -= c0;
				c0 ++;
			}
			else if (x < y) {
				ans -= c1;
				c1 ++;
			}
		}

		cout << ans << '\n';
	}

	return 0;
}
```
