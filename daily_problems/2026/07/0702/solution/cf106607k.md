**提示 1：** 将问题转化为区间数量。

**提示 2：** 分区间长度统计。

首先，相当于算的是每个左端点有多少个区间，满足跟 $n$ 互质。也就是统计满足跟 $n$ 互质的区间的个数。

而我们根据长度分类，分别统计可行的左端点个数。

如果 $n$ 的不同的质因子为 $p_1,p_2,\dots,p_k$ ，则我们只需考虑 $0\sim p_1p_2\dots p_k-1$ 的答案，后面部分的情况只是这部分的重复（因为我们只在乎区间里的数是否是 $p_1,p_2,\dots,p_k$ 的倍数）。

枚举长度 $i$ 时，相当于 $x,x+1,\dots,x+i-1$ 不能是任何一个 $p_j$ 的倍数，也就要求 $\bmod p_j$ 的取值只能是 $p_j-i,p_j-(i+1),\dots,1$ ，因此总共只有 $\max(p_j-i,0)$ 种选法。

而将这些选法相互组合，有 $\prod\limits_{j=1}^k\max(p_j-i,0)$ 种，而每种根据中国剩余定理都恰好在 $0\sim p_1p_2\dots p_k$ 中有一组解。

因此，只需求 $\sum\limits_{i=1}^{∞}\prod\limits_{j=1}^k\max(p_j-i,0)$ 。

而这里的 $i$ 显然只需从 $1$ 取到最小质因子减 $1$ 即可，因为更大的话这个乘积就已经是 $0$ 了。不妨设最小质因子是 $p_1$ 。

所以，当 $n$ 是 $1$ 时，直接输出 $1$ 。

当 $n$ 是质数时，上面这个式子就等于 $\sum\limits_{i=1}^{p_1-1} (p_1-i)$ ，也就是 $\frac{p_1(p_1-1)}{2}$ 。

否则，最小质因子不超过 $10^6$ ，枚举 $i$ 暴力进行上面 $\sum\limits_{i=1}^{∞}\prod\limits_{j=1}^k\max(p_j-i,0)$ 的计算即可。

时间复杂度，时间复杂度是 $\mathcal{O}(\sqrt{N})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 10 ** 6 + 5
    isPrime = [1] * M
    
    primes = []
    for i in range(2, M):
        if isPrime[i]:
            primes.append(i)
            for j in range(i, M, i):
                isPrime[j] = 0
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        
        if n == 1:
            outs.append(1)
            continue
        
        pr = []
        
        cur = n
        for i in primes:
            if i * i > cur: break
            if cur % i == 0:
            pr.append(i)
            while cur % i == 0:
                cur //= i
        
        if cur > 1: pr.append(cur)
        
        total = 0
        if len(pr) == 1: total = (pr[0] - 1) * pr[0] // 2
        else:
            for i in range(1, pr[0]):
                res = 1
                for p in pr:
                    res *= p - i
                total += res
        
        ans = n
        for p in pr:
            ans //= p
        
        outs.append(ans * total % 998244353)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 5;
	vector<int> isPrime(M, 1), primes;
	
	for (int i = 2; i < M; i ++) {
		if (isPrime[i]) {
			primes.emplace_back(i);
			for (int j = i; j < M; j += i) {
				isPrime[j] = 0;
			}
		}
	}

	int t, mod = 998244353;
	cin >> t;

	while (t --) {
		long long n;
		cin >> n;

		if (n == 1) {
			cout << 1 << '\n';
			continue;
		}

		vector<long long> pr;
		long long cur = n;

		for (auto &i: primes) {
			if (1ll * i * i > cur) break;
			if (cur % i == 0) {
				pr.emplace_back(i);
				while (cur % i == 0) {
					cur /= i;
				}
			}
		}

		if (cur > 1) pr.emplace_back(cur);
		
		long long total;

		if (pr.size() == 1)
			total = ((pr[0] - 1) % mod) * (pr[0] % mod) % mod * (mod + 1) / 2 % mod;
		else {
			total = 0;
			
			for (int i = 1; i < pr[0]; i ++) {
				long long res = 1;
				for (auto &x: pr) {
					res = res * (x - i) % mod;
				}
				total = (total + res) % mod;
			}
		}

		for (auto &p: pr) n /= p;
		cout << n * total % mod << '\n';
	}

	return 0;
}
```
