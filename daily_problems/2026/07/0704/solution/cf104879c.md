**提示 1：** 固定 $i,j$ 的时候选择的 $(i+a,j),(i,j+b)$ 需要满足什么条件？

**提示 2：** 如何统计？

固定 $(i,j)$ ，则 $(i+a,j)$ 的位置只能比它小 $a$ ，同时 $(i,j+b)$ 的位置只能比它小 $b$ 。

而满足这个条件时，也一定可以找到对应的 $\Delta$ ，因此只需统计每个点右侧满足条件的点数，乘以下面满足条件的点数即可。

右侧的点数，可以统计右侧 $i+grid[i][j]$ 出现的频率。下侧的点数，可以统计 $j+grid[i][j]$ 出现的频率。两者相乘即这个点的答案，对于所有 $i,j$ 求和即可。答案需要使用 long long 。

时间复杂度为 $\mathcal{O}(nm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, m = MII()
    grid = [LII() for _ in range(n)]
    
    rnd = random.getrandbits(30)
    
    cnt = [[0] * m for _ in range(n)]
    
    for i in range(n):
        d = Counter()
        for j in range(m - 1, -1, -1):
            cnt[i][j] = d[grid[i][j] + j + rnd]
            d[grid[i][j] + j + rnd] += 1
    
    ans = 0
    for j in range(m):
        d = Counter()
        for i in range(n - 1, -1, -1):
            ans += cnt[i][j] * d[grid[i][j] + i + rnd]
            d[grid[i][j] + i + rnd] += 1
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m;
	cin >> n >> m;

	vector<vector<int>> grid(n, vector<int>(m));
	for (auto &x: grid) for (auto &y: x) cin >> y;

	vector<vector<int>> cnt(n, vector<int>(m, 0));

	for (int i = 0; i < n; i ++) {
		map<int, int> d;
		for (int j = m - 1; j >= 0; j --) {
			cnt[i][j] = d[grid[i][j] + j];
			d[grid[i][j] + j] ++;
		}
	}

	long long ans = 0;

	for (int j = 0; j < m; j ++) {
		map<int, int> d;
		for (int i = n - 1; i >= 0; i --) {
			ans += cnt[i][j] * d[grid[i][j] + i];
			d[grid[i][j] + i] ++;
		}
	}

	cout << ans;

	return 0;
}
```
