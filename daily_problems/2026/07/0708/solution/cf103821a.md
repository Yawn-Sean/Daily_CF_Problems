**提示 1：** 模拟。

考虑从下往上扫描每一条横线。

考虑当前能找到的下标，则我们的操作是怎么样呢？

- 如果某一块没有任何被找到的下标，直接跳过。

- 否则，把照到它的下标全部删除。然后加上这一块的端点。

这件事儿可以用平衡树 / 线段树维护。

下面代码给的是线段树的。用 $1$ 表示能照亮的位置。前者，查询区间最大值 / 区间和。后者，二分找到左端点后第一个非 $0$ 的位置，不断将这个位置赋值 $0$ ，直到区间内没有 $1$ ，最后在端点赋值。

这么做复杂度为什么对呢？

每个加进去的点最多被删 $1$ 次，而总共加进去的点的数量级是 $\mathcal{O}(n+2q)=\mathcal{O}(n+q)$ 的，同时加点和删点复杂度都是 $\mathcal{O}(\log n)$ 的。

时间复杂度为 $\mathcal{O}((n+q)\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, q = MII()
        tmp = [[] for _ in range(n)]
        
        for i in range(q):
            x1, x2, y = MII()
            x1 -= 1
            x2 -= 1
            tmp[y].append((x1, x2))
        
        seg = SegTree(fmax, 0, [1] * n)
        
        for i in range(n):
            for l, r in tmp[i]:
                if seg.prod(l, r + 1):
                    while True:
                        p = seg.max_right(l, lambda x: x == 0)
                        if p <= r: seg.set(p, 0)
                        else: break
                    seg.set(l, 1)
                    seg.set(r, 1)
        
        outs.append(sum(seg.get(i) for i in range(n)))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int op(int x, int y) {return max(x, y);}

int e() {return 0;}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, q;
		cin >> n >> q;

		vector<vector<pair<int, int>>> tmp(n);
		
		while (q --) {
			int x1, x2, y;
			cin >> x1 >> x2 >> y;
			x1 --, x2 --;
			tmp[y].emplace_back(x1, x2);
		}

		vector<int> nums(n, 1);
		atcoder::segtree<int, op, e> seg(nums);

		for (int i = 0; i < n; i ++) {
			for (auto &[l, r]: tmp[i]) {
				if (seg.prod(l, r + 1)) {
					while (true) {
						int p = seg.max_right(l, [&] (int x) {return x == 0;});
						if (p <= r) seg.set(p, 0);
						else break;
					}
					seg.set(l, 1);
					seg.set(r, 1);
				}
			}
		}

		int ans = 0;
		for (int i = 0; i < n; i ++) ans += seg.get(i);

		cout << ans << '\n';
	}

	return 0;
}
```
