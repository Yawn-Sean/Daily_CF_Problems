**提示 1：** 当第一个半径确定时，剩余所有元素也都确定了。

**提示 2：** 第一个元素的备选方案？

首先，第一个半径如果是 $r_1=r$ ，则第二个半径是 $r_2=x_2-x_1-r_1$ ，类似地可以推出所有半径。

那如何确定第一个元素呢？

实际上我们可以找到一些特征。

上述推出的每个半径都是 $r$ 的一次函数，系数要么是 $1$ ，要么是 $-1$ 。所以半径序列可以分为两类： $r+b$ 和 $-r+b$ 。

而整个半径序列的最小值等于 $R=\min r_i$ ，所以，分别提取两个半径序列的最小值，分别令其等于 $R$ 可以解出两个不同的初始半径，分别检查是否满足要求即可。

时间复杂度为 $\mathcal{O}(n)$ ，如果你用线性方法判断两个数组是否构成元素相等。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    xs = LII()
    rs = LII()
    
    v_pos = [0]
    v_neg = []
    
    for i in range(1, n):
        if i % 2: v_neg.append(xs[i] - xs[i - 1] - v_pos[-1])
        else: v_pos.append(xs[i] - xs[i - 1] - v_neg[-1])
    
    rs.sort()
    min_r = rs[0]
    
    def check(x):
        tmp = [x]
        for i in range(1, n):
            tmp.append(xs[i] - xs[i - 1] - tmp[-1])
        
        if sorted(tmp) == rs:
            exit(print(' '.join(map(str, tmp))))
    
    v_mi = min(v_pos)
    check(min_r - v_mi)
    
    v_mi = min(v_neg)
    check(v_mi - min_r)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<long long> xs(n), rs(n);
	for (auto &x: xs) cin >> x;
	for (auto &r: rs) cin >> r;

	vector<long long> v_pos = {0}, v_neg;

	for (int i = 1; i < n; i ++) {
		if (i & 1) v_neg.emplace_back(xs[i] - xs[i - 1] - v_pos.back());
		else v_pos.emplace_back(xs[i] - xs[i - 1] - v_neg.back());
	}

	sort(rs.begin(), rs.end());

	auto check = [&] (long long x) -> void {
		vector<long long> tmp = {x};
		for (int i = 1; i < n; i ++) tmp.emplace_back(xs[i] - xs[i - 1] - tmp.back());

		auto to_check = tmp;
		sort(to_check.begin(), to_check.end());

		for (int i = 0; i < n; i ++) {
			if (to_check[i] != rs[i]) {
				return ;
			}
		}

		for (int i = 0; i < n; i ++) cout << tmp[i] << " \n"[i == n - 1];
		exit(0);
	};

	long long min_r = rs[0];
	long long v_mi;

	v_mi = *min_element(v_pos.begin(), v_pos.end());
	check(min_r - v_mi);
	v_mi = *min_element(v_neg.begin(), v_neg.end());
	check(v_mi - min_r);

	return 0;
}
```
