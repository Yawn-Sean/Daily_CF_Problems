**提示 2：** 能构造的情况下调整目标是什么？如何达到？

首先，因为最后的结果需要两组数的乘积相等，而两组数乘积之积不变，所以这个数需要是完全平方数。

中间的数都出现两遍，不影响是否是完全平方数，所以只需最后剩下的 $n+1$ 是完全平方数。

如果不是，则无法构造。

如果是，设 $n+1=k^2$ ，我们需要让第一个序列变成 $k$ 倍，每次的操作是交换对应元素，交换第 $i$ 个元素可以变成 $\frac{i+1}{i}$ 倍。

而 $k=\frac{2}{1}\frac{3}{2}\dots\frac{k}{k-1}$ ，因此只需交换前 $k-1$ 项即可。

时间复杂度为 $\mathcal{O}(\sqrt{n})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        n += 1
        
        v = math.isqrt(n)
        if v * v == n:
            outs.append(str(v - 1))
            outs.append(' '.join(map(str, range(1, v))))
        else:
            outs.append('-1')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n; n ++;

		int v = sqrt(n);
		if (v * v == n) {
			cout << v - 1 << '\n';
			for (int i = 1; i < v; i ++) cout << i << ' ';
			cout << '\n';
		}
		else cout << -1 << '\n';
	}

	return 0;
}
```
