**提示 1：** 枚举 $b$ ， $a$ 要满足什么条件？其数量如何计算？

枚举 $b$ 的话，因为 $ab=b^2-(b-a)b$ ，而 $b-a$ 整除于 $(b-a)b$ ，所以相当于需要 $b-a$ 是 $b^2$ 的因子，同时 $1\leq b-a\lt b$ 。

所以我们需要求 $b^2$ 的小于 $b$ 的因子个数。

而将 $b^2$ 的因子中乘积为 $b^2$ 的两两配对，总是一个小于 $b$ 一个大于 $b$ ，只有 $b$ 是单独的。所以数量等于 $b^2$ 的因子数量减 $1$ 再除以 $2$ 。

而因子数量可以用每个质因子的次数加一的乘积来计算， $b^2$ 的质因数分解可以由 $b$ 的质因数分解结果得到。因此只要能得到 $1\sim 2\times 10^6$ 中每个数的质因数分解即可。

于是素数筛，在每个元素位置标记其最小质因子即可。

最后查询相当于是一个前缀和，预处理后可以 $\mathcal{O}(1)$ 提取。

时间复杂度为 $\mathcal{O}(M\log\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 2 * 10 ** 6 + 1
    pr = list(range(M))
    
    for i in range(2, M):
        if pr[i] == i:
            for j in range(i, M, i):
                pr[j] = i
    
    cnt = [0] * M
    
    for i in range(1, M):
        cur = i
        res = 1
    
        while cur > 1:
            p = pr[cur]
            c = 1
            
            while cur % p == 0:
                cur //= p
                c += 2
            
            res *= c
        
        cnt[i] = res // 2
    
    for i in range(1, M):
        cnt[i] += cnt[i - 1]
    
    t = II()
    outs = []
    
    for _ in range(t):
        outs.append(cnt[II()])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 2e6 + 5;

	vector<int> pr(M);
	iota(pr.begin(), pr.end(), 0);

	for (int i = 2; i <= M; i ++) {
		if (pr[i] == i) {
			for (int j = i; j <= M; j += i) {
				pr[j] = i;
			}
		}
	}

	vector<int> cnt(M, 0);

	for (int i = 2; i < M; i ++) {
		int cur = i, res = 1;
		while (cur > 1) {
			int p = pr[cur], c = 0;
			while (cur % p == 0) cur /= p, c ++;
			res *= 2 * c + 1;
		}
		cnt[i] += res / 2;
	}

	for (int i = 1; i < M; i ++) cnt[i] += cnt[i - 1];

	int t;
	cin >> t;

	while (t --) {
		int n; cin >> n;
		cout << cnt[n] << '\n';
	}

	return 0;
}
```
