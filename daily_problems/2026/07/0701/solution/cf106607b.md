**提示 1：** 两次操作显然够用。怎么讨论？

首先，直接对整个数组进行一次操作，整个数组就变成同一个数了，再操作一次，就可以让整个数组都变成 $1$ ，所以最多只需要两次操作。

那其他情况呢？

$0$ 次操作：显然直接一开始就全都是 $1$ 。

$1$ 次操作：操作的子数组只能有一种数，不然的话后续的赋值会出现非 $1$ 的数。而除了操作的部分，剩余元素不变也就是都是 $1$ 。因此就是中间只有一段相同的非 $1$ 的数。

判断可以写成找到中间包含非 $1$ 数区间的起点和终点，看中间元素是否相同。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
    
        l = n
        r = -1
        for i in range(n):
            if nums[i] != 1:
                l = fmin(l, i)
                r = fmax(r, i)
        
        if l > r: outs.append(0)
        else:
            flg = True
            for i in range(l, r + 1):
                if nums[i] != nums[l]:
                    flg = False
            
            outs.append(1 if flg else 2)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;
		
		vector<int> nums(n);
		for (auto &v: nums) cin >> v;

		int l = n, r = -1;
		for (int i = 0; i < n; i ++) {
			if (nums[i] != 1) {
				l = min(l, i);
				r = max(r, i);
			}
		}

		if (l > r) cout << 0 << '\n';
		else {
			bool flg = true;
			for (int i = l; i <= r; i ++) {
				if (nums[i] != nums[l]) {
					flg = false;
				}
			}
			cout << (flg ? 1 : 2) << '\n';
		}
	}

	return 0;
}
```
