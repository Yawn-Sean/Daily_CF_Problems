**提示 1：** 容斥原理。

考虑循环节的个数。

我们考虑循环节的个数如果是一个质数 $p$ 的倍数，则 $\frac{n}{p}$ 是循环节，对应的方案数是 $2^{\frac{n}{p}}$ 个。

考虑完所有 $n$ 的质因子，将这些情况相加。我们多算了什么呢？循环节是两个或更多个质数的倍数。于是去掉这些。

此时你应该意识到这里就是一个容斥原理了，因此枚举所有质因子的组合就行。奇数个质因子对应的容斥系数是 $1$ ，否则是 $-1$ 。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 10 ** 6 + 5
    mod = 10 ** 9 + 7
    
    pr = list(range(M))
    phi = [1] * M
    
    phi[0] = 0
    
    for i in range(2, M):
        if pr[i] == i:
            for j in range(i, M, i):
                pr[j] = i
                if j % (i * i) == 0:
                    phi[j] = 0
                else:
                    phi[j] = -phi[j]
    
    cur = 1
    ans = [0] * M
    
    for i in range(1, M):
        cur = cur * 2 % mod
        
        for j in range(2 * i, M, i):
            ans[j] += (-phi[j // i]) * cur % mod
            ans[j] %= mod
    
    t = II()
    outs = []
    
    for _ in range(t):
        outs.append(ans[II()])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 5, mod = 1e9 + 7;

	vector<int> pr(M), phi(M, 1);
	iota(pr.begin(), pr.end(), 0);
	phi[0] = 0;

	for (int i = 2; i < M; i ++) {
		if (pr[i] == i) {
			for (int j = i; j < M; j += i) {
				pr[j] = i;
				if (j % i == 0 && j / i % i == 0) phi[j] = 0;
				else phi[j] = -phi[j];
			}
		}
	}

	vector<int> ans(M, 0);

	int cur = 1;
	for (int i = 1; i < M; i ++) {
		cur = 2ll * cur % mod;
		for (int j = i * 2; j < M; j += i) {
			ans[j] += (-phi[j / i]) * cur % mod;
			if (ans[j] >= mod) ans[j] -= mod;
			if (ans[j] < 0) ans[j] += mod;
		}
	}

	int t;
	cin >> t;

	while (t --) {
		int x; cin >> x;
		cout << ans[x] << '\n';
	}

	return 0;
}
```
