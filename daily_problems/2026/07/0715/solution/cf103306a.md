**提示 1：** 考虑某个子集形成 $y$ 个连通块的方案数作为 DP 计数的目标。怎么转移？

设 $dp[i][msk]$ 表示 $msk$ 形成 $i$ 个连通块的方案数。

如果 $i\geq 2$ ，则可以理解为 $i-1$ 个连通块和 $1$ 个连通块拼到一起。

为了后者那一个连通块的选择是唯一的，我们钦定其包含 $msk$ 中的最小元素。

我们枚举包含最小元素的子集 $msk'$ ，则需要将所有 $dp[i-1][msk-msk']\times dp[1][msk']$ 全部加过去。

$i=1$ 咋办？

直接用总方案数，也就是 $2^{|msk 中边的个数|}$ ，减去其他所有连通块数量的方案数得到（枚举 $2,3,\dots, n$ 即可）。

整体包含 $\mathcal{O}(2^n)$ 维度的枚举子集，这一部分是 $\mathcal{O}(3^n)$ 的，再乘以另一个维度就是总复杂度了。

时间复杂度为 $\mathcal{O}(n3^n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, m = MII()
    
    mod = 998244353
    cnt = [1] * (1 << n)
    
    for _ in range(m):
        u, v = GMI()
        
        for i in range(1 << n):
            if i >> u & 1 and i >> v & 1:
                cnt[i] = cnt[i] * 2 % mod
    
    dp = [[0] * (1 << n) for _ in range(n + 1)]
    
    for i in range(1, 1 << n):
        bit = 1 << i.bit_length() - 1
        cur = i
        
        while cur >= bit:
            for j in range(2, n + 1):
                dp[j][i] += dp[1][cur] * dp[j - 1][i - cur] % mod
                dp[j][i] %= mod
            cur = (cur - 1) & i
        
        dp[1][i] = cnt[i]
        for j in range(2, n + 1):
            dp[1][i] -= dp[j][i]
            dp[1][i] %= mod
    
    print('\n'.join(str(dp[i][(1 << n) - 1]) for i in range(1, n + 1)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m;
	cin >> n >> m;

	int mod = 998244353;
	vector<int> cnt(1 << n, 1);

	auto add = [&] (int &x, int y) -> void {
		x += y;
		if (x >= mod) x -= mod;
	};

	while (m --) {
		int u, v;
		cin >> u >> v;
		u --, v --;
		
		for (int i = 0; i < (1 << n); i ++) {
			if ((i >> u & 1) && (i >> v & 1)) {
				add(cnt[i], cnt[i]);
			}
		}
	}

	vector<vector<int>> dp(n + 1, vector<int>(1 << n, 0));

	for (int i = 1; i < (1 << n); i ++) {
		int bit = 1 << 31 - __builtin_clz(i);
		int cur = i;

		while (cur & bit) {
			for (int j = 2; j <= n; j ++)
				add(dp[j][i], 1ll * dp[1][cur] * dp[j - 1][i - cur] % mod);
			cur = (cur - 1) & i;
		}

		dp[1][i] = cnt[i];
		for (int j = 2; j <= n; j ++) add(dp[1][i], mod - dp[j][i]);
	}

	for (int i = 1; i <= n; i ++) cout << dp[i].back() << '\n';

	return 0;
}
```
