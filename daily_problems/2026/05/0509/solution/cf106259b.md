**提示 1：** 实际上如果结果非 $0$ ，被除的数的乘积不会很大，也就不会选很多非 $1$ 数。

**提示 2：** 如何设计非 $1$ 数的子序列，来统计整体计数。

首先看提示 1，我们把所有 $x_2,\dots, x_k$ 取出来，如果乘积大于 $n$ 了，那么最终结果就会变成 $0$ ，对答案毫无影响。

而如果乘积是 $V$ ，则最终结果是 $\lfloor x_1/V\rfloor$ 。

同时因为乘积不超过 $n$ ，所以要让非 $1$ 的数的乘积不超过 $n$ ，数的个数不超过 $\mathcal{O}(\log n)$ 个。

考虑 DP， $dp[i][x]$ 表示选取 $i$ 个数使得乘积为 $x$ 的方案数，则 $i$ 的数据范围是 $\mathcal{O}(\log n)$ 的。

而这里的转移，对于同一个 $i$ 中某个固定的 $x$ ，只需枚举子序列中的下一个元素，转移到 $1\times x,\dots,\lfloor n/x\rfloor\times x$ 即可。因此对于同一个 $i$ 而言，总转移次数是 $\mathcal{O}(n\log n)$ 的。

而这个子序列确定后，需要放入到 $k-1$ 个位置中的 $i$ 个，因此，后面 $k-1$ 个数中有 $i$ 个非 $1$ 数且乘积为 $x$ 的方案数是 $C_{k-1}^i dp[i][x]\times cnt_1^{k-1-i}$ ，其中 $cnt_1$ 表示 $1$ 的个数。

我们进一步可以统计所有 $x_2,\dots,x_k$ 乘积为 $V$ 的序列个数。设为 $calc[V]$

接下来，我们要求的就是 $\sum\limits_{x_1=a_1,\dots,a_n}\sum\limits_{V=1}^n\lfloor x_1/V\rfloor calc[V]$ 。

我们枚举 $V$ 和对应的 $\lfloor x_1/V\rfloor$ ，我们可以快速统计出满足条件的 $a_1,\dots, a_n$ 中的元素个数。而 $\lfloor x_1/V\rfloor$ 的求和是 $\mathcal{O}(n\log n)$ 数量级的，因此问题解决。

时间复杂度为 $\mathcal{O}(n\log^2 n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    def f(x1, x2):
        x = abs(x1 - x2)
        if x == 0: return x
        return (x - 1) % 2 + 1
    
    for _ in range(t):
        n, m = MII()
        r1, c1, r2, c2 = MII()
        outs.append(f(r1, r2) + f(c1, c2))
    
    print('\n'.join(map(str, outs)))
```
