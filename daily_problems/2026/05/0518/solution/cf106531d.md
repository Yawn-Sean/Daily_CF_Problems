**提示 1：** 想想每个元素的出现概率是什么？

**提示 2：** 相当于切东西，分东西。

考虑每首歌的选取概率。假设在每一组里面的出现概率分别是 $\frac{x_i}{K+1}$ ，则总出现概率就是 $\frac{\sum x_i}{n(K+1)}$ 。

而我们要最终出现概率是 $\frac{P_i}{\sum P_i}$ ，那么不妨让 $\sum P_i=K$ 。

接下来，把所有概率和想要达到的概率都乘以 $K+1$ 进行考虑。

每个下标对应的概率现在是 $K+1$ ，因此我们的总概率池是 $n(K+1)$ 。对于 $P_i$ ，我们要分配 $nP_i$ 的概率池给它。

相当于是有 $n$ 个大小为 $K+1$ 的物品，每个分成两部分，分配给 $1\sim n$ 中的每个人，最后每个人分别拿到 $P_1,P_2,\dots,P_n$ 。

唯一的比较不好的限制就是每个物品只能分成两部分。怎么办呢？

考虑每次解决至少一个人的需求。我们选取需求量最少的那个人，则其需求一定不超过总需求的均值，即 $K+1$ 。

如果需求刚好是 $K+1$ ，那么就完全分配给他，解决了问题。

否则，此时数组中一定存在一个需求至少为 $K+1$ 的（因为均值也就是 $K+1$ ），将他俩配对，先切一部分给第一个人，剩余部分给第二个人。这样也解决了一个人的问题。

上述两种情况都解决了一个人的需求。

因此 $n$ 个物品分给 $n$ 个人的问题可以转化为 $n-1$ 个物品分给 $n-1$ 个人的问题，因此根据数学归纳法，构造完成。

时间复杂度为 $\mathcal{O}(n^2)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    n = II()
    nums = LII()
    
    k = sum(nums)
    
    for i in range(n):
        nums[i] *= n
    
    print(k - 1)
    
    v1 = [0] * n
    v2 = [0] * n
    w = [0] * n
    
    for i in range(n):
        mi_idx = -1
        
        for j in range(n):
            if nums[j] and (mi_idx == -1 or nums[j] < nums[mi_idx]):
                mi_idx = j
        
        if nums[mi_idx] >= k:
            v1[i] = mi_idx
            v2[i] = mi_idx
            w[i] = 0
        else:
            to_pair = -1
            for j in range(n):
                if j != mi_idx and nums[j] and nums[mi_idx] + nums[j] >= k:
                    to_pair = j
            
            v1[i] = mi_idx
            v2[i] = to_pair
            w[i] = nums[mi_idx]
        
        nums[v1[i]] -= w[i]
        nums[v2[i]] -= k - w[i]
    
    print(*(x + 1 for x in v1))
    print(*(x + 1 for x in v2))
    print(*w)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<long long> nums(n);
	for (auto &x: nums) cin >> x;

	long long k = 0;
	for (auto &v: nums) {
		k += v;
		v *= n;
	}

	cout << k - 1 << '\n';

	vector<int> v1(n), v2(n);
	vector<long long> w(n);

	for (int i = 0; i < n; i ++) {
		int mi_idx = -1;

		for (int j = 0; j < n; j ++) {
			if (nums[j] && (mi_idx == -1 || nums[j] < nums[mi_idx])) {
				mi_idx = j;
			}
		}

		if (nums[mi_idx] >= k) {
			v1[i] = mi_idx;
			v2[i] = mi_idx;
			w[i] = 0;
		}
		else {
			int to_pair = -1;

			for (int j = 0; j < n; j ++) {
				if (j != mi_idx && nums[j] && nums[mi_idx] + nums[j] >= k) {
					to_pair = j;
				}
			}

			v1[i] = mi_idx;
			v2[i] = to_pair;
			w[i] = nums[mi_idx];
		}

		nums[v1[i]] -= w[i];
		nums[v2[i]] -= k - w[i];
	}

	for (int i = 0; i < n; i ++)
		cout << v1[i] + 1 << " \n"[i == n - 1];
	for (int i = 0; i < n; i ++)
		cout << v2[i] + 1 << " \n"[i == n - 1];
	for (int i = 0; i < n; i ++)
		cout << w[i] << " \n"[i == n - 1];

	return 0;
}
```
