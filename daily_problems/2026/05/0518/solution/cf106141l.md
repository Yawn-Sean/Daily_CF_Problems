**提示 1：** 首先，确定正负号。

**提示 2：** 选哪些乘以 $3$ 倍呢？

首先，我们假设所有数的绝对值都确定了，那么如何选定正负号呢？

跟原数组一致即可，因为这样会使得分子尽可能大，而分母保持不变。

接下来，就是决定每个元素到底选择 $1$ 还是 $3$ 作为其绝对值了。

假设我们已经决定了有 $k$ 个数是 $3$ 作为绝对值，那么分母又完全确定了。我们如何选取分子呢？因为分子中，数组中每个元素的权重就是其绝对值乘以 $1$ 或 $3$ ，因此肯定选取绝对值最大的 $k$ 个最好。

因此对于某个 $k$ ，计算绝对值最大的 $k$ 个元素（排序后就是前缀和，因此排序后整体只需要 $\mathcal{O}(n)$ 复杂度）对分子的影响，就可以得到 $k$ 对应的答案。于是枚举不同的 $k$ 取最大值即可。

因为带根号和分母，所以可以平方去根号再通分去分母来实现无精度误差的判断。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    n = II()
    nums = LII()
    
    vals = [1 if x >= 0 else -1 for x in nums]
    st_range = sorted(range(n), key=lambda x: -abs(nums[x]))
    
    total = sum(abs(x) for x in nums)
    val = n
    
    ma_total = total
    ma_val = n
    chosen = 0
    
    for i in range(n):
        total += 2 * abs(nums[st_range[i]])
        val += 8
        
        if total * total * ma_val > ma_total * ma_total * val:
            ma_total = total
            ma_val = val
            chosen = i + 1
    
    for i in range(chosen):
        vals[st_range[i]] *= 3
    
    print(*vals)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	vector<int> vals(n);
	int total = 0, val = n;
	for (int i = 0; i < n; i ++) {
		vals[i] = (nums[i] >= 0 ? 1 : -1);
		total += abs(nums[i]);
	}

	vector<int> order(n);
	iota(order.begin(), order.end(), 0);
	sort(order.begin(), order.end(), [&] (int i, int j) {return abs(nums[i]) > abs(nums[j]);});

	int ma_total = total, ma_val = val, chosen = -1;

	for (int i = 0; i < n; i ++) {
		total += 2 * abs(nums[order[i]]);
		val += 8;

		if (1ll * total * total * ma_val > 1ll * ma_total * ma_total * val) {
			ma_total = total;
			ma_val = val;
			chosen = i;
		}
	}

	for (int i = 0; i <= chosen; i ++) vals[order[i]] *= 3;

	for (auto &x: vals) cout << x << ' ';

	return 0;
}
```
