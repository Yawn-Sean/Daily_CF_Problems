**提示 1：** 如果选了两条及以上的街道？

**提示 2：** 枚举什么？

首先，如果选取了两条街道——

- 如果两条街道是垂直的，可以同时去掉一段变成纯的横着 / 竖着走。

- 如果两条街道是平行的，可以把其中一段平移回来。

所以最多只经过一个街道。

我们先计算不用街道的情况下的成本。

接下来，由于进入街道前没必要拐弯，街道可能进入的位置和出去的位置都是 $\mathcal{O}(1)$ 的，所以枚举即可。

时间复杂度为 $\mathcal{O}(M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    def dis(x1, y1, x2, y2):
        return abs(x1 - x2) + abs(y1 - y2)
    
    ax, ay, bx, by = MII()
    ans = dis(ax, ay, bx, by)
    
    t = II()
    for _ in range(t):
        b, k = MII()
        
        for x1 in [ax, (ay - b) * k]:
            y1 = k * x1 + b
            for x2 in [bx, (by - b) * k]:
                y2 = k * x2 + b
                
                ans = fmin(ans, dis(ax, ay, x1, y1) + abs(x1 - x2) + dis(x2, y2, bx, by))
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	auto dis = [&] (long long x1, long long y1, long long x2, long long y2) -> long long {
		return abs(x1 - x2) + abs(y1 - y2);
	};

	long long ax, ay, bx, by;
	cin >> ax >> ay >> bx >> by;

	long long ans = dis(ax, ay, bx, by);

	int t;
	cin >> t;

	while (t --) {
		int k;
		long long b;
		cin >> b >> k;

		for (auto x1: {ax, (ay - b) * k}) {
			for (auto x2: {bx, (by - b) * k}) {
				long long y1 = k * x1 + b;
				long long y2 = k * x2 + b;

				ans = min(ans, dis(ax, ay, x1, y1) + abs(x1 - x2) + dis(x2, y2, bx, by));
			}
		}
	}

	cout << ans;

	return 0;
}
```
