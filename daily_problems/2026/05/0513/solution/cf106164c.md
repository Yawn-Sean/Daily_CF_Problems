**提示 1：** 先找到最小的 $i$ 。怎么构造可以避开分类讨论？

首先，用 $1,2,\dots,N$ 能凑出来的数的数据范围的最小值是 $1-2-3-\dots N$ ，最大值是 $1+2+\dots+N$ ，所以范围是 $[2-\frac{N(N+1)}{2},\frac{N(N+1)}{2}]$ 。

同时设取负数的集合的和是 $X$ ，则整体的和是 $\frac{N(N+1)}{2}-2X$ ，所以 $X$ 只能跟 $\frac{N(N+1)}{2}$ 奇偶性相同。

除此之外，我们选取的子集不能包含 $1$ ，所以子集和不能是 $1$ 。类似地， $2+3+\dots+N-1$ 的这个子集和也是凑不出来的。

于是我们枚举 $N$ ，直到上述条件不会被判为非法，并确定我们需要凑的要取负数的子集和。

因为整体构造的发挥空间很大，可以自行尝试可行的做法。下面给出一种。

因为我们要留着小数给最后调整，所以我们从大到小遍历 $N$ 到 $2$ 的数，能放到要选的集合中就放进去。

但这样可能导致非法，因为可能出现最后剩下一个 $1$ 。在这种情况下，我们选取的数一定形成一个后缀（不然我们可以将其中某个数 $x$ 增大 $1$ 且 $x+1$ 没有被选到，而这就意味着我们到 $x+1$ 时本应选上  $x+1$ 却没选），且最小数不小于 $4$ 。于是删去我们选的最小的数 $y$ ，改选 $y-1$ 和 $2$ 即可。

时间复杂度为 $\mathcal{O}(\sqrt{X})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        
        for i in range(1, 1000):
            v = i * (i + 1) // 2
            
            if 2 - v <= n <= v and (v - n) % 2 == 0 and n != v - 2 and n != 4 - v:
                outs.append(str(i))
                
                chosen = [0] * (i + 1)
                target = (v - n) // 2
                
                for x in range(i, 1, -1):
                    if target >= x and target - x != 1:
                        target -= x
                        chosen[x] = 1
                
                outs.append('1' + ''.join("+-"[chosen[j]] + str(j) for j in range(2, i + 1)))
                break
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		for (int i = 1; i <= 1000; i ++) {
			int v = i * (i + 1) / 2;
			if (n <= v && n >= 2 - v && (n - v) % 2 == 0 && n != v - 2 && n != 4 - v) {
				vector<int> sign(i + 1, 0);
				int target = (v - n) / 2;
				
				for (int x = i; x > 1; x --) {
					if (target >= x && target - x != 1) {
						target -= x;
						sign[x] = 1;
					}
				}

				cout << i << '\n';
				cout << 1;
				for (int x = 2; x <= i; x ++)
					cout << "+-"[sign[x]] << x;
				cout << '\n';
				break;
			}
		}
	}

	return 0;
}
```
