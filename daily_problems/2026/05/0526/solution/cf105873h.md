**提示 1：** 任务越早完成越好。

**提示 2：** 我们只在乎当前完成了什么任务。

首先，任务早点完成没坏处。同时每个任务都可以在可行集内尽早完成。

而 $n=20$ 提示我们可以使用目前已经完成的任务作为状态。

此时我们枚举新任务。我们可以看当前任务集的总用时是否超过了这个新任务的截止时间，来判断这个任务能否获得初始的 $p$ ，如果可以再考虑新增益。

如果获得了增益，则增益期望数值显然是 $\frac{lx+rx}{2}$ 。而如果完成时间为 $T$ ，若 $T\le ly$ ，则一定能获得增益；若 $T\ge ry$ ，则一定无法获得增益；否则，以 $\frac{ry-T}{ry-ly}$ 的概率获得增益。

因此新任务的期望收益确定，接下来就是一个典型的 bitmask DP 问题。

时间复杂度为 $\mathcal{O}(n2^n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    ds = []
    ps = []
    cs = []
    lxs = []
    rxs = []
    lys = []
    rys = []
    
    for _ in range(n):
        d, p, c, lx, rx, ly, ry = MII()
        ds.append(d)
        ps.append(p)
        cs.append(c)
        lxs.append(lx)
        rxs.append(rx)
        lys.append(ly)
        rys.append(ry)
    
    dp = [0] * (1 << n)
    cur_time = [0] * (1 << n)
    
    for i in range(1, 1 << n):
        for j in range(n):
            if i >> j & 1:
                oi = i - (1 << j)
                cur_time[i] = cur_time[oi] + cs[j]
                
                nval = dp[oi]
                if cur_time[i] <= ds[j]:
                    nval += ps[j]
                    if cur_time[i] <= lys[j]:
                        nval += (lxs[j] + rxs[j]) / 2
                    elif cur_time[i] <= rys[j]:
                        nval += (lxs[j] + rxs[j]) / 2 * (rys[j] - cur_time[i]) / (rys[j] - lys[j])
                
                dp[i] = fmax(dp[i], nval)
    
    print(dp[-1])
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<array<int, 7>> items(n);
	for (auto &[d, p, c, lx, rx, ly, ry]: items)
		cin >> d >> p >> c >> lx >> rx >> ly >> ry;

	vector<long double> dp(1 << n, 0);
	vector<long long> cur_time(1 << n, 0);

	for (int i = 0; i < (1 << n); i ++) {
		for (int j = 0; j < n; j ++) {
			if (i >> j & 1) {
				int oi = i ^ (1 << j);
				auto [d, p, c, lx, rx, ly, ry] = items[j];

				cur_time[i] = cur_time[oi] + c;
				long double nval = dp[oi];

				if (cur_time[i] <= d) {
					nval += p;
					if (cur_time[i] <= ly)
						nval += (long double)(lx + rx) / 2;
					else if (cur_time[i] <= ry)
						nval += (long double)(lx + rx) / 2 * (ry - cur_time[i]) / (ry - ly);
				}

				dp[i] = max(dp[i], nval);
			}
		}
	}

	cout << fixed << setprecision(15) << dp.back();

	return 0;
}
```
