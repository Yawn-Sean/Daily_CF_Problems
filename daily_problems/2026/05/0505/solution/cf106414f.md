**提示 1：** 如果三元和的数值能较为连续的变化，那么就一定可以经理中间的 $[m,2m]$ 区间。

**提示 2：** 如果不行，说明有坏数。

本题直觉主要来自于提示 1 。

如果所有数值都是 $[1,m)$ 之间，那么我把一个三元组中的一个数替换为另一个数，数值不会变动超过 $m$ 。

于是我们找到所有不超过 $m$ 的元素，如果最大的三元组和小于 $m$ 或最小的三元组和大于 $2m$ ，则显然无法构造。否则，我们可以从最小的三元组不断替换元素变成最大的三元组，中间必然经过 $[m,2m]$ 的数。

否则，我们只能选不小于 $m$ 的数了。如果我们选了 $2$ 个或以上，显然不行。所以只能选一个。而此时已经满足了不小于 $m$ 的条件，要更好地满足不超过 $2m$ 的条件，我们干脆选最小的两个数一集不小于 $m$ 的最小的一个数进行判断，看其是否满足要求即可。

时间复杂度为 $\mathcal{O}(n\log n)$ ，当然如果你维护比 $m$ 小的前三大和前三小可以达到 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        nums = LII()
        
        order = sorted(range(n), key=lambda x: nums[x])
        
        ans = []
        bound = n
        
        for i in range(2, n):
            if nums[order[i]] >= m:
                if nums[order[0]] + nums[order[1]] + nums[order[i]] <= 2 * m:
                    ans = [order[0], order[1], order[i]]
                bound = i
                break
        
        if not ans and bound >= 3:
            for i in range(4):
                tmp = []
                for j in range(i):
                    tmp.append(order[j])
                for j in range(3 - i):
                    tmp.append(order[bound - 1 - j])
                
                total = 0
                for x in tmp:
                    total += nums[x]
                
                if m <= total <= 2 * m:
                    ans = tmp
                    break
        
        if ans: outs.append(' '.join(str(x + 1) for x in ans))
        else: outs.append('-1')
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, m;
		cin >> n >> m;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;

		vector<int> order(n);
		iota(order.begin(), order.end(), 0);
		sort(order.begin(), order.end(), [&] (int i, int j) {return nums[i] < nums[j];});

		int bound = n;
		vector<int> ans;

		for (int i = 2; i < n; i ++) {
			if (nums[order[i]] >= m) {
				if (nums[order[0]] + nums[order[1]] + nums[order[i]] <= 2 * m)
					ans = {order[0], order[1], order[i]};
				bound = i;
				break;
			}
		}
		
		if (ans.empty() && bound >= 3) {
			for (int i = 0; i <= 3; i ++) {
				vector<int> tmp;
				for (int j = 0; j < i; j ++) tmp.emplace_back(order[j]);
				for (int j = 0; j < 3 - i; j ++) tmp.emplace_back(order[bound - 1 - j]);

				int total = 0;
				for (auto &x: tmp) total += nums[x];

				if (total >= m && total <= 2 * m) {
					ans = tmp;
					break;
				}
			}
		}

		if (ans.empty()) cout << -1 << '\n';
		else cout << ans[0] + 1 << ' ' << ans[1] + 1 << ' ' << ans[2] + 1 << '\n';
	}

	return 0;
}
```
