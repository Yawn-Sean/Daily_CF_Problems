**提示 1：** 每个矩形应该怎么放？

**提示 2：** 枚举高。

首先，一定是短边拼到一起。不然的话如果有一个选的是长边，那么高只能是短边，此时对应的答案最大只能是短边，不优于把短边拼起来的情况。

而此时，可以枚举不拼到一起的那条边的长度。此时我们可以选取长边比这个长度长的所有矩形，因此另一条边的长度就是这些矩形的短边和。能拼出来的最大长边就是这两者取最小值。

因为要枚举长边大于等于某个数时的短边的和，因此可以根据长边长度进行倒序排序，这样取的就是前缀了。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    xs = []
    ys = []
    
    for _ in range(n):
        x, y = MII()
        
        if x > y: x, y = y, x
        
        xs.append(x)
        ys.append(y)
    
    ans = 0
    cur = 0
    
    for i in sorted(range(n), key=lambda x: -ys[x]):
        cur += xs[i]
        ans = fmax(ans, fmin(cur, ys[i]))
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<pair<int, int>> recs(n);
	for (auto &[x, y]: recs) {
		cin >> x >> y;
		if (x < y) swap(x, y);
	}

	sort(recs.rbegin(), recs.rend());

	int cur = 0, ans = 0;

	for (auto &[x, y]: recs) {
		cur = min(cur + y, x);
		ans = max(ans, cur);
	}

	cout << ans;

	return 0;
}
```
