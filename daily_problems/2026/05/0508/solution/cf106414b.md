**提示 1：** 找到一些不可能的情形，对剩余情况给出构造。

首先，如果只有一种字符，显然不行。

同时，如果总长是 $3$ ，且三个字符都不同，肯定也不行。

别的情况呢？

如果总共只有两种字符，那不用切割就满足要求。

否则，进行如下考虑：

- 如果第一个字符出现了多次，直接第一个字符后切割即可。

- 否则，如果第二个字符出现了多次，直接第二个字符后切割即可。

- 否则，在第三个字符后切割即可。

两侧的字符种类数的和总是 $4$ 。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        s = I()
        
        outs.append('NO' if min(s) == max(s) or (n == 3 and len(set(s)) == 3) else 'YES')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		string s;
		cin >> n >> s;

		set<char> vis(s.begin(), s.end());
		cout << (vis.size() == 1 || (n == 3 && vis.size() == 3) ? "NO" : "YES") << '\n';
	}

	return 0;
}
```
