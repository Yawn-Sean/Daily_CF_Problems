**提示 1：** 考虑长度为 $l$ 的数组的总贡献如何。

我们相当于对长度为 $l$ 的数组求和后，乘以 $\frac{1}{l}$ ，再求均值，因此一个自然的想法是把所有长度为 $l$ 的数组一起算。

这些数组的和如何计算呢？我们可以用前缀和。

考虑前缀和计算中正的部分和负的部分分别看，则一个是 $acc[l]+acc[l+1]+\dots+acc[n]$ ，另一个是 $acc[0]+acc[1]+\dots+acc[n-l]$ ，两个都是前缀和的一个区间和，可以用前缀和的前缀和表示。也可以利用 $l$ 变化 $\mathcal{O}(1)$ 时，两个元素的变化的维护都是 $\mathcal{O}(1)$ 的。

算出这个和后，除以 $l$ 就是长度为 $l$ 的部分的和，再关于不同的 $l$ 求和，就是所有数组的和，最后除以总数组个数 $C_{n+1}^2$ 就是答案了。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, m, g = MII()
    
    if g > 2: print(0)
    else:
        path = [[] for _ in range(n)]
        
        for _ in range(m):
            u, v = GMI()
            path[u].append(v)
            path[v].append(u)
        
        dis = [-1] * n
        
        que = [0]
        dis[0] = 0
        
        flg = True
        
        for u in que:
            for v in path[u]:
                if dis[v] == -1:
                    dis[v] = dis[u] ^ 1
                    que.append(v)
                elif dis[u] == dis[v]:
                    flg = False
        
        print(1 if flg and dis[-1] else 0)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m, g;
	cin >> n >> m >> g;

	if (g > 2) cout << 0;
	else {
		vector<vector<int>> path(n);
		while (m --) {
			int u, v;
			cin >> u >> v;
			u --, v --;
			path[u].emplace_back(v);
			path[v].emplace_back(u);
		}

		vector<int> dis(n, -1);
		dis[0] = 0;

		bool flg = true;

		auto dfs = [&] (auto &self, int u) -> void {
			for (auto &v: path[u]) {
				if (dis[v] == -1) {
					dis[v] = dis[u] ^ 1;
					self(self, v);
				}
				else if (dis[u] == dis[v]) {
					flg = false;
				}
			}
		};

		dfs(dfs, 0);

		if (flg && dis[n - 1]) cout << 1;
		else cout << 0;
	}

	return 0;
}
```
