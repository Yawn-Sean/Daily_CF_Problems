**提示 1：** 有些元素删不掉。

**提示 2：** 删不掉的元素怎么办？

首先， $1,2$ 两个元素永远删不掉，因为永远不可能比邻居都大。

考虑最后一次操作， $1,2$ 只能在两端。

所以初始的时候 $1,2$ 也只能在两端。

这充分吗？答案是肯定的。每次操作删除最大的元素，其一定也是极大值，也就完成了任务。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
        outs.append('YES' if nums[0] + nums[-1] == 3 else 'NO')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;

		cout << (nums[0] + nums[n - 1] == 3 ? "YES" : "NO") << '\n';
	}

	return 0;
}
```
