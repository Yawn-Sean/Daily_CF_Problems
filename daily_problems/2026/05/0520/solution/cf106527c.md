**提示 1：** Hall 定理。考虑清楚所有的条件。

我们要最小化超过 $x$ 题的人数，同时你自己刚好做了 $x$ 个题。

这两件事不太统一 （因为 $x$ 并不超过 $x$ ），我们干脆最大化做了不超过 $x$ 个题的人数。设这个人数是 $v$ 。

考虑使用 Hall 定理。

我们需要考虑的就是每一个子集需要满足的限制，再将这些限制取交集即可。考虑一个 $i$ 元的题目子集，则这个子集最多被做了多少次呢？ $\min(x,i)\times v+i\times(n-v)$ 。于是这个数值不小于 $i$ 元子集的总解决次数。

发现其中只有总解决次数最大的 $i$ 元子集有用，因此对数组排序后取最大的 $i$ 项子集进行讨论即可。最后将不同的 $i$ 给出的限制取交集即可。

因为 $n$ 范围很小，所以可以直接用基数排序等排序方法。

时间复杂度为 $\mathcal{O}(n+p)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    t = II()
    outs = []
    
    for _ in range(t):
        n, p, x = MII()
        nums = LII()
        
        x = p - x
        
        cnt = [0] * (n + 1)
        for v in nums:
            cnt[n - v] += 1
        
        nums.clear()
        
        for i in range(n + 1):
            for _ in range(cnt[i]):
                nums.append(i)
        
        cur = 0
        ans = n
        
        for i in range(p):
            cur += nums[i]
            if x - (p - i - 1) > 0:
                ans = fmin(ans, cur // (x - (p - i - 1)))
        
        outs.append(n - ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, p, x;
		cin >> n >> p >> x;

		x = p - x;

		vector<int> cnt(n + 1, 0);
		for (int i = 0; i < p; i ++) {
			int v;
			cin >> v;
			cnt[n - v] ++;
		}

		vector<int> nums;
		for (int i = 0; i <= n; i ++) {
			while (cnt[i] --) {
				nums.emplace_back(i);
			}
		}

		long long cur = 0, ans = n;
		for (int i = 0; i < p; i ++) {
			cur += nums[i];
			if (x - (p - i - 1) > 0) {
				ans = min(ans, cur / (x - (p - i - 1)));
			}
		}

		cout << n - ans << '\n';
	}

	return 0;
}
```
