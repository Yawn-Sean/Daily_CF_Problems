**提示 1：** 枚举 $\max b_{i_j}$ ，前面部分要最大化什么？

看提示。

枚举之后，我们相当于要在选取对应的 $a_{i_j}$ 的基础上，前面 $b_{i_j}$ 更小的元素，我们要选出 $k-1$ 个 $a_{i_j}$ 最大的。

所以可以将元素按照 $b$ 的数值排序，再枚举当前最大的 $b$ 数值，该数值对应的 $a$ 必须选取，而前缀部分选取 $k-1$ 个 $a$ 数值最大的元素。新增元素的过程中，需要维护前 $k-1$ 大，可以使用堆。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    bound = 10 ** 18
    
    for _ in range(t):
        n, k = MII()
        v1 = []
        v2 = []
        
        for _ in range(n):
            x, y = MII()
            v1.append(x)
            v2.append(y)
        
        st_range = sorted(range(n), key=lambda x: v2[x])
        
        ans = -bound
        
        cur = 0
        pq = []
        
        for i in st_range:
            if len(pq) == k - 1:
                ans = fmax(ans, cur + v1[i] - v2[i])
            
            cur += v1[i]
            heappush(pq, v1[i])
            
            if len(pq) == k:
                cur -= heappop(pq)
        
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, k;
		cin >> n >> k;

		vector<pair<int, long long>> vals(n);
		for (auto &[a, b]: vals) cin >> a >> b;

		sort(vals.begin(), vals.end(), [&] (pair<int, long long> &x, pair<int, long long> &y) {return x.second < y.second;});

		long long cur = 0, ans = -1e18;
		priority_queue<int, vector<int>, greater<int>> pq;

		for (auto &[x, y]: vals) {
			if (pq.size() == k - 1) {
				ans = max(ans, cur + x - y);
			}

			cur += x;
			pq.push(x);

			if (pq.size() == k) {
				cur -= pq.top();
				pq.pop();
			}
		}

		cout << ans << '\n';
	}

	return 0;
}
```
