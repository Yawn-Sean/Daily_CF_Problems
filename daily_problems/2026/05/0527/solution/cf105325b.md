**提示 1：** 走一条路径的时候如何分解为各个段的权？

**提示 2：** 什么情况是值得合并的。

考虑一段路径的权值从头到尾是 $[w_1,w_2,\dots,w_n]$ 则总权值是 $nw_1+(n-1)w_2+\dots+w_n$ 。

我们没必要走一个结点两次，所以路径长度最多为 $n-1$ 。

我们要考虑从 $1$ 到 $i$ 的路径。但第一条边的权重不好确定，因为不知道要走几步。但最后一条边的权重是 $1$ 。

所以我们只有知道是倒数第几条边，才能知道答案。

所以不妨设当前要走倒数第 $i$ 条边且在第 $u$ 个结点，这样枚举 $u$ 走的边 $(v,w)$ ，就可以走到状态 $(i-1,v)$ ，且权值加上 $i\times w$ 。

因为开头只能在 $0$ 位置，而接下来走几条边都可以，取最优情况即可，所以初值是所有 $(i,0)$ 的位置标记为 $0$ 。而最后我们要走到剩余 $0$ 条边的情况，因此输出 $(0,1),(0,2),\dots,(0, n)$ 位置的数值即可。

时间复杂度为 $\mathcal{O}(n(n+m))$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    inf = 10 ** 9
    
    for _ in range(t):
        n, m = MII()
        path = [[] for _ in range(n)]
        
        for _ in range(m):
            u, v, w = MII()
            path[u].append(w * n + v)
        
        dp = [[inf] * n for _ in range(n)]
        
        for i in range(n): dp[i][0] = 0
        
        for i in range(n - 1, 0, -1):
            for j in range(n):
                for msk in path[j]:
                    w, nj = divmod(msk, n)
                    dp[i - 1][nj] = fmin(dp[i - 1][nj], dp[i][j] + i * w)
        
        outs.append(' '.join(map(str, (x if x < inf else -1 for x in dp[0][1:]))))
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t, inf = 1e9;
	cin >> t;

	while (t --) {
		int n, m;
		cin >> n >> m;

		vector<vector<pair<int, int>>> path(n);
		
		while (m --) {
			int u, v, w;
			cin >> u >> v >> w;
			path[u].emplace_back(v, w);
		}

		vector<vector<int>> dis(n, vector<int>(n, inf));

		for (int i = 0; i < n; i ++) dis[i][0] = 0;

		for (int i = n - 1; i > 0; i --) {
			for (int u = 0; u < n; u ++) {
				for (auto &[v, w]: path[u]) {
					dis[i - 1][v] = min(dis[i - 1][v], dis[i][u] + i * w);
				}
			}
		}

		for (int i = 1; i < n; i ++) {
			cout << (dis[0][i] < inf ? dis[0][i] : -1) << " \n"[i == n - 1];
		}
	}

	return 0;
}
```
