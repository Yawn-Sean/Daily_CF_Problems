**提示 1：** 先探讨先手必胜的条件。

**提示 2：** 如何计算必胜元素对数量。

什么情况下必胜呢？

如果开局两个元素相同，则必胜。接下来认为 $x,y$ 不相等。

否则 $x,y$ 不断操作，只能得到 $\mathrm{gcd}(x,y)$ 的倍数。且一旦有一个倍数没有出现，总能进行操作。

因此最后数的数量就是 $\frac{\max(x,y)}{\mathrm{gcd}(x,y)}$ ，而先手必胜就要求这个数是奇数。

考虑枚举 $\mathrm{gcd}(x,y)=g$ ，则 $x=x_0g,y=y_0g$ ，我们再枚举 $\max(x_0,y_0)=M\gt 1$ ，则要求 $M$ 是奇数，且另一个数的取法有欧拉函数 $\phi(M)$ 种。而两个数可以交换顺序，所以方案数乘 $2$ 。

所以对于每个 $g$ 而言，我们要求的就是不超过 $\frac{n}{g}$ 的奇数的 $\phi(x)$ 之和的两倍再加上 $1$ 。

因此只需我们通过素数筛预处理欧拉函数，再取前缀和就可以 $\mathcal{O}(1)$ 快速得到每个 $g$ 对应的答案，相加即可。也可以通过整除分块进一步得到更优的复杂度。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    M = 10 ** 6 + 5
    phi = list(range(M))
    
    for i in range(2, M):
        if phi[i] == i:
            for j in range(i, M, i):
                phi[j] = phi[j] // i * (i - 1)
    
    for i in range(2, M):
        if i % 2 == 0:
            phi[i] = 0
        else:
            phi[i] = 2 * phi[i]
    
    for i in range(1, M):
        phi[i] += phi[i - 1]
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        ans = 0
        
        for i in range(1, n + 1):
            ans += phi[n // i]
        
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 5;

	vector<long long> phi(M);
	iota(phi.begin(), phi.end(), 0);

	for (int i = 2; i < M; i ++) {
		if (phi[i] == i) {
			for (int j = i; j < M; j += i) {
				phi[j] = phi[j] / i * (i - 1);
			}
		}
	}

	for (int i = 2; i < M; i ++) {
		if (i & 1) phi[i] = 2 * phi[i];
		else phi[i] = 0;
	}

	for (int i = 1; i < M; i ++) {
		phi[i] += phi[i - 1];
	}

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		long long ans = 0;

		for (int i = 1; i <= n; i ++) {
			ans += phi[n / i];
		}

		cout << ans << '\n';
	}

	return 0;
}
```
