**提示 1：** 因为你可以把多余的操作反复作用在倒数几个元素中，所以后面的操作很可能是没啥用的。

**提示 2：** 特殊情况。

首先，如果 $m\lt n$ ，则前 $m$ 项可以 $n,n-1,\dots,n-(m-1)$ ，分别操作到结尾。剩余元素只能按照升序排在后面了。

而如果 $m\geq n$ 呢？

如果 $n=2$ ，则每次操作固定，答案只跟 $m$ 奇偶性有关。

否则，我们答案一定是 $[n,n-1,\dots,1]$ 。为什么呢？

直观来看，如果 $n-1$ 和 $m$ 奇偶性相同，我们可以先用 $n-1$ 恢复为升序，再不断交换最后两个元素。

否则，我们可以先用 $3$ 次操作把最后的 $[3,2,1]$ 变成 $[1,2,3]$ ：

$[3,2,1]\to [2,1,3]\to [1,3,2]\to[1,2,3]$ 

再用 $n-3$ 次操作把整个数组变成升序，最后再不断交换最后两个元素即可。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        
        if n == 2:
            if m % 2: outs.append('2 1')
            else: outs.append('1 2')
        else:
            m = fmin(n, m)
            ans = []
            
            for i in range(m):
                ans.append(n - i)
            
            for i in range(n - m):
                ans.append(i + 1)
            
            outs.append(' '.join(map(str, ans)))
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, m;
		cin >> n >> m;

		if (n == 2) {
			if (m & 1) cout << "2 1\n";
			else cout << "1 2\n";
		}
		else {
			m = min(n, m);
			for (int i = 0; i < m; i ++) cout << n - i << ' ';
			for (int i = 1; i <= n - m; i ++) cout << i << ' ';
			cout << '\n';
		}
	}

	return 0;
}
```
