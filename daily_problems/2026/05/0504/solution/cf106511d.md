**提示 1：** 考虑每个数码最少要用多少个？

看提示。

那我们就要找到对于每个数码而言，如果出现了 $k$ 次，最小的数字是多少。

可以发现对于 $0,1,2,\dots,9$ 结果分别是 $10\dots 0,11\dots 1, 22\dots 2,\dots, 99\dots 9$ 。

因此我们枚举这些数码，计算最大的不超过 $n$ 的这个数码对应的数有多少个位为这个数码即可。

时间复杂度为 $\mathcal{O}(\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    ans = 0
    
    for i in range(10):
        cur = i if i else 10
        while cur <= n:
            ans += 1
            cur *= 10
            cur += i
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, ans = 0;
	cin >> n;

	for (int i = 0; i < 10; i ++) {
		long long cur = (i ? i : 10);
		while (cur <= n) {
			ans ++;
			cur = cur * 10 + i;
		}
	}

	cout << ans;

	return 0;
}
```
