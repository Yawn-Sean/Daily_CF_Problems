**提示 1：** 路径拆为两部分拼起来，求的实际上是什么？

显然是一个树上的 DP 问题。

路径相关的问题可以考虑从某个结点往更深的地方连的路径。这样每条路径一定可以拆分成两条这样的路径。

而这题中，如果我们选定了 $u$ 出发的向两个不同的儿子方向的往下的路径，我们要求的东西实际上是什么呢？也就是我们需要维护什么呢？

考虑去绝对值，则发现其中一条路径中，如果端点的 $x,y,z$ 对应的权值是 $v_x,v_y,v_z$ ，则另一条路径的端点的权值是 $-v_x,-v_y,-v_z$ 。

而 $|x_1-x_2|+|y_1-y_2|+|z_1-z_2|$ 等价于 $\max\limits_{w_1,w_2,w_3} w_1(x_1-x_2)+w_2(y_1-y_2)+w_3(z_1-z_2)$ 。

所以我们无需真的知道 $x_1,x_2$ 一类的大小关系，只需枚举其中一端的 $2^3=8$ 种可能的权重，另一端与其相反，最终结果取最大值就跟绝对值是一个效果了。

所以维护从每个结点出发，往更深处的端点且端点权值分别为 $(a,b,c)$ 的最大权值路径，再将不同子树对应的答案合并即可。考虑第 $i$ 棵子树的时候只在意前面的子树的最大权值，因此可以边遍历边维护当前权值组合的最大权。

时间复杂度为 $\mathcal{O}(n2^d)$ ， $d$ 是坐标维数。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    n = II()
    path = [[] for _ in range(n)]
    
    for _ in range(n - 1):
        u, v, w = MII()
        u -= 1
        v -= 1
        path[u].append(w * n + v)
        path[v].append(w * n + u)
    
    pts = [[] for _ in range(3)]
    
    for _ in range(n):
        pt = LII()
        
        for i in range(3):
            pts[i].append(pt[i])
    
    parent = [-1] * n
    que = [0]
    
    for u in que:
        for msk in path[u]:
            v = msk % n
            if parent[u] != v:
                parent[v] = u
                que.append(v)
    
    ans = 0
    
    ma = [[0] * n for _ in range(8)]
    
    for i in reversed(que):
        for j in range(8):
            for k in range(3):
                if j >> k & 1:
                    ma[j][i] += pts[k][i]
                else:
                    ma[j][i] -= pts[k][i]
        
        for msk in path[i]:
            w, v = divmod(msk, n)
            if parent[v] == i:
                for j in range(8):
                    ans = fmax(ans, ma[j][v] + w + ma[7 - j][i])
                for j in range(8):
                    ma[j][i] = fmax(ma[j][i], ma[j][v] + w)
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<vector<pair<int, int>>> path(n);

	for (int i = 0; i < n - 1; i ++) {
		int u, v, w;
		cin >> u >> v >> w;
		u --, v --;
		path[u].emplace_back(v, w);
		path[v].emplace_back(u, w);
	}

	vector<array<long long, 3>> pts(n);
	for (auto &[x, y, z]: pts) cin >> x >> y >> z;

	long long total_ans = 0;

	auto dfs = [&] (auto &self, int u, int p) -> array<long long, 8> {
		array<long long, 8> ans;
		for (int i = 0; i < 8; i ++) {
			ans[i] = 0;
			for (int j = 0; j < 3; j ++) {
				if (i >> j & 1) ans[i] += pts[u][j];
				else ans[i] -= pts[u][j];
			}
		}

		for (auto &[v, w]: path[u]) {
			if (v != p) {
				auto res = self(self, v, u);
				
				for (int i = 0; i < 8; i ++)
					total_ans = max(total_ans, ans[i] + res[7 ^ i] + w);

				for (int i = 0; i < 8; i ++)
					ans[i] = max(ans[i], res[i] + w);
			}
		}

		return ans;
	};

	dfs(dfs, 0, -1);

	cout << total_ans;

	return 0;
}
```
