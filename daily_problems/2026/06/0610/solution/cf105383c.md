**提示 1：** 把 $p$ 的存在理解为同时进行多次不同位置的交换。

**提示 2：** 条件是什么？

**提示 3：** 简化问题。有啥容易理解的做法吗？对其加速。

首先， $p$ 的存在相当于以同一种方式打乱 $a,b$ 两个排列。而每种打乱方式都可以理解为若干次选取两个元素交换得到的。

而交换一次，逆序对的奇偶性改变。所以 $a,b$ 两个序列的奇偶性的变换次数一样。所以一开始只能是有相同的逆序对数量，否则无法构造。

对于剩余的情况，为了方便处理，我们把第一个序列变成 $1,2,\dots,n$ 。此时第二个序列也应当有偶数个逆序对，设为 $2k$ 。

比较朴素的想法是，最后让两个序列的逆序对都刚好是 $k$ 个。

发现可以对第二个序列进行冒泡排序，执行到第 $k$ 次结束后，刚好满足要求。

那怎么加速这个过程呢？前若干轮是完整进行的冒泡排序，第 $k$ 轮所在的一轮是走到一半的。因此前面的部分可以直接排序，第 $k$ 轮模拟对应的次数即可。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    p1 = LGMI()
    p2 = LGMI()
    
    p = [0] * n
    for i in range(n):
        p[p1[i]] = p2[i]
    
    fen = FenwickTree(n)
    notes = []
    rev_pairs = 0
    
    for x in p:
        notes.append(fen.rsum(x + 1, n - 1))
        rev_pairs += notes[-1]
        fen.add(x, 1)
    
    if rev_pairs % 2:
        print('No')
    else:
        rev_pairs //= 2
        
        q = list(range(n))
        
        for i in range(n):
            if rev_pairs < notes[i]:
                q[:i] = sorted(q[:i], key=lambda x: p[x])
                p[:i] = sorted(p[:i])
                
                for j in range(rev_pairs):
                    p[i - j], p[i - j - 1] = p[i - j - 1], p[i - j]
                    q[i - j], q[i - j - 1] = q[i - j - 1], q[i - j]
                
                break
            else:
                rev_pairs -= notes[i]
        
        print('Yes')
        print(' '.join(str(x + 1) for x in q))
        print(' '.join(str(x + 1) for x in p))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> p1(n), p2(n);

	for (auto &x: p1) cin >> x, x --;
	for (auto &x: p2) cin >> x, x --;

	vector<int> p(n);
	for (int i = 0; i < n; i ++) p[p1[i]] = p2[i];

	long long rev_pairs = 0;
	vector<int> notes(n);
	atcoder::fenwick_tree<int> fen(n);

	for (int i = 0; i < n; i ++) {
		notes[i] = fen.sum(p[i] + 1, n);
		rev_pairs += notes[i];
		fen.add(p[i], 1);
	}

	if (rev_pairs & 1) cout << "No";
	else {
		vector<int> q(n);
		iota(q.begin(), q.end(), 0);

		rev_pairs /= 2;

		for (int i = 0; i < n; i ++) {
			if (rev_pairs < notes[i]) {
				sort(q.begin(), q.begin() + i, [&] (int i, int j) {return p[i] < p[j];});
				sort(p.begin(), p.begin() + i);

				for (int j = 0; j < rev_pairs; j ++) {
					swap(q[i - j], q[i - j - 1]);
					swap(p[i - j], p[i - j - 1]);
				}
				
				break;
			}
			rev_pairs -= notes[i];
		}

		cout << "Yes\n";
		for (int i = 0; i < n; i ++) cout << q[i] + 1 << " \n"[i == n - 1];
		for (int i = 0; i < n; i ++) cout << p[i] + 1 << " \n"[i == n - 1];
		
	}

	return 0;
}
```
