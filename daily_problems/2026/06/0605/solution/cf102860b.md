**提示 2：** 枚举最靠近内部的那条边。

先看提示。我们用总数减去在外部的三角形个数。

因为整体是一个环形，所以将其转化为链，即排序后，将每个元素 $+l$ 放到末尾，表示第二遍走到环。

而圆心在外部只需最靠近内部的那条边对应的弧长不超过一半。所以对于每个点 $l$ ，可以双指针维护最大的右端点 $r$ （下面代码的 $r$ 不是这个含义，稍有不同）。

考虑枚举这条边的左端点（这里说左端点主要是数组中双指针的左端点），剩余点都在这个点接下来的半圆内。我们固定左端点 $l$ 的话，三角形的剩余两个点就是在 $l+1,\dots, r$ 中选两个点，方案数是 $\frac{(r-l)(r-l-1)}{2}$ 。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, l = MII()
    nums = LII()
    nums.sort()
    
    for i in range(n):
        nums.append(nums[i] + l)
    
    ans = n * (n - 1) * (n - 2) // 6
    r = 0
    
    for i in range(n):
        while 2 * (nums[r] - nums[i]) < l:
            r += 1
        
        v = r - i - 1
        ans -= v * (v - 1) // 2
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, l;
	cin >> n >> l;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	sort(nums.begin(), nums.end());
	for (int i = 0; i < n; i ++) nums.emplace_back(nums[i] + l);

	int r = 0;
	long long ans = 1ll * n * (n - 1) * (n - 2) / 6;

	for (int i = 0; i < n; i ++) {
		while (2 * (nums[r] - nums[i]) < l) r ++;
		int v = r - i - 1;
		ans -= 1ll * v * (v - 1) / 2;
	}

	cout << ans;

	return 0;
}
```
