**提示 1：** 如何判断操作的方向？

**提示 2：** 如何执行操作？

首先，对于第一类查询，如何判断是往上翻还是往右翻呢？

直接找到上面的两点，看是在同一条横线上还是竖线上即可，因此查询坐标就足够完成这件事（同时为了解决第二类查询，也得求坐标，所以也是必要的）。

接下来怎么维护操作呢？

往上翻的过程中， $x$ 坐标依次减少 $0,1,\dots$ ， $y$ 坐标依次增加 $0,1,\dots$ 。

往右翻的过程中， $x$ 坐标依次增加 $0,1,\dots$ ， $y$ 坐标依次减少 $0,1,\dots$ 。

所以就是区间加等差数列。可以用树状数组维护，相当于区间加了个一次函数，分别差分维护一次项系数和常数系数即可。

时间复杂度为 $\mathcal{O}(n+q\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, l = MII()
    nums = LII()
    nums.sort()
    
    for i in range(n):
        nums.append(nums[i] + l)
    
    ans = n * (n - 1) * (n - 2) // 6
    r = 0
    
    for i in range(n):
        while 2 * (nums[r] - nums[i]) < l:
            r += 1
        
        v = r - i - 1
        ans -= v * (v - 1) // 2
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, l;
	cin >> n >> l;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	sort(nums.begin(), nums.end());
	for (int i = 0; i < n; i ++) nums.emplace_back(nums[i] + l);

	int r = 0;
	long long ans = 1ll * n * (n - 1) * (n - 2) / 6;

	for (int i = 0; i < n; i ++) {
		while (2 * (nums[r] - nums[i]) < l) r ++;
		int v = r - i - 1;
		ans -= 1ll * v * (v - 1) / 2;
	}

	cout << ans;

	return 0;
}
```
