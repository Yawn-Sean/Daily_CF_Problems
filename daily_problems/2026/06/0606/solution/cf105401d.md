**提示 1：** 三角形上，两数之和等于第三个数。

**提示 2：** 分配数字。

首先看提示 1 ，所以中间的一行的骨架如果搭好了，上下两层的差值也定下来了。

中间的边的数量恰好是奇数的数量，且这样作差刚好是偶数，留给上下两侧。

考虑从大到小一个个凑偶数的差值，发现直接中间用 $2k-1,1,2k-3,3,\dots$ 这样的序列填就行。

然后用这些边权恢复成点的赋值，交错加减即可。

最后，如果你的做法出现了负数，所有数增加一个足够大的数即可。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    
    ans = [1]
    
    l = 1
    r = 2 * n + 1
    
    for i in range(n + 1):
        if i % 2 == 0:
            ans.append(ans[-1] + r)
            r -= 2
        else:
            ans.append(ans[-1] - l)
            l += 2
    
    print(' '.join(map(str, ans)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	long long cur = 1;
	int l = 1, r = 2 * n + 1;
	cout << 1;
	for (int i = 1; i < n + 2; i ++) {
		if (i & 1) cur += r, r -= 2;
		else cur -= l, l += 2;
		cout << ' ' << cur;
	}

	return 0;
}
```
