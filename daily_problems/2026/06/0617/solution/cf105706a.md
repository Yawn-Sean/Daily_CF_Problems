**提示 1：** 要模拟什么？如何加速？

考虑这个序列需要如何构造。发现从头开始贪心就行，第一项开始不断赋值为 $1$ ，遇到 $1$ 变成 $2$ ，遇到 $2$ 变成 $3$ ，以此类推，看最后变成啥。

那也就是说我们遇到了 $x$ 就要找下一个 $x+1$ 。所以当我们遍历到某个位置 $i$ 后，下一个要找的位置一定是已经确定了的。

所以对于某个查询 $(l,r)$ ，我们大致流程是这样：

- 先找到 $l$ 后的第一个 $1$ 的位置。

- 接下来不断寻找下一个数字的位置，直到跳出区间。

- 走到的最后的一个数字加一就是答案。

如果第一步就已经超过 $r$ ，答案输出 $1$ 。

否则，我们很容易通过预处理找每个位置后面的第一个 $1$ 的位置，因此第一步完成。

接下来的时间上需要加速的主要是第二步，可能要不断寻找某个位置后第一个比它大 $1$ 的数的位置。

既然我们要走很多步，我们就可以选择倍增，这样就能一次走 $2^k$ 步。

于是预处理倍增，接下来就是判断最多能走多少步。从大到小依次遍历 $2^i$ ，能走即走，就可以得到最多能走的步数，上面的第二个流程也就解决了。

时间复杂度为 $\mathcal{O}((n+q)\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, q = MII()
        nums = LII()
        
        next_pos1 = [n] * n
        next_pos = [[n] * (n + 1) for _ in range(20)]
        
        pos = [n] * (n + 2)
        
        for i in range(n - 1, -1, -1):
            next_pos[0][i] = pos[nums[i] + 1]
            pos[nums[i]] = i
            next_pos1[i] = pos[1]
        
        for i in range(19):
            for j in range(n):
                next_pos[i + 1][j] = next_pos[i][next_pos[i][j]]
    
        ans = []
        for _ in range(q):
            l, r = GMI()
            
            if next_pos1[l] > r: ans.append(1)
            else:
                l = next_pos1[l]
                res = 2
                
                for i in range(19, -1, -1):
                    if next_pos[i][l] <= r:
                        l = next_pos[i][l]
                        res += 1 << i
                
                ans.append(res)
        
        outs.append(' '.join(map(str, ans)))
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, q;
		cin >> n >> q;

		vector<int> nums(n);
		for (auto &v: nums) cin >> v;

		vector<int> next_pos1(n, n), pos(n + 2, n);
		vector<vector<int>> next_pos(20, vector<int>(n + 1, n));

		for (int i = n - 1; i >= 0; i --) {
			next_pos[0][i] = pos[nums[i] + 1];
			pos[nums[i]] = i;
			next_pos1[i] = pos[1];
		}

		for (int i = 0; i < 19; i ++) {
			for (int j = 0; j < n; j ++) {
				next_pos[i + 1][j] = next_pos[i][next_pos[i][j]];
			}
		}

		for (int i = 0; i < q; i ++) {
			int l, r;
			cin >> l >> r;
			l --, r --;

			if (next_pos1[l] > r) cout << 1;
			else {
				l = next_pos1[l];
				int res = 2;

				for (int i = 19; i >= 0; i --) {
					if (next_pos[i][l] <= r) {
						l = next_pos[i][l];
						res += 1 << i;
					}
				}

				cout << res;
			}

			cout << " \n"[i == q - 1];
		}
	}

	return 0;
}
```
