**提示 1：** 拆贡献。每一组贡献怎么算？怎么求和？

最大值减去最小值的期望，等于最大值和最小值之间包含 $[i,i+1]$ 的概率对不同 $i$ 求和。最大值减去最小值关于不同子集求和，就是后者条件满足的方案数的和。

而 $[i,i+1]$ 在区间内，相当于 $[1,i]$ 中至少取一个数， $[i+1,n]$ 中至少取一个数。

因此共有 $(2^i-1)(2^{n-i}-1)$ 种方案。

求和后，就是 $\sum\limits_{i=1}^{n-1}(2^i-1)(2^{n-i}-1)=\sum\limits_{i=1}^{n-1}(2^n-2^i-2^{n-i}+1)=(n-1)2^n-(2^n-2)-(2^n-2)+(n-1)=(n-3)2^n+n+3$ 。快速幂即可计算。

时间复杂度为 $\mathcal{O}(\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    mod = 10 ** 9 + 7
    
    for _ in range(t):
        n = II()
        outs.append((pow(2, n, mod) * (n - 3) % mod + n + 3) % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t, mod = 1e9 + 7;
	cin >> t;

	while (t --) {
		long long n;
		cin >> n;
		cout << ((n - 3) % mod * quickPow(2, n, mod) % mod + n + 3) % mod << '\n';
	}

	return 0;
}
```
