**提示 2：** 如何让 $0$ 变成废的。

如果整个数组中不存在 $0$ ，直接正数取正，负数取负就是能凑出来的最大的数了。接下来第 $i$ 项乘以 $2^{i-1}$ 即可。

有 $0$ 怎么办？我们让有数值的那些位置，都是更大的 $2^{29},2^{28},\dots$ （正负性与前一段一致），剩余位置填上更小的 $2^0,2^{1},\dots$ ，这样为了能被 $m$ 整除，更低的位只能全取 $0$ ，剩余部分只能最大化绝对值，因此保证了只有两个解。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    nums = LII()
    
    cur = 1 << n - 1
    m = 0
    
    ans = [0] * n
    
    for i in range(n):
        if nums[i]:
            ans[i] = cur // nums[i]
            m += cur
            cur //= 2
    
    for i in range(n):
        if nums[i] == 0:
            ans[i] = cur
            cur //= 2
    
    print(m)
    print(*ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	int m = 0;
	int cur = 1 << (n - 1);
	vector<int> ans(n);

	for (int i = 0; i < n; i ++) {
		if (nums[i]) {
			ans[i] = cur / nums[i];
			m += cur;
			cur >>= 1;
		}
	}

	for (int i = 0; i < n; i ++) {
		if (!nums[i]) {
			ans[i] = cur;
			cur /= 2;
		}
	}

	cout << m << '\n';
	for (auto &x: ans) cout << x << ' ';

	return 0;
}
```
