**提示 1：** 大球不容易被盖住，先考虑。

**提示 2：** 似乎可以贪心。

看提示 1 。如果我们一开始不选最大的球，后面如果盖不住它就肯定不行了，但后面的球形也很难覆盖它。

所以考虑将球按照半径从大到小排序。

如果遇到一个球没有被之前所选的球的扩大后的范围覆盖，我们肯定得选它。下面我们说明这么贪心做的构造就一定满足要求。

如果一个球没有被之前所选的任意一个球的扩大后的范围覆盖，则对于之前的任意一个球，两者的距离至少为 $3r_i-r$ （ $r_i$ 为之前那个球的半径， $r$ 为当前圆的半径）。 而 $3r_i-r\geq r_i+r$ ，所以这两个球的初始位置相离或外切，满足要求。

时间复杂度为 $\mathcal{O}(n^2)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    xs = []
    ys = []
    zs = []
    rs = []
    
    for _ in range(n):
        x, y, z, r = MII()
        xs.append(x)
        ys.append(y)
        zs.append(z)
        rs.append(r)
    
    ans = []
    for i in sorted(range(n), key=lambda x: -rs[x]):
        flg = True
        
        for j in ans:
            dx = xs[j] - xs[i]
            dy = ys[j] - ys[i]
            dz = zs[j] - zs[i]
            totr = rs[j] * 3 - rs[i]
            
            if dx * dx + dy * dy + dz * dz < totr * totr:
                flg = False
        
        if flg:
            ans.append(i)
    
    print(len(ans))
    print(*(x + 1 for x in ans))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<array<int, 4>> balls(n);
	for (auto &[x, y, z, r]: balls) {
		cin >> x >> y >> z >> r;
	}

	vector<int> order(n);
	iota(order.begin(), order.end(), 0);
	sort(order.begin(), order.end(), [&] (int v1, int v2) {return balls[v1][3] > balls[v2][3];});

	vector<int> ans;

	for (auto &i: order) {
		bool flg = true;
		
		for (auto &j: ans) {
			auto [x1, y1, z1, r1] = balls[i];
			auto [x2, y2, z2, r2] = balls[j];
			int dx = x1 - x2, dy = y1 - y2, dz = z1 - z2;
			int totr = r2 * 3 - r1;

			if (dx * dx + dy * dy + dz * dz < totr * totr) {
				flg = false;
			}
		}

		if (flg) ans.emplace_back(i);
	}

	cout << ans.size() << '\n';
	for (auto &x: ans) cout << x + 1 << ' ';

	return 0;
}
```
