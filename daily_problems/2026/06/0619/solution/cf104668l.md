**提示 1：** 第二个条件啥意思？

**提示 2：** 第一个条件如何处理？

第二个条件就是说交换所有 $2x,2x+1$ ，逆序对数量的改变量是 $D$ 。

但实际上，除了交换的元素之间，其他逆序对不会发生改变。所以相当于 $\lfloor n/2\rfloor$ 对逆序关系反转，最后改变量是 $D$ 。

假设其中有 $k$ 个开始是逆序关系，则该变量是 $(\lfloor n/2\rfloor-k)-k$ ，这个东西需要等于 $D$ ，所以可以解出对应的 $k$ 。于是这里相当于 $\lfloor n/2\rfloor$ 对元素中选取 $k$ 对作为逆序关系。

而开头的时候，我们需要任意一组 $2x,2x+1$ 不相邻。这件事不太好直接算，可以用容斥（二项式反演），求至少 $0,1,2,\dots$ 组 $2x,2x+1$ 相邻的方案数然后间接求出答案。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, a, b = MII()
    nums = LII()
    
    ma1 = 0
    ma2 = 0
    
    for x in nums:
        if x > ma1:
            ma1, ma2 = x, ma1
        elif x > ma2:
            ma2 = x
    
    if a == b:
        val = 0
        for x in nums:
            val ^= x % (a + 1)
        print('Petyr' if val else 'Varys')
    
    elif a > b:
        if ma1 > b: print('Petyr')
        else:
            val = 0
            for x in nums:
                val ^= x % (b + 1)
            print('Petyr' if val else 'Varys')
    
    else:
        if ma2 > a: print('Varys')
        else:
            val = 0
            for x in nums:
                val ^= x % (a + 1)
            
            if ma1 <= a:
                print('Petyr' if val else 'Varys')
            else:
                target = (ma1 % (a + 1)) ^ val
                print('Petyr' if target <= a and ma1 - target <= a else 'Varys')
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, a, b;
	cin >> n >> a >> b;

	vector<int> nums(n);
	for (auto &v: nums) cin >> v;

	int ma1 = 0, ma2 = 0;

	for (auto &x: nums) {
		if (x > ma1) swap(ma1, ma2), ma1 = x;
		else ma2 = max(ma2, x);
	}

	if (a == b) {
		int val = 0;
		for (auto &v: nums) val ^= v % (a + 1);
		cout << (val ? "Petyr" : "Varys");
	}
	else if (a > b) {
		if (ma1 > b) cout << "Petyr";
		else {
			int val = 0;
			for (auto &v: nums) val ^= v % (b + 1);
			cout << (val ? "Petyr" : "Varys");
		}
	}
	else {
		if (ma2 > a) cout << "Varys";
		else {
			int val = 0;
			for (auto &v: nums) val ^= v % (a + 1);
			
			if (ma1 <= a) cout << (val ? "Petyr" : "Varys");
			else {
				int target = (ma1 % (a + 1)) ^ val;
				cout << (target <= a && ma1 - target <= a ? "Petyr" : "Varys");
			}
		}
	}

	return 0;
}
```
