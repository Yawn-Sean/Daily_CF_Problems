**提示 1：** 想想简单的情况。把复杂的情况转化过去。

$A=B$ 是简单的情况，此时两个人的操作是一致的，是对称的博弈。所以可以计算每一堆的 SG 函数，最后取异或和，根据异或和是否为 $0$ 判断先手还是后手胜利。（如果这个不熟悉得去学习博弈论的 SG 函数了）

然后这个经典博弈的 SG 函数是直接关于 $k+1$ 取模，你可以打表自己看一下。

当然，整个数组最大值如果不超过 $A,B$ 中较小的那个，跟前面的情况也完全一致。

接下来，先手且操作空间更大的情况，是更简单的情况（因为两个条件看起来都更有利）。如果 $A\gt B$ ，且数组最大元素大于 $B$ ，此时先手方可以操作这个元素（至少可以减去 $1,2,\dots, B+1$ ），然后接下来选择让自己的操作也在 $1\sim B$ 之间。可以说明此时先手必胜。

因为第一次操作后变成对称博弈，且先手方可以让第一次操作后的 SG 函数的异或和，变成任意一个想要的数，也就可以让结果变成 $0$ ，进而让自己获胜。

如果 $A\lt B$ 且数组最大元素大于 $A$ 呢？此时，先手方操作完后就是前面这类讨论的情形了。

于是先手方获胜的目标就是：把所有数都变成不超过 $A$ 的，且整体数组异或和为 $0$ 。

因为先手方只能操作一次，所以不能有两个超过 $A$ 的元素。没有超过 $A$ 的元素的情况已经讨论过；而只有一个的情况，就操作它。而操作的目标就是异或和为 $0$ ，所以其最终目标唯一确定，取的石头数量也唯一确定。因此，只要其目标小于等于 $A$ 且取的石头数量在 $1\sim A$ 之间就满足要求。

否则就相当于只可能给后手留必胜局面。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, a, b = MII()
    nums = LII()
    
    ma1 = 0
    ma2 = 0
    
    for x in nums:
        if x > ma1:
            ma1, ma2 = x, ma1
        elif x > ma2:
            ma2 = x
    
    if a == b:
        val = 0
        for x in nums:
            val ^= x % (a + 1)
        print('Petyr' if val else 'Varys')
    
    elif a > b:
        if ma1 > b: print('Petyr')
        else:
            val = 0
            for x in nums:
                val ^= x % (b + 1)
            print('Petyr' if val else 'Varys')
    
    else:
        if ma2 > a: print('Varys')
        else:
            val = 0
            for x in nums:
                val ^= x % (a + 1)
            
            if ma1 <= a:
                print('Petyr' if val else 'Varys')
            else:
                target = (ma1 % (a + 1)) ^ val
                print('Petyr' if target <= a and ma1 - target <= a else 'Varys')
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, a, b;
	cin >> n >> a >> b;

	vector<int> nums(n);
	for (auto &v: nums) cin >> v;

	int ma1 = 0, ma2 = 0;

	for (auto &x: nums) {
		if (x > ma1) swap(ma1, ma2), ma1 = x;
		else ma2 = max(ma2, x);
	}

	if (a == b) {
		int val = 0;
		for (auto &v: nums) val ^= v % (a + 1);
		cout << (val ? "Petyr" : "Varys");
	}
	else if (a > b) {
		if (ma1 > b) cout << "Petyr";
		else {
			int val = 0;
			for (auto &v: nums) val ^= v % (b + 1);
			cout << (val ? "Petyr" : "Varys");
		}
	}
	else {
		if (ma2 > a) cout << "Varys";
		else {
			int val = 0;
			for (auto &v: nums) val ^= v % (a + 1);
			
			if (ma1 <= a) cout << (val ? "Petyr" : "Varys");
			else {
				int target = (ma1 % (a + 1)) ^ val;
				cout << (target <= a && ma1 - target <= a ? "Petyr" : "Varys");
			}
		}
	}

	return 0;
}
```
