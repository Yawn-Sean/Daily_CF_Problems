**提示 1：** 第一个元素一定不同。不如先确定其中一个，另一个怎么选？

首先，每个元素两两不同，所以相当于字典序的比较只需比较第一个元素。

于是，考虑枚举第二个序列的第一个元素 $x$ ，这个元素的交换次数也就此确定。接下来，第一个序列的第一个元素的所有选择就是所有小于 $x$ 的元素，我们只需找这些元素的最小下标，将对应元素操作到开头即可。

而这对应于关于元素升序排序后的一个前缀，我们取排序后的前缀原下标最小值即可。这里值域很小，不需要真的排序，开数组记录元素下标即可。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, ca, cb = MII()
        v1 = LII()
        v2 = LII()
        
        p1 = [0] * n
        p2 = [0] * n
        
        for i in range(n):
            p1[(v1[i] - 1) // 2] = i
            p2[(v2[i] - 1) // 2] = i
        
        ans = 10 ** 18
        cur = n
        
        for i in range(n):
            cur = fmin(cur, p1[i])
            ans = fmin(ans, cur * ca + p2[i] * cb)
        
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, ca, cb;
		cin >> n >> ca >> cb;

		vector<int> p1(n), p2(n);

		for (int i = 0; i < n; i ++) {
			int x; cin >> x;
			p1[(x - 1) / 2] = i;
		}

		for (int i = 0; i < n; i ++) {
			int x; cin >> x;
			p2[(x - 1) / 2] = i;
		}

		long long ans = 1e18;
		int cur = n;

		for (int i = 0; i < n; i ++) {
			cur = min(cur, p1[i]);
			ans = min(ans, 1ll * cur * ca + 1ll * p2[i] * cb);
		}

		cout << ans << '\n';
	}

	return 0;
}
```
