**提示 1：** 答案是一系列结果的异或和。

**提示 2：** 考虑同一组 $l,r$ 出现了多少次。

先看提示 1，因此我们只需看那些出现次数是奇数的子数组，计算它们的和后异或即可，

如果一个区间是 $[l,r]$ ，则它出现在多少种分割方案里呢？

实际上是 $2^{\max(0,l-2)}\times 2^{\max(0,n-r-1)}$ 。因为前面可以分割的位置和后面可以分割的位置有这么多，每个位置都可以独立选择。

所以只有 $l\leq 2$ 且 $r\geq n-1$ 的情况下出现奇数次，因此计算这四个子数组的和，异或起来就行。

$n=1,2$ 需不需要特判看你的代码实现了。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
        outs.append(sum(nums) ^ sum(nums[1:]) ^ sum(nums[:-1]) ^ sum(nums[1:-1]))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;

		long long total = 0;
		for (auto &v: nums) total += v;

		long long ans = 0;
		ans ^= total;
		ans ^= total - nums[0];
		ans ^= total - nums[n - 1];
		if (n > 1) ans ^= total - nums[0] - nums[n - 1];

		cout << ans << '\n';
	}

	return 0;
}
```
