**提示 1：** $k$ 较大时怎么构造？

**提示 2：** 考虑特殊情况。

首先，如果 $k-n$ 足够大，那么我们可以在 $[1,2,1,2,\dots]$ 或 $[2,1,2,1,\dots]$ 序列的基础上进行操作。

假设需要增加的数值是 $V$ ，如果 $V$ 是偶数，任意选择一个 $2$ 增加这个数值，可以保证答案为 $0$ 。

否则

- 如果 $n$ 是奇数，则我们可以选择 $[1,2,\dots,1]$ 和 $[2,1,\dots,2]$ 的一种，这两种情况总有一种使得 $V$ 为偶数，按照前面的处理即可。

- 如果 $n$ 是偶数，则：

    - 若 $V=1$ ，即 $k=3n/2+1$ ，则答案不可能为 $0$ ，可以这么证明：

        - 如果答案为 $0$ ，则不能有 $n/2+1$ 个奇数，否则必然有一对奇数相邻。于是数组的最小和是 $n/2+n/2\times 2=3/2 n$ ，此时 $n/2$ 个 $1$ ， $n/2$ 个 $2$ ，只能选择其一加上 $1$ ，但加上了总不满足要求，因此不可行。

        - 此时可以构造 $[1,1,4,1,2,1,\dots]$ 。
    
    - 若 $V\gt 1$ ，则：

        - 如果 $n\gt 2$ ，则我们可以以 $[2,4,1,2,\dots]$ 为基准，接下来对第四个元素 $2$ 加剩余的数量即可。

        - 如果 $n=2$ ，此时就是拆两个正整数使其与和最小。此时如果 $k$ 有多个二进制位，拆出来一个即可使得答案为 $0$ ，否则，答案至少为 $1$ ，且 $1,k-1$ 满足要求。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, k = MII()
        
        if n == 2:
            if k & -k == k: outs.append(1)
            else: outs.append(0)
        else:
            v = k - n
            if n % 2 == 0 and k - n == n // 2 + 1: outs.append(1)
            else: outs.append(fmax(0, n - 1 - 2 * v))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, k;
		cin >> n >> k;

		if (n == 2) {
			if ((k & -k) == k) cout << 1;
			else cout << 0;
		}
		else {
			int v = k - n;
			if (n % 2 == 0 && k - n == n / 2 + 1) cout << 1;
			else cout << max(0, n - 1 - 2 * v);
		}
		cout << '\n';
	}

	return 0;
}
```
