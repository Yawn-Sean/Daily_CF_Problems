**提示 1：** 问题实际上是个图论上求距离的问题。

**提示 2：** 如何建图？

考虑建图。为了边数不超标，我们在所有人和他会的语言之间连一条边。

那么两个人需要翻译多少次就是他们之间的路径最少要经过多少个人。

而这个图是个二分图，所以实际上这等于两个人之间的距离除以 $2$ 再减去 $1$ 。

但直接求距离仍然不可行。考虑到语言数量很少，计算每种语言到每个人的距离，用 BFS 即可。

而任意两个人的距离等于某种语言到两个人的距离和，因此枚举中介语言求这个距离和的最小值即可。

时间复杂度为 $\mathcal{O}(nm^2+qm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, m, q = MII()
    path = [[] for _ in range(n + m)]
    
    for u in range(n):
        _, *idxs = GMI()
        
        for v in idxs:
            path[u].append(n + v)
            path[n + v].append(u)
    
    dis = [[-1] * (n + m) for _ in range(m)]
    
    for x in range(m):
        que = [n + x]
        dis[x][n + x] = 0
        
        for u in que:
            for v in path[u]:
                if dis[x][v] == -1:
                    dis[x][v] = dis[x][u] + 1
                    que.append(v)
    
    outs = []
    inf = 100000
    
    for _ in range(q):
        u, v = GMI()
        ans = inf
        
        for x in range(m):
            if dis[x][u] != -1 and dis[x][v] != -1:
                ans = fmin(ans, dis[x][u] + dis[x][v])
        
        outs.append(ans // 2 - 1 if ans < inf else -1)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m, q;
	cin >> n >> m >> q;

	vector<vector<int>> path(n + m);

	for (int u = 0; u < n; u ++) {
		int k;
		cin >> k;
		while (k --) {
			int v; cin >> v; v --;
			path[u].emplace_back(n + v);
			path[n + v].emplace_back(u);
		}
	}

	vector<vector<int>> dis(m, vector<int>(n + m, -1));

	for (int x = 0; x < m; x ++) {
		dis[x][n + x] = 0;
		queue<int> q; q.push(n + x);

		while (!q.empty()) {
			int u = q.front(); q.pop();
			for (auto &v: path[u]) {
				if (dis[x][v] == -1) {
					dis[x][v] = dis[x][u] + 1;
					q.push(v);
				}
			}
		}
	}

	int inf = 1e6;

	while (q --) {
		int u, v;
		cin >> u >> v;
		u --, v --;

		int ans = inf;

		for (int i = 0; i < m; i ++) {
			if (dis[i][u] != -1 && dis[i][v] != -1) {
				ans = min(ans, dis[i][u] + dis[i][v]);
			}
		}

		cout << (ans < inf ? ans / 2 - 1 : -1) << '\n';
	}

	return 0;
}
```
