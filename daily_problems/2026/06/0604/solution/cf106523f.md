**提示 1：** 首先 $C(x)$ 得满足什么条件？

**提示 2：** 枚举可行的 $C(x)$ 。

由于 $A(x)C(x)=B(x)$ ，所以 $A(1)C(1)=B(1)$ ，于是 $C(1)=1$ 。而 $C$ 每个系数都是非负整数，所以 $C(x)$ 只能是 $x^a$ 的形式。

所以枚举 $C(x)=x^0,x^1,\dots,x^{n-1}$ ，设枚举的是 $x^a$ 。

也就是下标子集中如果选取了 $idx$ ，则数组中得选取 $idx-a$ 。同样的，数组中如果选取了 $val$ ，则子集中必然选择了下标 $val+a$ 。

把这样的关系考虑成边。因为下标是唯一确定的，所以从下标出发。同时每个下标只连出来一条边，甚至是个基环树。

考虑 $1\leq i\leq n$ ，则从 $i$ 向 $nums[i]+a$ 连边，表示选了 $i$ 下标后 $nums[i]+a$ 这个下标也得选。因此子集满足要求的条件是，每个点一旦被选，其指向的点也得被选。

这样最后选取的子集一定是一些环构成的。

所以统计这个图中的环的个数 $x$ ，方案数就是 $2^x-1$ ，最后关于不同的 $a$ 相加即可。

时间复杂度为 $\mathcal{O}(n^2)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    nums = LGMI()
    
    mod = 998244353
    pw2 = [1] * (n + 1)
    
    for i in range(n):
        pw2[i + 1] = pw2[i] * 2 % mod
    
    ans = 0
    uf = UnionFind(n + 1)
    
    for i in range(n):
        uf.init()
    
        for j in range(n):
            uf.merge(j, fmin(nums[j] + i, n))
        
        cur = 0
        
        for j in range(n):
            if uf.find(j) == j and uf.find(j) != uf.find(n):
                cur += 1
        
        ans += pw2[cur] - 1
        ans %= mod
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x, x --;

	int mod = 998244353;

	vector<int> pw2(n + 1, 1);
	for (int i = 1; i <= n; i ++) pw2[i] = pw2[i - 1] * 2 % mod;

	int ans = 0;
	atcoder::dsu uf(n + 1);

	for (int i = 0; i < n; i ++) {
		uf.init();

		for (int j = 0; j < n; j ++) uf.merge(j, min(nums[j] + i, n));

		int cur = 0;
		for (int j = 0; j < n; j ++) {
			if (uf.leader(j) == j && uf.leader(j) != uf.leader(n)) {
				cur ++;
			}
		}

		ans += pw2[cur] - 1;
		ans %= mod;
	}

	cout << ans;

	return 0;
}
```
