**提示 1：** 第 $i$ 个人的获胜概率如何？何时取最大值？

假设第 $i$ 个人获胜，则前 $i-1$ 个人都选错过。

每个人错的位置两两不同，总共有 $C_m^{i-1}$ 种方案。

而真正二选一的猜测共有 $m$ 轮，等可能事件有 $2^m$ 种，因此概率是 $C_m^{i-1}/2^m$ 。

（其实本质上就是一个长度为 $n$ 的由猜对猜错构成的序列，其中猜错的刚好有 $i-1$ 个的概率。这里每一种情况都对应于一种 $i$ 的获胜情况。）

组合数在中间的时候取最大值，因此使得组合数最大化的最小的 $i=\lfloor m/2\rfloor +1$ 。当然选取的 $i$ 不超过 $n$ ，如果 $n$ 没有到上述数值，则关于 $i$ 单调，取 $n$ 即可。

综上，输出 $\min(\lfloor m/2\rfloor +1, n)$ 。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        outs.append(fmin(n, m // 2 + 1))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, m;
		cin >> n >> m;
		cout << min(n, m / 2 + 1) << '\n';
	}

	return 0;
}
```
