**提示 1：** 怎样的一对元素会形成引用？

**提示 2：** 如何最大化？

只要是不同天的一对就可以形成一对引用。

所以我们要最大化这件事，因此应该让组内的对越少越好。

如果某两组满足一个比另一个大 $2$ ，且前者数量可以减少 $1$ ，后者可以增加 $1$ ，则可以减少组内对的数量。

所以最后的分配是有一个数 $M$ ，每个元素取 $\min(a_i,M)$ 或者 $M+1$ .

而 $M$ 是可以二分确定的，求最大的使得 $\min(a_i,M)$ 不超过数量上界的 $M$ 即可。剩余的数量分配为一堆 $+1$ 即可。

时间复杂度为 $\mathcal{O}(n\log M)$ 。（这里认为 $n$ 是数组长度，与题中定义不同）

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        k, n = MII()
        nums = LII()
        
        l, r = 0, 10 ** 9
        while l <= r:
            mid = (l + r) // 2
            
            val = 0
            for x in nums:
                val += fmin(x, mid)
            
            if val > k: r = mid - 1
            else: l = mid + 1
    
        ans = [fmin(v, r) for v in nums]
        k -= sum(ans)
        
        for i in range(n):
            if ans[i] < nums[i] and k:
                k -= 1
                ans[i] += 1
        
        outs.append(' '.join(map(str, ans)))
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		long long k;
		int n;
		cin >> k >> n;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;

		int l = 0, r = 1e9;

		while (l <= r) {
			int mid = (l + r) / 2;
			long long val = 0;

			for (auto &x: nums) val += min(mid, x);

			if (val > k) r = mid - 1;
			else l = mid + 1;
		}

		vector<int> ans(n);
		for (int i = 0; i < n; i ++) {
			ans[i] = min(nums[i], r);
			k -= ans[i];
		}

		for (int i = 0; i < n; i ++) {
			if (ans[i] < nums[i] && k) {
				ans[i] ++;
				k --;
			}
		}

		for (int i = 0; i < n; i ++) cout << ans[i] << " \n"[i == n - 1];
	}

	return 0;
}
```
