**提示 1：** 数组有何性质？我们要求的是什么方程的解的个数？

首先，把乘积的那个等式相邻两项拎出来。因为数组的每一项都是正数，所以约掉相同的部分，可以得到这个数组满足 $a_i=a_{i+T}$ 。

于是，前 $T$ 项确定了，则整个数组也确定了。

前 $N$ 个数中，这 $T$ 项分别出现 $f_i$ 次，则 $f_i$ 最多只有两个取值。

如果出现次数为 $f$ 的数的和为 $X$ ，而数的个数是 $Y$ ，则 $Y$ 个正整数凑出来 $X$ 的方案数可以用隔板法，即 $X$ 个物品， $X-1$ 个空位，放入 $Y-1$ 个隔板就能拆出 $Y$ 组，也就得到了分为 $Y$ 组的方案数。因此共有 $C_{X-1}^{Y-1}$ 的方案。

如果 $f_i$ 只有一个取值，则周期内元素的和已经确定，用上面的方法就行。

否则，设两个频率是 $V_1,V_2$ ，且分别有 $x, T-x$ 个这样的数。设两组的在前 $T$ 项的和为 $W_1,W_2$ ，则 $W_1\times V_1+W_2\times V_2=K$ 。

所以枚举 $W_1$ 可以解出所有的 $W_1,W_2$ ，再对 $C_{W_1-1}^{x-1}\times C_{W_2-1}^{T-x-1}$ 求和即可。

时间复杂度为 $\mathcal{O}(M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 5 * 10 ** 6 + 5
    mod = 10 ** 9 + 7
    
    f = Factorial(M, mod)
    
    k, s, t = MII()
    x, y = divmod(k, t)
    
    ans = 0
        
    for i in range(s + 1):
        if (s - x * i) % (x + 1) == 0:
            j = (s - x * i) // (x + 1)
            
            ans += f.combi(i - 1, t - y - 1) * f.combi(j - 1, y - 1)
            ans %= mod
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 5e6 + 5, mod = 1e9 + 7;

	vector<int> f(M), g(M);

	f[0] = 1;
	for (int i = 1; i < M; i ++) f[i] = 1ll * i * f[i - 1] % mod;
	g[M - 1] = quickPow(f[M - 1], mod - 2, mod);
	for (int i = M - 1; i > 0; i --) g[i - 1] = 1ll * i * g[i] % mod;

	int k, s, t;
	cin >> k >> s >> t;

	int x = k / t, y = k % t, ans = 0;

	auto comb = [&] (int x, int y) -> int {
		if (x == y) return 1;
		if (y < 0 || x < 0 || y > x) return 0;
		return 1ll * f[x] * g[y] % mod * g[x - y] % mod;
	};

	for (int i = 1; i <= s; i ++) {
		if (x * i > s) break;
		if ((s - i * x) % (x + 1) == 0) {
			int j = (s - i * x) / (x + 1);
			ans += 1ll * comb(i - 1, t - y - 1) * comb(j - 1, y - 1) % mod;
			ans %= mod;
		}
	}

	cout << ans;

	return 0;
}
```
