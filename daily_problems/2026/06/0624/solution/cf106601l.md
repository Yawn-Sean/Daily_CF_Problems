**提示 1：** 数字要满足什么条件？

**提示 2：** 如何找？

首先，这个博弈过于简单了，就是要求每个数位的异或和为 $0$ 。

而如果数位的数量大于等于 $3$ ，那么 $B^2\leq N\leq 10^{12}$ ，所以 $B\leq 10^6$ ，可以直接枚举判断。

否则，数位如果恰好为 $1$ ，则要求两个数位完全一致，不妨设为 $x$ ，则 $x\times B+x=x\times (B+1)$ ，而 $x\leq B-1$ ，所以我们只需将 $n$ 拆分为两个数的乘积，且这两个数差至少为 $2$ ，其一分配给 $x$ ，其二分配给 $B+1$ 即可。这一部分是可以枚举 $x$ 的，因为 $x$ 也不可能超过 $10^6$ 。

时间复杂度为 $\mathcal{O}(X\sqrt{\log X})$ ，如果你选择合适的分块位置。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    x = II()
    ans = 0
    
    for i in range(2, 10 ** 6 + 5):
        cnt = 0
        xor_val = 0
        cur = x
        
        while cur:
            cnt += 1
            xor_val ^= cur % i
            cur //= i
        
        if cnt <= 2: break
        
        if xor_val == 0:
            ans += 1
    
    for i in range(1, 10 ** 6 + 5):
        if i * (i + 2) > x: break
        if x % i == 0:
            ans += 1
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	long long n;
	cin >> n;

	int ans = 0;

	for (int i = 2; i <= 1'000'000; i ++) {
		int xor_val = 0, cnt = 0;
		long long cur = n;

		while (cur) {
			xor_val ^= cur % i;
			cnt ++;
			cur /= i;
		}

		if (cnt <= 2) break;
		if (xor_val == 0) ans ++;
	}

	for (int i = 1; i <= 1'000'000; i ++) {
		if (1ll * i * (i + 2) > n) break;
		if (n % i == 0) ans ++;
	}

	cout << ans;

	return 0;
}
```
