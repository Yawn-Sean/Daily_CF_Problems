**提示 1：** 我们要维护的结构需要满足什么操作？怎么维护？

我们需要支持一个结构，支持操作：

- 整体加上一个数值。

- 取出最大元素。

- 插入元素。

如果只是进行后两种操作，用优先队列即可。

最前面的那个呢？整体加一个 lazy 标记对整个结构进行的操作。

具体而言，执行操作如下：

- 整体加上一个数值：对 lazy 标记增加一个数值。

- 取出最大元素：用优先队列取出最大元素，加上 lazy 就是其当前数值。

- 插入元素：将该元素减去 lazy 后插入优先队列，这样就跟优先队列的其他元素一样，加上 lazy 才是其真实数值。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    nums = LII()
    
    pq = [-x for x in nums]
    heapify(pq)
    lazy = 0
    
    for v in MII():
        lazy += v
        val = -heappop(pq) + lazy
        val -= val // 2
        heappush(pq, -(val - lazy))
    
    print(sum(-x + lazy for x in pq))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	priority_queue<long long> pq;
	long long total = 0, lazy = 0;

	for (int i = 0; i < n; i ++) {
		long long x;
		cin >> x;
		total += x;
		pq.push(x);
	}

	for (int i = 0; i < n; i ++) {
		long long x;
		cin >> x;
		lazy += x;

		auto v = pq.top() + lazy; pq.pop();
		total += x * n - v / 2;
		v -= v / 2;
		pq.push(v - lazy);
	}

	cout << total;

	return 0;
}
```
