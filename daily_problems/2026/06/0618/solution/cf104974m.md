**提示 1：** 连通块数量如何表示呢？

**提示 2：** 如何维护？如何求答案？

连通块个数等于所有满足自己被选取但自己的父结点没被选取的结点的数量。

一个结点 $u$ 被选取且 $parent[u]$ 没被选取，则 $i$ 不是 $nums[u]$ 的因子，但是需要是 $nums[parent[u]]$ 的因子（这样父结点才能被删掉）。

所以，不妨考虑成需要是 $nums[parent[u]]$ 的因子，但不是 $nums[u]$ 的因子，也就不是 $\mathrm{gcd}(nums[u], nums[parent[u]])$ 的因子

所以只需看 $i$ 是多少个 $nums[parent[u]]$ 的因子，又是多少个 $\mathrm{gcd}(nums[u], nums[parent[u]])$ 的因子。

这件事只需将两件事在一个长度为 $10^6+1$ 的数组的对应数值的位置标记，查询 $i$ 时，遍历 $i,2i,\dots$ ，看标记了多少次。

这里你不需要真的找每一个点的父结点是啥。每条边都会计算 $\mathrm{gcd}(nums[u], nums[v])$ 一次。

而每个点本身会作为 $parent$ 出现几次只跟这个点的度数有关。

最后注意，根节点需要特判。

时间复杂度为 $\mathcal{O}((n+M)\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, k = MII()
    nums = LII()
    
    M = 10 ** 6 + 5
    
    notes = [0] * M
    
    for _ in range(n - 1):
        u, v = GMI()
        notes[nums[u]] += 1
        notes[nums[v]] += 1
        notes[math.gcd(nums[u], nums[v])] -= 1
    
    for x in nums:
        notes[x] -= 1
    
    notes[nums[0]] += 1
    
    ans = []
    
    for i in range(1, k + 1):
        res = 0
        for j in range(i, M, i):
            res += notes[j]
        if nums[0] % i:
            res += 1
        ans.append(res)
    
    print(' '.join(map(str, ans)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	priority_queue<long long> pq;
	long long total = 0, lazy = 0;

	for (int i = 0; i < n; i ++) {
		long long x;
		cin >> x;
		total += x;
		pq.push(x);
	}

	for (int i = 0; i < n; i ++) {
		long long x;
		cin >> x;
		lazy += x;

		auto v = pq.top() + lazy; pq.pop();
		total += x * n - v / 2;
		v -= v / 2;
		pq.push(v - lazy);
	}

	cout << total;

	return 0;
}
```
