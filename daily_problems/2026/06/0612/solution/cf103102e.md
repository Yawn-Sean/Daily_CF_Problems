**提示 1：** 如何用前缀和来算一个子段的取模结果呢？

判断一个子数组是否满足要求有啥快速的方法吗？

我们可以维护前缀 $v_1$ 和以及平方的前缀和 $v_2$ 。则 $(v_1[r+1]-v_1[l])^2-(v_2[r+1]-v_2[l])$ 就是要求的东西的两倍。要判断它是否是 $3$ 的倍数，只需看这四个变量关于 $3$ 的取模的结果。

所以维护前缀中 $v_1[i]\bmod 3=x, v_2[i]\bmod 3=y$ 的个数记录在 $(x,y)$ 位置上。这样我们新遍历一个新的位置的时候，就枚举前面的位置的 $(x,y)$ ，如果满足条件，把对应 $(x,y)$ 的计数加入到答案中即可。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    nums = LII()
    
    ans = 0
    
    cnt = [[0] * 3 for _ in range(3)]
    cnt[0][0] = 1
    
    cur1 = 0
    cur2 = 0
    
    for x in nums:
        x %= 3
        cur1 = (cur1 + x) % 3
        cur2 = (cur2 + x * x) % 3
        
        for i in range(3):
            for j in range(3):
                v1 = (cur1 - i) % 3
                v2 = (cur2 - j) % 3
                
                if v1 * v1 % 3 == v2:
                    ans += cnt[i][j]
        
        cnt[cur1][cur2] += 1
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<vector<int>> cnt(3, vector<int>(3, 0));
	long long ans = 0;
	int cur1 = 0, cur2 = 0;

	cnt[0][0] = 1;

	while (n --) {
		int x;
		cin >> x;

		x %= 3;
		cur1 = (cur1 + x) % 3;
		cur2 = (cur2 + x * x) % 3;

		for (int i = 0; i < 3; i ++) {
			for (int j = 0; j < 3; j ++) {
				int v1 = (cur1 - i) % 3;
				int v2 = (cur2 - j) % 3;

				if ((v1 * v1 - v2) % 3 == 0) ans += cnt[i][j];
			}
		}

		cnt[cur1][cur2] ++;
	}

	cout << ans;

	return 0;
}
```
