**提示 1：** 字典序最优，所以显然依次确定。

整体思路就是一个个填上最小的字符。

当我们填上一个字符的时候，其邻居就排除掉一个字符了。如果这个邻居只有一种选择了，就填上，继续往它的邻居走。

整体是个 DFS 的过程。这样做为啥是对的呢？为啥不会导致构造出矛盾呢？

考虑每次这样填，最后没填的部分就是一系列的树，其中树上每个点都至少有两个选择。

则如果下一次钦定树上某个点的选择（认定其为根），则从这个点开始 DFS，这棵树上每个点只需要不等于父结点，而其至少有两个选择，因此肯定不会无法构造。

所以每次贪心填完字母都不会导致无法构造。

整体有 $\mathcal{O}(n)$ 个状态要记录，就是某个位置的可行的选择，而 DFS 的过程中只会访问每个状态 $\mathcal{O}(1)$ 次，因此整体复杂度就是线性的。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    path = [[] for _ in range(n)]
    
    for _ in range(n - 1):
        u, v = GMI()
        path[u].append(v)
        path[v].append(u)
    
    possible = [[1] * n for _ in range(3)]
    ans = [-1] * n
    
    for i in range(n):
        if ans[i] == -1:
            chosen = 0
            for j in range(3):
                if possible[j][i] == 1:
                    chosen = j
                    break
            
            for j in range(3):
                if j != chosen:
                    possible[j][i] = 0
            
            ans[i] = chosen
            que = [i]
            
            for u in que:
                for v in path[u]:
                    if ans[v] == -1:
                        possible[ans[u]][v] = 0
                        
                        c = 0
                        for j in range(3):
                            c += possible[j][v]
                        
                        if c == 1:
                            for j in range(3):
                                if possible[j][v]:
                                    ans[v] = j
                            que.append(v)
    
    print(''.join(chr(ord('a') + x) for x in ans))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<vector<int>> path(n);

	for (int i = 0; i < n - 1; i ++) {
		int u, v;
		cin >> u >> v;
		u --, v --;
		path[u].emplace_back(v);
		path[v].emplace_back(u);
	}

	vector<vector<int>> possible(3, vector<int>(n, 1));
	vector<int> ans(n, -1);

	auto dfs = [&] (auto &self, int u) -> void {
		for (auto &v: path[u]) {
			if (ans[v] == -1) {
				possible[ans[u]][v] = 0;

				int c = 0;
				for (int j = 0; j < 3; j ++) c += possible[j][v];

				if (c == 1) {
					for (int j = 0; j < 3; j ++) {
						if (possible[j][v]) {
							ans[v] = j;
							self(self, v);
						}
					}
				}
			}
		}
	};

	for (int i = 0; i < n; i ++) {
		if (ans[i] == -1) {
			int chosen;
			for (int j = 0; j < 3; j ++) {
				if (possible[j][i]) {
					chosen = j;
					break;
				}
			}
			
			for (int j = 0; j < 3; j ++) {
				if (j != chosen) {
					possible[j][i] = 0;
				}
			}

			ans[i] = chosen;
			dfs(dfs, i);
		}
	}

	for (auto &x: ans) cout << (char)('a' + x);

	return 0;
}
```
