**提示 1：** 考虑用数学归纳法证明可行。先凑 $n$ 。

我们相当于首尾元素依次相配，和为质数。

这个题直接考虑构造似乎不太行。考虑用归纳法证明可以构造。

首先 $n=1,2$ 显然可以构造。

而假设 $n\leq M$ 的情况可以构造。考虑 $n=M+1$ 。

此时，考虑跟 $n$ 配凑的元素 $i$ ，则 $n+1\leq n+i\leq 2n-1$ 。

对于任意正整数 $n$ ，这个范围内有至少一个质数（这个东西不是很好证明，但这个数据范围内你很容易确认）。所以考虑用这个质数 $p$ 。

则 $p-n,p-n+1,\dots,n$ 依次和 $n,n-1,\dots, p-n$ 相配。这样问题就转化为 $1\sim p-1$ 的范围内的配对问题了，也就是归纳假设的情况。因此可以用数学归纳法解决证明。

而真正的构造方案也可以从证明看出。先凑出大的 $n$ ，再将问题不断转化为小范围的问题。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    M = 10 ** 6 + 10
    isPrime = [1] * M
    isPrime[0] = 0
    isPrime[1] = 0
    
    for i in range(2, M):
        if isPrime[i]:
            for j in range(2 * i, M, i):
                isPrime[j] = 0
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
    
        ans = [0] * n
        l, r = 0, n - 1
        
        while n:
            for x in range(1, n + 1):
                if isPrime[n + x]:
                    
                    for i in range(x, n + 1):
                        if i > n + x - i: break
                        ans[l] = i
                        ans[r] = n + x - i
                        l += 1
                        r -= 1
                    
                    n = x - 1
                    break
        
        outs.append(' '.join(map(str, ans)))
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 100;

	vector<int> isPrime(M, 1);
	isPrime[0] = 0, isPrime[1] = 0;

	for (int i = 2; i < M; i ++) {
		for (int j = i * 2; j < M; j += i) {
			isPrime[j] = 0;
		}
	}

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> ans(n);
		int l = 0, r = n - 1;

		while (n) {
			for (int x = 1; x <= n; x ++) {
				if (isPrime[n + x]) {

					for (int i = x; i <= n; i ++) {
						if (i > n + x - i) break;
						ans[l ++] = i;
						ans[r --] = n + x - i;
					}

					n = x - 1;
					break;
				}
			}
		}

		for (auto &v: ans) cout << v << ' '; cout << '\n';
	}

	return 0;
}
```
