**提示 1：** 有些先后顺序没法变。

EZPC 中的逆序对只能一开始就有，不然的话排序操作没法造出来。

所以 E 必须在 C 前面， Z 必须在 P、C 前面， P 必须在 C 前面。

这样就够了吗？答案是肯定的。

上述条件满足时，一定有 ZPC 的子序列。

如果 E 在 C 前 P 后，操作一个包含 PE 的子串，再操作一个包含 ZE 的子串。

如果 E 在 P 前 Z 后，操作一个包含 ZE 的子串。

因此都可行。

时间复杂度为 $\mathcal{O}(|s|)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    xe = ord('e') - ord('a')
    xz = ord('z') - ord('a')
    xp = ord('p') - ord('a')
    xc = ord('c') - ord('a')
    
    for _ in range(t):
        s = [ord(c) - ord('a') for c in I()]
    
        pos = [0] * 26
        for i in range(26):
            pos[s[i]] = i
        
        pe = pos[xe]
        pz = pos[xz]
        pp = pos[xp]
        pc = pos[xc]
        
        if pe < pc and pz < pp and pp < pc: outs.append('YES')
        else: outs.append('NO')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		string s;
		cin >> s;

		vector<int> pos(26);
		for (int i = 0; i < 26; i ++) pos[s[i] - 'a'] = i;

		int pe = pos['e' - 'a'];
		int pz = pos['z' - 'a'];
		int pp = pos['p' - 'a'];
		int pc = pos['c' - 'a'];

		if (pe < pc && pz < pp && pp < pc) cout << "YES\n";
		else cout << "NO\n";
	}

	return 0;
}
```
