**提示 1：** 考虑中间的点，求出上界。

**提示 2：** 偶数的情况比较容易，奇数基于此进行构造。

考虑中间的点，到中间的点的距离不超过 $\lfloor n/2\rfloor\sqrt{2}$ ，就给出了合法的 $d$ 的上界。

如果是偶数的情况，把整个盘面分为左上左下右上右下四个部分，把左上的点向右向下平移 $n/2$ 个单位跟它配对，右上的点向左向下平移 $n/2$ 个单位跟它配对，这样就满足任意两点距离都刚好是 $n/2\times\sqrt{2}$ 。

奇数呢？我们先取前 $2\lfloor n/2\rfloor$ 像偶数情况一样构造。剩下一条链，总长为 $2n-1$ ，依次标记为第 $i$ 个点。则标号差距为 $n-1$ 或更大的点距离都超过了 $\lfloor n/2\rfloor\sqrt{2}$ ，因此考虑将第 $i+1$ 个点指向第 $(i+n)\bmod (2n-1)+1$ 个点即可。

时间复杂度为 $\mathcal{O}(n^2)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, d = MII()
        
        if (n // 2) * (n // 2) * 2 >= d:
            outs.append('YES')
            
            x = n // 2 * 2
            for i in range(x):
                for j in range(x):
                    outs.append(f'{i + 1} {j + 1} {(i + x // 2) % x + 1} {(j + x // 2) % x + 1}')
            
            if n % 2:
                for i in range(1, n + 1):
                    outs.append(f'{i} {n} {n} {n + 1 - i}')
                
                for i in range(1, n):
                    outs.append(f'{n} {i} {n - i} {n}')
        else:
            outs.append('NO')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, d;
		cin >> n >> d;

		if ((n / 2) * (n / 2) * 2 >= d) {
			cout << "YES\n";

			int x = n / 2 * 2;
			for (int i = 0; i < x; i ++) {
				for (int j = 0; j < x; j ++) {
					cout << i + 1 << ' ' << j + 1 << ' ' << (i + x / 2) % x + 1 << ' ' << (j + x / 2) % x + 1 << '\n';
				}
			}

			if (n & 1) {
				for (int i = 1; i <= n; i ++)
					cout << i << ' ' << n << ' ' << n << ' ' << n + 1 - i << '\n';
				for (int i = 1; i < n; i ++)
					cout << n << ' ' << i << ' ' << n - i << ' ' << n << '\n';
			}
		}
		else cout << "NO\n";
	}

	return 0;
}
```
