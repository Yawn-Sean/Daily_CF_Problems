**提示 1：** 这是个字符串题。

我们只需判断是否下面的字符串 $T$ 的旋转字符串是否是已知字符串 $S$ 。

为此，只需判断 $T$ 是否是 $S+S$ 的子串即可。这相当于是把原本旋转字符串的环断为一条链来方便判断。

这件事用 KMP 等方式都能快速解决。

你也可以开头将 $n$ 个旋转字符串插入到一个字典树中，后面判断，从正确性上来说也是可行的，虽然可能时间空间上会不太好。

时间复杂度为 $\mathcal{O}(nm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    def prep(p):
        pi = [0] * len(p)
        j = 0
        for i in range(1, len(p)):
            while j != 0 and p[j] != p[i]:
                j = pi[j - 1]
            if p[j] == p[i]:
                j += 1
            pi[i] = j
        return pi
    
    n, m = MII()
    nums = LII()
    
    for i in range(1, n + 1):
        if m in prep(LII() + [-1] + nums + nums):
            print(i)
            break
    else:
        print(-1)
```

C++ 做法如下——

```cpp []
vector<int> prep(const vector<int> &p) {
    vector<int> pi(p.size(), 0);
    int j = 0;
    for (int i = 1; i < (int)p.size(); i++) {
        while (j != 0 && p[j] != p[i])
            j = pi[j - 1];
        if (p[j] == p[i])
            j ++;
        pi[i] = j;
    }
    return pi;
}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m;
	cin >> n >> m;

	vector<int> nums(m);
	for (auto &x: nums) cin >> x;

	for (int i = 1; i <= n; i ++) {
		vector<int> vals(m);
		for (auto &x: vals) cin >> x;
		vals.emplace_back(-1);
		vals.insert(vals.end(), nums.begin(), nums.end());
		vals.insert(vals.end(), nums.begin(), nums.end());

		auto kmp = prep(vals);
		for (auto &v: kmp) if (v == m) return cout << i, 0;
	}

	cout << -1;

	return 0;
}
```
