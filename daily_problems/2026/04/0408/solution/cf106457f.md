**提示 1：** 对于每个不同的起点，我们找到第一个起跳的操作。

**提示 2：** 起跳后的部分是可以预处理的。

首先，如果从 $i$ 出发，完全没到达过 $X$ ，则直接从 $i$ 走 $\sum a_i$ 步到达目标位置就行。

否则，我们找到第一个使得起跳发生的操作，则到这个操作的总步数关于 $N$ 取模的结果是 $X-i$ 。于是我们只需找到满足对应条件的前缀即可，这件事可以用哈希表，记录每个前缀的取模结果。当然这里数字范围很小，直接用数组显然也行。

而一旦起跳后，我们实际需要研究的就是从第 $i$ 次操作起跳后最终能到达的位置，这样的问题实际上只有 $k$ 个。

我们从后往前处理，我们可以利用上面类似的逻辑，对于每个起跳的位置，可以得到下一次起跳位置需要满足的前缀和条件，进而找到下一次起跳的位置。

而我们已经知道了下一次起跳位置对应的终点（因为我们是从后往前处理的），所以赋值给当前的位置即可。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, q = MII()
    cnt = [[0] * n for _ in range(n)]
    
    for _ in range(q):
        l, r = GMI()
        cnt[l][r] += 1
    
    for l in range(n):
        for r in range(n - 1, 0, -1):
            cnt[l][r - 1] += cnt[l][r]
    
    for l in range(1, n):
        for r in range(n):
            cnt[l][r] += cnt[l - 1][r]
    
    dp = [[0] * n for _ in range(n)]
    
    for i in range(n):
        dp[i][i] = cnt[i][i]
    
    for step in range(1, n):
        for l in range(n - step):
            r = l + step
            dp[l][r] = fmax(dp[l + 1][r], dp[l][r - 1]) + cnt[l][r]
    
    print(dp[0][n - 1])
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, q;
	cin >> n >> q;

	vector<vector<int>> cnt(n, vector<int>(n, 0)), dp(n, vector<int>(n, 0));

	while (q --) {
		int l, r;
		cin >> l >> r;
		l --, r --;
		cnt[l][r] ++;
	}

	for (int l = 0; l < n; l ++) {
		for (int r = n - 1; r > 0; r --) {
			cnt[l][r - 1] += cnt[l][r];
		}
	}

	for (int l = 1; l < n; l ++) {
		for (int r = 0; r < n; r ++) {
			cnt[l][r] += cnt[l - 1][r];
		}
	}

	for (int i = 0; i < n; i ++) dp[i][i] = cnt[i][i];

	for (int l = n - 1; l >= 0; l --) {
		for (int r = l + 1; r < n; r ++) {
			dp[l][r] = max(dp[l + 1][r], dp[l][r - 1]) + cnt[l][r];
		}
	}

	cout << dp[0][n - 1];

	return 0;
}
```
