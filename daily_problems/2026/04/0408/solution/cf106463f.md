**提示 1：** 排列如何构造？

**提示 2：** 如何利用上述构造思路设计 DP 并进行转移？

首先，一个排列的不同子数组的 MEX 大概是什么样子呢？

考虑 $0,1,\dots,i$ 的位置，则 MEX 要大于等于 $i+1$ 就必须区间包含这些位置的最小值和最大值。

因此我们应该让这些位置排到一起。更具体的，我们先把 $0$ 放中间，别的 $1,2,\dots$ 应该依次放到前面或者后面。否则，如果某个 $i$ 不满足条件，我们把 $i$ 交换到相邻的地方，一定不使得 MEX 变小。

因此，考虑 $[l,r]$ 刚好是 $0,1,\dots,r-l$ 的排列的情况对结果 MEX 和的贡献，设为 $dp[l][r]$ 。

我们将整体 MEX 的和拆分为 $MEX \geq 1,2,\dots$ 的区间个数之和。其中 $MEX \geq i+1$ 的区间个数等于包含 $0,1,\dots,i$ 中所有元素的区间个数。

于是 $dp[l][r]$ 可以拆分为两部分。第一部分——一种情况是 $[l+1,r]$ 包含了 $0,\dots,r-l-1$ 中所有元素，另一种情况是 $[l,r-1]$ 包含了 $0,\dots,r-l-1$ 中所有元素，两种情况的贡献取最大值，也就包括了所有 $MEX\geq 1,2,\dots,r-l$ 的区间数量和的贡献。而区间 $[l,r]$ 会产生新的贡献，因为其包含了 $0,1,\dots,r-l$ 中的所有数字，产生了 $MEX\geq r-l+1$ 的区间数量的贡献。

具体而言前者等于 $\max(dp[l+1][r],dp[l][r-1])$ ，后者是包含 $[l,r]$ 的区间的个数。

我们就需要快速算出来包含 $[l,r]$ 的区间个数，这件事类似于前缀和（ $l'\leq l,r'\geq r$ ，如果把 $r',r$ 变成 $n-r',n-r$ 就是真的完全前缀和了）。

所以 $dp[l][r]=\max(dp[l+1][r],dp[l][r-1])+包含 [l,r] 的区间的个数$ 。区间 DP 即可。

时间复杂度为 $\mathcal{O}(n^2+q)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, q = MII()
    cnt = [[0] * n for _ in range(n)]
    
    for _ in range(q):
        l, r = GMI()
        cnt[l][r] += 1
    
    for l in range(n):
        for r in range(n - 1, 0, -1):
            cnt[l][r - 1] += cnt[l][r]
    
    for l in range(1, n):
        for r in range(n):
            cnt[l][r] += cnt[l - 1][r]
    
    dp = [[0] * n for _ in range(n)]
    
    for i in range(n):
        dp[i][i] = cnt[i][i]
    
    for step in range(1, n):
        for l in range(n - step):
            r = l + step
            dp[l][r] = fmax(dp[l + 1][r], dp[l][r - 1]) + cnt[l][r]
    
    print(dp[0][n - 1])
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, q;
	cin >> n >> q;

	vector<vector<int>> cnt(n, vector<int>(n, 0)), dp(n, vector<int>(n, 0));

	while (q --) {
		int l, r;
		cin >> l >> r;
		l --, r --;
		cnt[l][r] ++;
	}

	for (int l = 0; l < n; l ++) {
		for (int r = n - 1; r > 0; r --) {
			cnt[l][r - 1] += cnt[l][r];
		}
	}

	for (int l = 1; l < n; l ++) {
		for (int r = 0; r < n; r ++) {
			cnt[l][r] += cnt[l - 1][r];
		}
	}

	for (int i = 0; i < n; i ++) dp[i][i] = cnt[i][i];

	for (int l = n - 1; l >= 0; l --) {
		for (int r = l + 1; r < n; r ++) {
			dp[l][r] = max(dp[l + 1][r], dp[l][r - 1]) + cnt[l][r];
		}
	}

	cout << dp[0][n - 1];

	return 0;
}
```
