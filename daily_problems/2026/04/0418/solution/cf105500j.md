**提示 1：** 将求和式拆分为若干部分。

因为不同的 $p_i$ 的数值之间关系太小了，所以不如单独考虑每个 $p_i$ 。

我们求所有 $p_i=k$ 的部分对答案的贡献，则贡献数值是 $k^2$ 乘 $p_i=V$ 的出现次数之和。

出现次数之和也不好求，我们可以转而求出现次数大于等于 $1,2,\dots,k$ 的方案数之和。

为此，对于出现次数大于等于 $i$ 的方案数，我们只需求和为 $n-iV$ 的 $k-i$ 个元素的方案数。

这件事可以 DP 来求，设 $x$ 个元素和为 $y$ 的方案数为 $dp[x][y]$ ，其中各个元素不记顺序。

则我们考虑集合的构造方式是通过两个步骤进行的 —— 所有元素 $+1$ ，新增元素 $1$ 。容易发现，每个多元集合一一对应于一个操作序列。

所以 $dp[x][y]=dp[x][y-x]+dp[x-1][y-1]$ ，于是可以平方量级解决。

同时这题给的空间不大，可以滚动数组优化空间。

时间复杂度为 $\mathcal{O}(nk)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, k = MII()
    mod = 10 ** 9 + 7
    
    ans = 0
    dp = [0] * (n + 1)
    dp[0] = 1
    
    for i in range(k):
        ndp = [0] * (n + 1)
        
        for j in range(n + 1):
            if i and i + j <= n:
                dp[i + j] += dp[j]
                dp[i + j] %= mod
            if i + 1 <= n and j + 1 <= n:
                ndp[j + 1] += dp[j]
                ndp[j + 1] %= mod
        
        y = k - i
        for x in range(1, n // y + 1):
            ans += x * x * dp[n - y * x] % mod
            ans %= mod
        
        dp = ndp
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, k, mod = 1e9 + 7;
	cin >> n >> k;

	int ans = 0;

	vector<int> dp(n + 1, 0);
	dp[0] = 1;

	for (int i = 0; i < k; i ++) {
		vector<int> ndp(n + 1, 0);

		for (int j = 0; j <= n; j ++) {
			if (i && i + j <= n) dp[i + j] = (dp[i + j] + dp[j]) % mod;
			if (i + 1 <= n && j + 1 <= n) ndp[j + 1] = (ndp[j + 1] + dp[j]) % mod;
		}

		int y = k - i;
		for (int x = 1; x <= n / y; x ++) ans = (ans + 1ll * x * x * dp[n - y * x]) % mod;

		dp.swap(ndp);
	}

	cout << ans;

	return 0;
}
```
