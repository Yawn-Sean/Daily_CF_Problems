**提示 1：** 如果 $w=1$ ，那么相当于就是求逆序对。

**提示 2：** 这题，考虑每一组逆序对，对答案的贡献如何？

先看提示 1 。

于是我们考虑逆序对。每一组逆序对肯定都得经过交换，因此肯定对结果产生 $W_iW_j$ 的贡献。

这些求和是答案吗？答案是肯定的，构造和 $w=1$ 的情况一致，就选取使得逆序对数减小的对进行交换即可。

所以我们对于每个元素，只需求其前面比它 $H$ 大的数的 $W$ 之和。这件事用树状数组很容易完成。（当然用归并排序也行的）

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n = II()
    v1 = LII()
    v2 = LII()
    
    st_range = sorted(range(n), key=lambda x: v1[x])
    
    fen = FenwickTree(n)
    
    ans = 0
    for i in st_range:
        ans += fen.rsum(i, n - 1) * v2[i]
        fen.add(i, v2[i])
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	vector<int> v1(n), v2(n);
	for (auto &x: v1) cin >> x;
	for (auto &x: v2) cin >> x;

	vector<int> order(n);
	iota(order.begin(), order.end(), 0);
	sort(order.begin(), order.end(), [&] (int i, int j) {return v1[i] < v1[j];});

	atcoder::fenwick_tree<int> fen(n);
	long long ans = 0;

	for (auto &i: order) {
		ans += 1ll * fen.sum(i, n) * v2[i];
		fen.add(i, v2[i]);
	}

	cout << ans;

	return 0;
}
```
