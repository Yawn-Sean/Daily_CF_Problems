**提示 2：** 组合恒等式。

将期望拆为，最小值不小于 $1,2,\dots$ 的方案数之和。

也就是 $C_n^k+C_{n-1}^k+\dots+C_k^k$ 。

这个可以这么算：

$C_n^k+C_{n-1}^k+\dots+C_k^k=C_n^k+C_{n-1}^k+\dots+C_k^k+C_k^{k+1}$

再不断把最后两项用组合恒等式 $C_a^b=C_{a-1}^b+C_{a-1}^{b-1}$ 不断进行计算，则得到答案是 $C_{n+1}^{k+1}$ 。

预处理后时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    M = 10 ** 6 + 5
    mod = 998244353
    f = Factorial(M, mod)
    
    t = II()
    outs = []
    
    for _ in range(t):
        n, k = MII()
        outs.append(f.combi(n + 1, k + 1))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 1e6 + 5, mod = 998244353;
	vector<int> f(M), g(M);

	f[0] = 1;
	for (int i = 1; i < M; i ++) f[i] = 1ll * i * f[i - 1] % mod;

	g[M - 1] = quickPow(f[M - 1], mod - 2, mod);
	for (int i = M - 1; i >= 1; i --) g[i - 1] = 1ll * i * g[i] % mod;

	int t;
	cin >> t;

	while (t --) {
		int n, k;
		cin >> n >> k;
		cout << 1ll * f[n + 1] * g[k + 1] % mod * g[n - k] % mod << '\n';
	}

	return 0;
}
```
