**提示 1：** 分别讨论奇数和偶数怎么取。

各种数在最后要求和是奇数的情况只需分为奇数和偶数两类。

只有两种可能：选两个奇数一个偶数；选三个偶数。

分别的方案都很好求，加起来即可。时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    mod = 10 ** 9 + 7
    
    for _ in range(t):
        n = II()
        c0 = n // 2
        c1 = (n + 1) // 2
        outs.append((math.comb(c0, 3) + math.comb(c1, 2) * c0) % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t, mod = 1e9 + 7;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		int c0 = n / 2, c1 = n - c0;
		cout << (1ll * c0 * (c0 - 1) * (c0 - 2) / 6 + 1ll * c0 * c1 * (c1 - 1) / 2) % mod << '\n';
	}

	return 0;
}
```
