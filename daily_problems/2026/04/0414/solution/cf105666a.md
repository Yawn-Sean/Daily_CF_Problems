**提示 1：** 满足条件的数并不多。怎么用一个好的方式找到所有的备选的数？

**提示 2：** 如何判断哪些数是可以到达的？

首先，满足条件的数并不多，为什么？

因为每个数的质因子都只能在 $2,3,5,7$ 中取 —— 不然除法除不掉。直接估计分别的次数的话，每个因子的次数不超过 $\mathcal{O}(\log n)$ ，因此备选数数量不超过 $\mathcal{O}(\log^4 n)$ 。

怎么得到它们呢？我们依次枚举各个质因子。考虑了一部分的因子形成一个备选数的集合，对于当前集合中的数，不断乘新因子，直到突破了 $n$ 的范围。这样就能得到所有可能的数（实在不理解可以看代码）。

最后，可达性可以用 DP 快速得到，就看每个数能否通过除以自己的某一位得到另一个可达数。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    possible_vals = [1]
    bound = II()
    
    for x in [2, 3, 5, 7]:
        for i in range(len(possible_vals)):
            cur = possible_vals[i]
            while cur * x <= bound:
                cur *= x
                possible_vals.append(cur)
    
    k = len(possible_vals)
    d = {v: i for i, v in enumerate(possible_vals)}
    
    dp = [0] * k
    dp[0] = 1
    
    for i in range(k):
        if dp[i]:
            for x in range(2, 10):
                if possible_vals[i] * x <= bound and str(x) in str(possible_vals[i] * x):
                    dp[d[possible_vals[i] * x]] = 1
    
    print(sum(dp))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	vector<long long> possible_vals = {1};
	long long bound; cin >> bound;

	for (auto &x: {2, 3, 5, 7}) {
		int k = possible_vals.size();
		for (int i = 0; i < k; i ++) {
			long long cur = possible_vals[i];
			while (cur <= bound / x) {
				cur *= x;
				possible_vals.emplace_back(cur);
			}
		}
	}

	int k = possible_vals.size();

	unordered_map<long long, int> mp;
	for (int i = 0; i < k; i ++) mp[possible_vals[i]] = i;

	int ans = 0;

	vector<int> dp(k, 0);
	dp[0] = 1;

	for (int i = 0; i < k; i ++) {
		ans += dp[i];
		if (dp[i]) {
			for (int x = 2; x <= 9; x ++) {
				if (possible_vals[i] <= bound / x) {
					long long val = possible_vals[i] * x;
					bool flg = false;

					while (val) {
						if (val % 10 == x) flg = true;
						val /= 10;
					}

					if (flg) dp[mp[possible_vals[i] * x]] = 1;
				}
			}
		}
	}

	cout << ans;

	return 0;
}
```
