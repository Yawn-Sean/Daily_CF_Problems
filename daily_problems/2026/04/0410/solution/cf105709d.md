**提示 1：** 如果没有那个不计重量的东西，就是简单背包。那么多设计一个状态表示那个东西。

先看提示 1，如果没有不计重量这件事，则我们用 $dp[i][x]$ 表示前 $i$ 个物品总重为 $x$ 的情况下的最大价值，这样背包 DP 就做完了。

而这里有一个东西可以不计重量。于是我们用 $dp[i][x][msk]$ 表示，新增状态 $msk$ 表示不计重量这件事有没有被用过。

于是，除了之前的状态转移外，还有 $(i,x,0)$ 到 $(i+1,x,1)$ 的转移，其中要算上新的物品的价值，即用 $dp[i][x][0]+v$ 更新 $dp[i][x][1]$ 。

时间复杂度为 $\mathcal{O}(nm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, m = MII()
    dp = [[0] * (m + 1) for _ in range(2)]
    
    for _ in range(n):
        w, v = MII()
        
        for i in range(m, w - 1, -1):
            dp[1][i] = fmax(dp[1][i], dp[1][i - w] + v)
        
        for i in range(m + 1):
            dp[1][i] = fmax(dp[1][i], dp[0][i] + v)
        
        for i in range(m, w - 1, -1):
            dp[0][i] = fmax(dp[0][i], dp[0][i - w] + v)
    
    print(dp[1][m])
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		int ans = n - 1, cur = 0, v = 0;

		while (n --) {
			int x;
			cin >> x;

			if (cur < 0) {
				cur += v;
				ans ++;
			}

			v = max(v, x);
			cur --;
		}

		cout << ans << '\n';
	}

	return 0;
}
```
