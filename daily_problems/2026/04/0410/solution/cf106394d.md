**提示 1：** 考虑到某个时刻没有耐力了，此时需要增加耐力——事实上可选的增加耐力的方式有哪些？

考虑从前往后模拟。

假设到某个时刻没有耐力了，此时需要找一个耐力补给站。

而此时可选的耐力补给站的单日最大补给是前缀的最大值，为什么？

因为我们可以在之前到达那个位置的时候多补给一次，以使得当前耐力增加对应的数值。

整体可以理解为，我们可以将之前所有的耐力补给存起来，到需要的时候可以任选一个，时空穿越回去，补给后再穿越回来。

所以直接认为补给前缀最大值即可。只需顺序模拟，因此复杂度线性。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
        ans = n - 1
        
        cur = 0
        v = 0
        
        for i in range(n):
            if cur < 0:
                cur += v
                ans += 1
            
            v = fmax(v, nums[i])
            cur -= 1
        
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		int ans = n - 1, cur = 0, v = 0;

		while (n --) {
			int x;
			cin >> x;

			if (cur < 0) {
				cur += v;
				ans ++;
			}

			v = max(v, x);
			cur --;
		}

		cout << ans << '\n';
	}

	return 0;
}
```
