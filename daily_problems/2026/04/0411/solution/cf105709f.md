**提示 1：** 枚举不同的 $\max b$ ，则对应要求的是什么？

我们升序遍历所有的 $\max b$ ，则我们不断引入新的元素。此时我们要求的是当前集合中 $w$ 不超过 $k-\max b$ 的元素的 $v$ 之和。

我们可以维护每个 $w$ 对应的 $v$ 之和，则我们求的就是一个前缀和。这很容易用树状数组做到。

而我们遍历 $\max b$ 时添加元素，也很容易用树状数组进行维护，因此问题解决。

我们只需考虑 $b+w\leq k$ 的部分，因此树状数组只需要开到 $k$ 就行。

时间复杂度为 $\mathcal{O}(n+k\log k)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, k = MII()
    bs = LII()
    ws = LII()
    vs = LII()
    
    tmp = [[] for _ in range(k)]
    
    for i in range(n):
        if bs[i] + ws[i] <= k:
            tmp[bs[i]].append(i)
    
    fen = FenwickTree(k)
    ans = 0
    
    for i in range(1, k):
        for idx in tmp[i]:
            fen.add(ws[idx], vs[idx])
        ans = fmax(ans, fen.rsum(0, k - i))
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, k;
	cin >> n >> k;

	vector<int> bs(n), ws(n), vs(n);
	for (auto &x: bs) cin >> x;
	for (auto &x: ws) cin >> x;
	for (auto &x: vs) cin >> x;

	vector<vector<int>> tmp(k);
	for (int i = 0; i < n; i ++) {
		if (bs[i] + ws[i] <= k) {
			tmp[bs[i]].emplace_back(i);
		}
	}

	atcoder::fenwick_tree<long long> fen(k);
	long long ans = 0;

	for (int i = 1; i < k; i ++) {
		for (auto &idx: tmp[i]) fen.add(ws[idx], vs[idx]);
		ans = max(ans, fen.sum(0, k - i + 1));
	}

	cout << ans;

	return 0;
}
```
