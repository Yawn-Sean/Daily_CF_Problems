**提示 1：** 如果我们知道了最后各个数值的出现顺序，我们如何计数最小交换次数呢？

**提示 2：** 任意两个数字之间必要交换次数如何？

先看提示 1 。

如果 $a$ 出现在 $b$ 前面，那么每一个 $b,a$ 子序列都需要发生一次交换，我们考虑所有的 $\mathcal{O}(M^2)$ 种组合，将这些交换次数加起来就是总交换次数的下界。

这个下界可以达到吗？答案是肯定的，因为不满足最终条件的情况下一定存在相邻的两项是某一个 $b,a$ （这件事可以用反证法证明：如果不存在任何相邻两项的 $b,a$ ，最后序列就已经排好了）。

所以我们只需统计在所有的最终数值排列方案中，最小的 $b,a$ 对数量。

为此可以使用 bitmask DP ，假设前面已经放好了一个子集的数字，枚举下一个数字就可以进行状态转移，同时计算新增的 $b,a$ 对数量。

时间复杂度为 $\mathcal{O}(nM+M^22^M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n = II()
    nums = LGMI()
    
    cnt = [0] * 10
    pairs = [[0] * 10 for _ in range(10)]
    
    for x in nums:
        for i in range(10):
            pairs[i][x] += cnt[i]
        cnt[x] += 1
    
    inf = 10 ** 14
    
    dp = [inf] * (1 << 10)
    dp[0] = 0
    
    for i in range(1 << 10):
        for j in range(10):
            if i >> j & 1: continue
            
            ni = i | (1 << j)
            new_rev = 0
            
            for k in range(10):
                if i >> k & 1:
                    new_rev += pairs[k][j]
            
            dp[ni] = fmin(dp[ni], dp[i] + new_rev)
    
    print(dp[-1])
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	vector<int> cnt(10, 0);
	vector<vector<long long>> pairs(10, vector<long long>(10, 0));

	int n;
	cin >> n;

	while (n --) {
		int x;
		cin >> x;
		x --;

		for (int i = 0; i < 10; i ++)
			pairs[i][x] += cnt[i];
		cnt[x] ++;
	}

	long long inf = 1e14;

	vector<long long> dp(1 << 10, inf);
	dp[0] = 0;

	for (int i = 0; i < (1 << 10); i ++) {
		for (int j = 0; j < 10; j ++) {
			if (i >> j & 1) continue;

			int ni = i | (1 << j);
			long long new_rev = 0;

			for (int k = 0; k < 10; k ++) {
				if (i >> k & 1) {
					new_rev += pairs[k][j];
				}
			}

			dp[ni] = min(dp[ni], dp[i] + new_rev);
		}
	}

	cout << dp.back();

	return 0;
}

```
