**提示 1：** 我们要求的东西是什么呢？

我们要求的是从 $1$ 结点出发能够到达的平均权值最大的环的平均权值。

这个问题可以用 Karp 算法。一般而言，取 $dis[x][v]$ 为 $x$ 步到达 $v$ 的最大路径权值和，则最大的平均权值等于：

$\max_{i}\min_{j}\frac{dis[n][i]-dis[j][i]}{n-j}$

这里只能从 $1$ 结点出发，所以第一层 $dis[0]$ 的部分只有 $1$ 的位置赋初值 $0$ ，其他位置赋值负无穷。注意上述求最大路径权值和的拓展过程中一定保证拓展的起点是可以到达的位置。

时间复杂度为 $\mathcal{O}(nm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, m = MII()
    path = [[] for _ in range(n)]
    
    for _ in range(m):
        u, v, w = MII()
        u -= 1
        v -= 1
        path[u].append(w * n + v)
    
    inf = 10 ** 15
    dis = [[-inf] * n for _ in range(n + 1)]
    
    dis[0][0] = 0
    
    for i in range(n):
        for u in range(n):
            if dis[i][u] != -inf:
                for msk in path[u]:
                    w, v = divmod(msk, n)
                    dis[i + 1][v] = fmax(dis[i + 1][v], dis[i][u] + w)
    
    ans = 0
    for u in range(n):
        if dis[n][u] != -inf:
            tmp = inf
            for i in range(n):
                if dis[i][u] != -inf:
                    tmp = fmin(tmp, (dis[n][u] - dis[i][u]) / (n - i))
            ans = fmax(ans, tmp)
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m;
	cin >> n >> m;

	vector<array<int, 3>> edges(m);
	for (auto &[u, v, w]: edges)
		cin >> u >> v >> w, u --, v --;

	long long inf = 1e15;
	vector<vector<long long>> dis(n + 1, vector<long long>(n, -inf));
	dis[0][0] = 0;

	for (int i = 0; i < n; i ++) {
		for (auto &[u, v, w]: edges) {
			if (dis[i][u] != -inf) {
				dis[i + 1][v] = max(dis[i + 1][v], dis[i][u] + w);
			}
		}
	}

	long double ans = 0;

	for (int u = 0; u < n; u ++) {
		if (dis[n][u] != -inf) {
			long double tmp = inf;
			for (int i = 0; i < n; i ++) {
				if (dis[i][u] != -inf) {
					tmp = min(tmp, (long double)(dis[n][u] - dis[i][u]) / (n - i));
				}
			}
			ans = max(ans, tmp);
		}
	}

	cout << fixed << setprecision(15) << ans;

	return 0;
}
```
