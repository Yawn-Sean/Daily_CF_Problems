**提示 1：** $k$ 个一循环，所以考虑 $k$ 步后到哪里。

**提示 2：** 不同情况，如何理解？

首先看提示 1，我们对每个格子先进行模拟。

如果在模拟的过程中已经到了 $S$ ，则标记。否则假设从 $x$ 格子到了 $y$ 。

接下来对这些位置连从 $x$ 到 $y$ 的有向边，也就是我们要看每个点能否到达 $S$ 。

一种直接的方式是建反图，看 $S$ 出发能到达哪些位置。

但实际上这个图中不存在环，所以可以直接将图视为无向图，看每个连通部分里是否有 $S$ 。这件事也有更快的办法——把所有 $S$ 连向一个虚拟结点，这样只需判是否跟虚拟节点在同一个连通块即可。

时间复杂度为 $\mathcal{O}(n^2m)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    d = {'.': 0, 'S': 1, '#': 2}
    dir = {
        'N': (-1, 0),
        'S': (1, 0),
        'W': (0, -1),
        'E': (0, 1)
    }
    
    n, m = MII()
    grid = [[d[c] for c in I()] for _ in range(n)]
    s = [dir[c] for c in I()]
    
    uf = UnionFind(n * n + 1)
    
    def f(i, j): return i * n + j
    
    for i in range(n):
        for j in range(n):
            if grid[i][j] == 1:
                uf.merge(f(i, j), n * n)
            
            elif grid[i][j] == 0:
                x, y = i, j
                
                for dx, dy in s:
                    nx = x + dx
                    ny = y + dy
                    if 0 <= nx < n and 0 <= ny < n and grid[nx][ny] != 2:
                        x = nx
                        y = ny
                        if grid[nx][ny] == 1:
                            break
                
                uf.merge(f(i, j), f(x, y))
    
    ans = [[0] * n for _ in range(n)]
    
    for i in range(n):
        for j in range(n):
            if uf.find(f(i, j)) == uf.find(n * n):
                ans[i][j] = 1
    
    print('\n'.join(''.join(map(str, x)) for x in ans))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	vector<pair<int, int>> dirs(300);
	dirs['N'] = {-1, 0};
	dirs['S'] = {1, 0};
	dirs['W'] = {0, -1};
	dirs['E'] = {0, 1};

	int n, m;
	cin >> n >> m;

	vector<string> grid(n);
	for (auto &s: grid) cin >> s;

	string op;
	cin >> op;

	atcoder::dsu uf(n * n + 1);
	auto f = [&] (int i, int j) {return i * n + j;};

	for (int i = 0; i < n; i ++) {
		for (int j = 0; j < n; j ++) {
			if (grid[i][j] == 'S') uf.merge(f(i, j), n * n);
			else if (grid[i][j] == '.') {
				int x = i, y = j;

				for (auto &ch: op) {
					auto [dx, dy] = dirs[ch];
					int nx = x + dx, ny = y + dy;

					if (nx >= 0 && nx < n && ny >= 0 && ny < n && grid[nx][ny] != '#') {
						x = nx, y = ny;
						if (grid[x][y] == 'S') break;
					}
				}

				uf.merge(f(i, j), f(x, y));
			}
		}
	}

	vector<string> ans(n, string(n, '0'));

	for (int i = 0; i < n; i ++) {
		for (int j = 0; j < n; j ++) {
			if (uf.leader(f(i, j)) == uf.leader(n * n)) {
				ans[i][j] = '1';
			}
		}
	}

	for (auto &x: ans) cout << x << '\n';

	return 0;
}
```
