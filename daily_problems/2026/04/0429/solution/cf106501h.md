**提示 1：** $b$ 的数值很少，所以可以根据 $b$ 的数值分为两段。

**提示 2：** 可以视为前后缀。先取该取的数，再看剩下的数。

$b$ 只有 $1,2$ 两种取值。所以我们枚举分割位置，前面只选 $1$ 类数，后面只选 $2$ 类数。

我们先将所有 $a$ 中元素减去 $K$ ，这样我们就只需考虑选的子序列的和非负了。

假设我们固定了边界点，则前面部分的 $1$ 类数中的非负数肯定都要，后面部分的 $2$ 类数种的非负数肯定都要。对于剩余的前面部分的 $1$ 类数，后面部分的 $2$ 类数，我们要选的尽可能多，同时保证整体的和不小于 $0$ 。

所以肯定选绝对值小的一些数即可。我们不妨维护这些数的绝对值。

但分割点变动的情况下怎么处理这件事呢？

我们不妨让分割点从左到右移动，那么第一部分的非负数的和是很容易维护的。第二部分，我们要维护一些数的集合，保证能算出来绝对值的和不超过某个上界。

这个集合需要不断增删元素（因为在移动指针的过程中需要处理这个变动），而整体逻辑是贪心地选最小的若干个数，所以肯定是使用有序结构的。

不妨使用线段树，将每个数赋予一个下标，其中绝对值越大的数对应越大的下标，而线段树维护区间和。这样我们就能快速找到和不超过某个上界的最长前缀。

此时，我们可以用另一个线段树维护区间内当前可选的数的个数（在对应位置标记一个 $1$ ），根据前缀长度加上这个线段树就可以查询得到新取的数的个数。

这里只涉及前缀操作，所以树状数组也行。

时间复杂度为 $\mathcal{O}(n\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    n, k = MII()
    nums = LII()
    
    for i in range(n):
        nums[i] -= k
    
    tags = LII()
    
    st_range = sorted(range(n), key=lambda x: -nums[x])
    pos = [0] * n
    
    for i in range(n):
        pos[st_range[i]] = i
    
    seg = SegTree(add, 0, n)
    seg_cnt = SegTree(add, 0, n)
    
    total = 0
    cnt = 0
    
    for i in range(n):
        if tags[i] == 2:
            if nums[i] >= 0:
                total += nums[i]
                cnt += 1
            else:
                seg.set(pos[i], -nums[i])
                seg_cnt.set(pos[i], 1)
    
    p = seg.max_right(0, lambda x: x <= total)
    ans = cnt + seg_cnt.prod(0, p)
    
    for i in range(n):
        if tags[i] == 1:
            if nums[i] >= 0:
                total += nums[i]
                cnt += 1
            else:
                seg.set(pos[i], -nums[i])
                seg_cnt.set(pos[i], 1)
        else:
            if nums[i] >= 0:
                total -= nums[i]
                cnt -= 1
            else:
                seg.set(pos[i], 0)
                seg_cnt.set(pos[i], 0)
    
        p = seg.max_right(0, lambda x: x <= total)
        ans = fmax(ans, cnt + seg_cnt.prod(0, p))
    
    print(ans)
```

C++ 做法如下——

```cpp []
long long e() {return 0;}
long long op(long long x, long long y) {return x + y;}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, k;
	cin >> n >> k;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x, x -= k;

	vector<int> tags(n);
	for (auto &x: tags) cin >> x;

	vector<int> order(n);
	iota(order.begin(), order.end(), 0);
	sort(order.begin(), order.end(), [&] (int i, int j) {return nums[i] > nums[j];});

	vector<int> pos(n);
	for (int i = 0; i < n; i ++) pos[order[i]] = i;

	atcoder::segtree<long long, op, e> seg(n), seg_cnt(n);

	long long total = 0, cnt = 0;

	for (int i = 0; i < n; i ++) {
		if (tags[i] == 2) {
			if (nums[i] >= 0) {
				total += nums[i];
				cnt ++;
			}
			else {
				seg.set(pos[i], -nums[i]);
				seg_cnt.set(pos[i], 1);
			}
		}
	}

	int p = seg.max_right(0, [&] (long long v) {return v <= total;});
	long long ans = cnt + seg_cnt.prod(0, p);

	for (int i = 0; i < n; i ++) {
		if (tags[i] == 1) {
			if (nums[i] >= 0) {
				total += nums[i];
				cnt ++;
			}
			else {
				seg.set(pos[i], -nums[i]);
				seg_cnt.set(pos[i], 1);
			}
		}
		else {
			if (nums[i] >= 0) {
				total -= nums[i];
				cnt --;
			}
			else {
				seg.set(pos[i], 0);
				seg_cnt.set(pos[i], 0);
			}
		}

		p = seg.max_right(0, [&] (long long v) {return v <= total;});
		ans = max(ans, cnt + seg_cnt.prod(0, p));
	}

	cout << ans;

	return 0;
}
```
