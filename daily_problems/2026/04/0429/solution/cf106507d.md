**提示 1：** 不考虑那些特殊点时，整个函数如何？满足条件吗？

**提示 2：** 如果新增点，可以新增哪些点？

如果 $S$ 是空的，整个函数是一个倒 U 形的满足条件。

而在 $S$ 中加入元素，会使得某些单点位置的数值不变 / 变大，如果变大，则这个点 $x$ 和 $x-\epsilon,x-2\epsilon$ 之间会形成不满足条件的一组数，所以这个点的数值不变。

因此这个点的位置，各条直线的最大值和最小值应该一样。

如果有两线平行，这件事显然不成立；重合的线也可以当作只有一条；如果两线相交，那么交点唯一确定，也就是我们要找的点的位置。

如果所有直线都重合，显然 $S$ 可以任取，因为相当于输入一条直线。

否则，我们要找到所有直线的公共交点，并判断是否所有直线是否都经过它。我们只需不断枚举其他直线看是否与第一条直线相交，直到找到相交的直线，算出交点。最后计算是否所有直线都过这个交点即可。 $S$ 中只能包含等于这个交点的元素。

时间复杂度为 $\mathcal{O}(n+m)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        
        ks = []
        bs = []
        
        for _ in range(n):
            k, b = MII()
            ks.append(k)
            bs.append(b)
        
        flg = True
        single = False
        x, y = 0, 0
    
        for i in range(1, n):
            if ks[i] == ks[0]:
                if bs[i] != bs[0]: flg = False
            else:
                k1, k2 = ks[0], ks[i]
                b1, b2 = bs[0], bs[i]
                single = True
                
                x = -(b2 - b1) // (k2 - k1)
                y = k1 * x + b1
        
        if single:
            for i in range(n):
                k = ks[i]
                b = bs[i]
                if y != k * x + b:
                    flg = False
        
        if m:
            nums = LII()
            
            for x0 in nums:
                if single and x0 != x:
                    flg = False
        else:
            flg = True
        
        outs.append('YES' if flg else 'NO')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, m;
		cin >> n >> m;

		vector<pair<int, int>> lines(n);
		for (auto &[k, b]: lines) cin >> k >> b;

		bool flg = true, single = false;
		int x;
		long long y;

		for (int i = 1; i < n; i ++) {
			auto [k1, b1] = lines[0];
			auto [k2, b2] = lines[i];
			if (k1 == k2) {
				if (b1 != b2) flg = false;
			}
			else {
				single = true;

				x = -(b2 - b1) / (k2 - k1);
				y = 1ll * k1 * x + b1;
			}
		}

		if (single) {
			for (int i = 0; i < n; i ++) {
				auto [k, b] = lines[i];
				if (y != 1ll * k * x + b) {
					flg = false;
				}
			}
		}

		if (m) {
			while (m --) {
				int x0;
				cin >> x0;
				if (single && x0 != x) {
					flg = false;
				}
			}
		}
		else flg = true;

		cout << (flg ? "YES" : "NO") << '\n';
	}

	return 0;
}
```
