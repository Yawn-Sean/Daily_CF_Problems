**提示 2：** 如何进行最优化？

如果 $e=x$ ，则 $w=e-p$ ，而 $n-s=r, n+s=y-e-w=y+p-2x$ ，所以 $n=(y+p+r)/2-x, s=(y+p-r)/2-x$ 。

所以整体就是一系列线性绝对值函数的最大值，画图可以发现是一个 $V$ 字形的函数。

当然可以分类讨论，但是这种简单的函数形式，显然写三分找最小值点会很容易。

时间复杂度为 $\mathcal{O}(\log\epsilon^{-1})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        p, r, y = LFI()
        left, right = -2, 2
        
        def f(x):
            return max(abs(x), abs(x - r), abs((p + y + r) / 2 - x), abs((y + r - p) / 2 - x))
        
        for _ in range(100):
            mid1 = (left * 2 + right) / 3
            mid2 = (left + right * 2) / 3
            
            if f(mid1) < f(mid2): right = mid2
            else: left = mid1
    
        x = (left + right) / 2
        x = round(x, 10)
        outs.append(f'{x} {(p + y + r) / 2 - x} {x - r} {(y + r - p) / 2 - x}')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	cout << fixed << setprecision(10);

	while (t --) {
		double p, r, y;
		cin >> p >> r >> y;
		double left = -2, right = 2;

		auto f = [&] (double x) -> double {
			return max({abs(x), abs(x - r), abs((p + y + r) / 2 - x), abs((y + r - p) / 2 - x)});
		};

		for (int i = 0; i < 100; i ++) {
			double mid1 = (left * 2 + right) / 3;
			double mid2 = (left + right * 2) / 3;
			if (f(mid1) < f(mid2)) right = mid2;
			else left = mid1;
		}

		double x = (left + right) / 2;

		cout << x << ' ' << (p + y + r) / 2 - x << ' ' << x - r << ' ' << (y + r - p) / 2 - x << '\n';
	}

	return 0;
}
```
