**提示 1：** 考虑每个元素的权重，我们应该怎么分配元素？

**提示 2：** 如何确定最小交换次数？

先特判 $n=1$ 的情况。接下来考虑 $n\gt 1$ 。

先看提示 1 。

中间的每个元素都分别做一次十位和个位，因此总权重是 $11$ ，而第一个位置权重是 $10$ ，最后一个位置权重是 $1$ 。

因此，要把最大的数分配给最小的权值 $1$ ，第二大的数分配给第二小的权值 $10$ ，剩余元素任意分配，根据这个求出 $F(s)$ 的最小值。

交换几次呢？很明显，最多交换两次，分别挪到对应位置就行。

$0$ 次操作下，只需判开头是否满足条件即可。

$1$ 次操作有两种情况：一种情况是已经有一个元素到位了，另一种情况是刚好是首尾交换，判断一下就行。

剩余情况需要操作 $2$ 次。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        s = [int(c) for c in I()]
        
        if n == 1: outs.append('0 0')
        else:
            ans = sum(s) * 11
            x, y = nlargest(2, s)
            ans -= x * 10
            ans -= y
            
            cnt = 0
            if s[0] != y: cnt += 1
            if s[-1] != x: cnt += 1
            
            if s[0] == x and s[-1] == y:
                cnt = fmin(cnt, 1)
            
            outs.append(f'{cnt} {ans}')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		string s;
		cin >> n >> s;

		if (n == 1) {
			cout << "0 0\n";
			continue;
		}

		int ans = 0;
		char ma1 = '0', ma2 = '0';
		for (auto &c: s) {
			ans += c - '0';
			if (c > ma1) swap(ma1, ma2), ma1 = c;
			else ma2 = max(c, ma2);
		}

		int cnt = (s[0] != ma2) + (s.back() != ma1);
		if (s[0] == ma1 && s.back() == ma2) cnt = min(cnt, 1);

		cout << cnt << ' ' << 11 * ans - 10 * (ma1 - '0') - (ma2 - '0') << '\n';
	}

	return 0;
}
```
