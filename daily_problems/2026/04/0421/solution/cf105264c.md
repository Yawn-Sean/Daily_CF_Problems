**提示 1：** 答案很小。

**提示 2：** DP 状态设计。

答案很小。所以考虑最后的数字有 $k$ 种时的最小操作次数。

假设已经选定了 $k$ 个目标数值，则每个数显然挑选离它最近的一个数变。

因此发现，是排序后连续的一段变成一个数。

因此开头，我们进行排序，考虑长度为 $i$ 的前缀分为 $j$ 段，每段数值相同的最小成本。状态转移只需枚举下一段的长度，而这一段变成相同数值的最小成本就是都变成中位数的最小成本，可以用前缀和快速计算。

也可以利用目标数值在 $n$ 个数值中取，直接对每个区间 $\mathcal{O}(n)$ 预处理计算得到。

我们只需找到最小的能使得 $dp[n][j]\leq k$ 的 $j$ 。关于第二个维度可以滚动处理，因此空间可以压缩到 $\mathcal{O}(n)$ 。

时间复杂度为 $\mathcal{O}(n^3)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    inf = 10 ** 13
    
    for _ in range(t):
        n, k = MII()
        nums = LII()
        nums.sort()
        
        acc = [0] * (n + 1)
        for i in range(n):
            acc[i + 1] = acc[i] + nums[i]
    
        dp = [inf] * (n + 1)
        dp[0] = 0
        
        for i in range(1, n + 1):
            ndp = [inf] * (n + 1)
            
            for x in range(n):
                for y in range(x, n):
                    mid = (x + y) // 2
                    cost = ((acc[y + 1] - acc[mid]) - (y + 1 - mid) * nums[mid]) + ((mid - x) * nums[mid] - (acc[mid] - acc[x]))
                    ndp[y + 1] = fmin(ndp[y + 1], dp[x] + cost)
            
            if ndp[-1] <= k:
                outs.append(i)
                break
            
            dp = ndp
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	long long inf = 1e13;

	while (t --) {
		int n;
		long long k;
		cin >> n >> k;

		vector<int> nums(n);
		for (auto &x: nums) cin >> x;
		sort(nums.begin(), nums.end());

		vector<long long> acc(n + 1, 0);
		for (int i = 0; i < n; i ++) acc[i + 1] = acc[i] + nums[i];

		vector<long long> dp(n + 1, inf);
		dp[0] = 0;

		for (int i = 1; i <= n; i ++) {
			vector<long long> ndp(n + 1, inf);
			
			for (int x = 0; x < n; x ++) {
				for (int y = x; y < n; y ++) {
					int mid = (x + y) / 2;
					long long cost = (acc[y + 1] - acc[mid] - 1ll * (y + 1 - mid) * nums[mid]) + (1ll * (mid - x) * nums[mid] - (acc[mid] - acc[x]));
					ndp[y + 1] = min(ndp[y + 1], dp[x] + cost);
				}
			}
			
			if (ndp[n] <= k) {cout << i << '\n'; break;}

			dp.swap(ndp);
		}
	}

	return 0;
}
```
