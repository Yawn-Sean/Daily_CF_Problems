**提示 1：** 列方程。解。看看特殊情况。

我们把题目中的条件列为方程。设有 $x_0$ 个参赛者， $x_1$ 个获奖者， $x_2$ 个最终胜者，则：

$x_1+x_2\leq x_0$ 且有 $x_0+a(x_1+2x_2)=c, x_0+b(x_1+2x_2)=d$ 。

两个方程相减可得： $(b-a)(x_1+2x_2)=d-c$ 。

在 $a\neq b$ 时，可以得到 $x_1+2x_2$ 的唯一可能取值，不妨设为 $V$ ，也就可以得到 $x_0$ 的数值。我们需要去掉那些不合法的情况。可能是——数值非整数，两个数值变成了负数，或者 $x_1+x_2\gt x_0$ 。最后一件事的判断需要求一下 $x_1+x_2$ 的最小值，其实就是 $\lceil V/2\rceil$ 。

在 $a=b$ 时，其实我们只需求解 $x_0+a(x_1+2x_2)=c$ 中 $x_0$ 的最小值。

也就是我们要最大化 $x_1+2x_2$ ，设为 $X$ ，则 $x_0\geq x_1+x_2\geq X/2$ 。

所以 $c=x_0+a(x_1+2x_2)\geq X/2+aX$ ，所以 $X\leq \frac{2c}{2a+1}$ 。

而 $\left\lfloor\frac{2c}{2a+1}\right\rfloor$ 是可以取到的，因为 $c\geq X/2+aX$ 满足时，  $c\geq\lceil X/2\rceil +aX$ 也一定成立，而 $x_0$ 是可以取到 $\lceil X/2\rceil$ 的，所以一定能有对应方案。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    a = II()
    b = II()
    c = II()
    d = II()
    
    if a == b:
        if c != d: print(0)
        else:
            v = 2 * c // (2 * a + 1)
            print(c - a * v)
    else:
        if (d - c) % (b - a): print(0)
        else:
            v = (d - c) // (b - a)
            if v < 0: print(0)
            elif c - a * v < (v + 1) // 2: print(0)
            else: print(c - a * v)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	long long a, b, c, d;
	cin >> a >> b >> c >> d;

	if (a == b) {
		if (c != d) cout << 0;
		else {
			long long v = 2 * c / (2 * a + 1);
			cout << c - a * v;
		}
	}
	else {
		if ((d - c) % (b - a)) cout << 0;
		else {
			long long v = (d - c) / (b - a);
			if (v < 0) cout << 0;
			else if (c - a * v < (v + 1) / 2) cout << 0;
			else cout << c - a * v;
		}
	}

	return 0;
}
```
