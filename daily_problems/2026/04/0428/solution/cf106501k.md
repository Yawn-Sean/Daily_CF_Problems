**提示 1：** 首先，开始的查询不能太长，只能 “试探”。

**提示 2：** 最后得到一个目标区间后，如何查询呢？

首先，前几次查询不能太长——不然后面的查询也只能是这么长的了。

所以直观上来看，查询区间要变长只能慢慢变长，因为如果过长了，这个区间没办法在 $[1,n]$ 上自由移动，同时也可能无论如何移动都包含至少一个 $1$ ，这样我们查询就没用了。所以对区间扩长要谨慎。

而最终锁定答案所在的区间后，可以用二分找其中 $1$ 的位置。为了二分能找到一个确切位置的 $1$ ，我们需要先得到一个相对干净的区间（其实就是全是 $0$ 的区间），这样从这个区间出发进行二分才能找到 $1$ 。（用二分做的事——从这个区间出发最少移动多少格才能找到 $1$ ）。

考虑前面的区间变长的过程。第一次查询查 $[1,1]$ ，接下来第 $i$ 次查询查 $[2^{i-2},2^{i-1}]$ ，直到找到一个包含 $1$ 的区间。

这样，找到一个长度为 $l$ 的区间 $X$ 时，在这个区间前面已经查询了总长为 $l$ 的区间，也就留足了查询长度为 $l$ 的区间的空间了。

接下来直接在区间 $X=[vl,vr]$ 内进行二分即可。每次查询 $[vl,mid]$ 中是否有 $1$ ，可以直接用 $[mid-l+1,mid]$ 的查询来进行替代，这也利用了前面 $[1,l]$ 都是空，方便区间的移动。

查询次数为 $\mathcal{O}(\log n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    def query(l, r):
        print('?', l + 1, r + 1, flush=True)
        return II()
    
    def answer(p):
        print('!', p + 1)
    
    n = II()
    
    if query(0, 0):
        answer(0)
    else:
        block = 1
        l_bound = 1
        r_bound = 1
        for i in range(1, 30):
            r = 1 << i
            l = r // 2
            
            if r >= n:
                r_bound = n - 1
                break
            
            block = r - l
            if query(l, r - 1):
                r_bound = r - 1
                break
            
            l_bound = r
        
        l = l_bound
        r = r_bound
        
        while l <= r:
            mid = (l + r) // 2
            if query(mid - block + 1, mid): r = mid - 1
            else: l = mid + 1
        
        answer(l)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n;
	cin >> n;

	auto query = [&] (int l, int r) -> int {
		cout << "? " << l + 1 << ' ' << r + 1 << endl;
		int x; cin >> x;
		return x;
	};

	auto answer = [&] (int x) -> void {
		cout << "! " << x + 1;
	};

	if (query(0, 0)) answer(0);
	else {
		int block = 1, l_bound = 1, r_bound = 1;

		for (int i = 1; i < 30; i ++) {
			int r = 1 << i;
			int l = r / 2;

			if (r >= n) {
				r_bound = n - 1;
				break;
			}

			block = r - l;
			if (query(l, r - 1)) {
				r_bound = r - 1;
				break;
			}

			l_bound = r;
		}

		int l = l_bound, r = r_bound;

		while (l <= r) {
			int mid = (l + r) / 2;
			if (query(mid - block + 1, mid)) r = mid - 1;
			else l = mid + 1;
		}

		answer(l);
	}

	return 0;
}
```
