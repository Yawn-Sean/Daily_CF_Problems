**提示 1：** 如果你想知道某一列的高度，怎么办？

**提示 2：** 如何减少查询列的数量呢？

首先，每一列如果你要确定高度，可以直接二分。

但不能每一列都二分怎么办呢？

一旦我们可以确定某一列的最高位置不会比之前的最高点更高了，我们就直接不考虑这一列。

怎么利用这件事呢？我们直接打乱各个列，这样前缀最大值的个数是期望 $\mathcal{O}(\log n)$ 的，也就只需要进行这么多次二分，查询次数就对了。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    def query(x, y):
        print('?', x, y, flush=True)
        return I()[0] == 'b'
    
    w, h = MII()
    
    to_check = list(range(1, w + 1))
    random.shuffle(to_check)
    
    idx = 1
    cur_h = 0
    
    for i in to_check:
        if cur_h < h and query(i, cur_h + 1):
            l, r = cur_h + 1, h
            
            while l <= r:
                mid = (l + r) // 2
                if query(i, mid): l = mid + 1
                else: r = mid - 1
            
            idx = i
            cur_h = r
    
    print('!', idx, cur_h)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	auto query = [&] (int x, long long y) -> bool {
		cout << "? " << x << ' ' << y << endl;
		string s;
		cin >> s;
		return s[0] == 'b';
	};

	int w;
	long long h;
	cin >> w >> h;

	vector<int> to_check(w);
	iota(to_check.begin(), to_check.end(), 1);
	shuffle(to_check.begin(), to_check.end(), random_device());

	int idx = 1;
	long long cur_h = 0;

	for (auto &i: to_check) {
		if (cur_h < h && query(i, cur_h + 1)) {
			long long l = cur_h + 1, r = h;

			while (l <= r) {
				long long mid = (l + r) / 2;
				if (query(i, mid)) l = mid + 1;
				else r = mid - 1;
			}

			idx = i, cur_h = r;
		}
	}

	cout << "! " << idx << ' ' << cur_h;

	return 0;
}
```
