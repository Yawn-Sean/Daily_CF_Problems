**提示 2：** 最小的角度如何找到？

首先判断能否相交只需看 $(0,0)$ 到对应线段的距离是否小于等于 $r$ 。只需用面积法 $\frac{xy}{\sqrt{x^2+y^2}}$ 和 $r$ 比较就行。通分去括号后可以无误差比较。

接下来求角度。如果 $r\geq x$ ，显然答案是 $0$ 。

否则我们就要找最小角度，容易直接作圆找到。此时角度怎么算呢？

我们发现底下的三角形， $r$ 对应的角度正弦值很好找。而底边 $x$ 很容易确定，所以我们用正弦定理可以确定上面的角的正弦值。而用反正弦函数就可以确定上面的对应的外角，减去右侧的角就是答案了。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    y, x, r = MII()
    x = abs(x)
    
    if x * x * y * y > r * r * (x * x + y * y):
        print(-1)
    elif r >= x:
        print(0)
    else:
        print((math.asin(y / math.hypot(x, y) / r * x) - math.atan2(y, x)) / math.pi * 180)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int y, x, r;
	cin >> y >> x >> r;

	x = abs(x);

	if (1ll * x * x * y * y > 1ll * r * r * (x * x + y * y)) cout << -1;
	else if (r >= x) cout << 0;
	else {
		cout << fixed << setprecision(20);
		long double ans = 0;
		ans += asinl((long double)y / hypotl(x, y) / r * x);
		ans -= atan2l(y, x);
		ans /= acos(-1);
		ans *= 180;
		cout << ans;
	}

	return 0;
}
```
