**提示 1：** 需要满足什么条件？

考虑最后的数组最小值和最大值分别是 $m,M$ ，则大于 $m$ 且小于 $M$ 的元素永远不能被操作到，不然的话会变成最小值 / 最大值。同时，最后变成 $m$ 的数开始一定比 $m$ 小，最后变成 $M$ 的数开始一定比 $M$ 大。

所以分别检查每个元素，判断其与最小值和最大值的关系就行。

还有特殊情况，就是每个元素都相等，此时初始情况完全不重要，直接输出 $10^n$ 就行。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        nums = LII()
        
        mi = min(nums)
        ma = max(nums)
        
        if mi == ma: outs.append(10 ** n)
        else:
            ans = 1
            for x in nums:
                if x == mi:
                    ans *= x
                elif x == ma:
                    ans *= 11 - x
            outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		vector<int> nums(n);
		for (auto &v: nums) cin >> v;

		int ma = *max_element(nums.begin(), nums.end());
		int mi = *min_element(nums.begin(), nums.end());

		int ans = 1;
		if (ma == mi)
			for (int i = 0; i < n; i ++) ans *= 10;
		else {
			for (auto &x: nums) {
				if (x == mi) ans *= x;
				else if (x == ma) ans *= 11 - x;
			}
		}

		cout << ans << '\n';
	}

	return 0;
}
```
