**提示 1：** 考虑将 LCM 问题转化为 GCD 问题。

**提示 2：** 枚举 GCD 。

考虑 GCD 的枚举。

首先 GCD 只可能是 $n$ 的因子，不妨设为 $g$ 。设 $p=n+v$

则 $\mathrm{gcd}(n,n+k)=n\times \frac{n+v}{g}$ 。

而 $v$ 一定也是 $g$ 的倍数，而 $v$ 越小越好，所以 $v=g$ 。

因此答案是 $n\times \frac{n+g}{g}$ 。 $g$ 显然越大越好，所以我们只需令 $g$ 是 $n$ 最大的不超过 $k$ 的因子就行。

时间复杂度为 $\mathcal{O}(\sqrt{M})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n = II()
    k = II()
    
    ans = n + 1
    
    for i in range(1, 10 ** 6 + 1):
        if n % i == 0:
            if i <= k: ans = fmax(ans, n + i)
            if n // i <= k: ans = fmax(ans, n + n // i)
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	long long n, k;
	cin >> n >> k;

	long long ans = n + 1;

	for (int i = 1; i <= 1'000'000; i ++) {
		if (n % i == 0) {
			if (i <= k) ans = max(ans, n + i);
			if (n / i <= k) ans = max(ans, n + n / i);
		}
	}

	cout << ans;

	return 0;
}
```
