**提示 1：** 找下规律。根据规律证明一下结论。

首先，这题怎么暴力得到结果呢？我们固定一个 $k$ ，考虑 $i$ 个 chip 有无必胜策略。则如果 $i-1$ 或者 $i-k$ 没有必胜策略，则 $i$ 有必胜策略；否则， $i$ 没有必胜策略。

于是我们可以基于此对不同的 $k$ 进行打表。此时你发现是 $k+1$ 个一循环的，且胜负规律比较明显。根据规律进行输出即可。

证明可以用数学归纳法，也是很容易的。时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    fin = open('dotak.in', 'r')

    t = II()
    outs = []

```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	ifstream fin("dotak.in");

	int t;
	fin >> t;

	while (t --) {
		int n, k;
		fin >> n >> k;

		int v = 1ll * n * k % (k + 1);
		cout << ((v & 1) || (v == k) ? "Omda" : "Teemo") << '\n';
	}

	return 0;
}
```
