**提示 2：** 把奇数列和偶数列的分别统计。

观察图，发现从左到右每一列的个数是 $2$ 个一循环的。

分别考虑奇数列和偶数列上有多少染色方块即可。而奇数列和偶数列的个数也可以直接计算得到。相乘后再相加即可得到答案。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    x1, y1, x2, y2 = MII()
    x = x2 - x1 + 1
    y = y2 - y1 + 1

    print(((x + 1) // 2) * ((y + 1) // 2) + ((x - 1) // 2) * ((y - 1) // 2))
```

C++ 做法如下——

```cpp []
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);

    int x1, y1, x2, y2;
    cin >> x1 >> y1 >> x2 >> y2;

    int x = x2 - x1 + 1, y = y2 - y1 + 1;
    cout << 1ll * ((x + 1) / 2) * ((y + 1) / 2) + 1ll * ((x - 1) / 2) * ((y - 1) / 2);

    return 0;
}
```
