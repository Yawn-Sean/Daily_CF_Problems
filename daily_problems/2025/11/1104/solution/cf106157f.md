**提示 1：** 枚举最小权。

**提示 2：** 其实只需考虑连通性。

我们考虑枚举最小权，接下来只需要最小化最大权。

我们选取所有经过的点都不小于最小权的边，接下来认为每条边的边权等于较高的位置的高度，看为了使得 $1,2$ 连通，使用的最大边权最小是多少。

这件事很像最小生成树，因此只需按照排序后的边不断使用并查集，看何时 $1,2$ 连通即可。

由于枚举不同最小权时，边的排序结果是保持不变的（除了某些边我们可能不要了），因此直接预先排序，之后每次枚举最小权时使用即可。

时间复杂度为 $\mathcal{O}(m\log m+nm)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, m = MII()
    nums = LII()
    
    us = []
    vs = []
    
    for _ in range(m):
        u, v = GMI()
        if nums[u] > nums[v]:
            u, v = v, u
        us.append(u)
        vs.append(v)
    
    st_range = sorted(range(m), key=lambda x: nums[vs[x]])
    
    uf = UnionFind(n)
    
    ans = 10 ** 8
    
    for i in range(n):
        uf.init()
        
        for j in st_range:
            if nums[us[j]] >= nums[i]:
                uf.merge(us[j], vs[j])
                if uf.find(0) == uf.find(1):
                    ans = fmin(ans, nums[vs[j]] - nums[i])
                    break
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m;
	cin >> n >> m;

	vector<int> nums(n);
	for (auto &v: nums) cin >> v;

	vector<pair<int, int>> edges(m);
	for (auto &[u, v]: edges) {
		cin >> u >> v;
		u --, v --;
		if (nums[u] > nums[v]) swap(u, v);
	}

	sort(edges.begin(), edges.end(), [&] (pair<int, int> x, pair<int, int> y) {
		return nums[x.second] < nums[y.second];
	});


	atcoder::dsu uf(n);
	int ans = 1e8;

	for (int i = 0; i < n; i ++) {
		uf.init();
		
		for (auto &[u, v]: edges) {
			if (nums[u] >= nums[i]) {
				uf.merge(u, v);
				if (uf.leader(0) == uf.leader(1)) {
					ans = min(ans, nums[v] - nums[i]);
					break;
				}
			}
		}
	}

	cout << ans;

	return 0;
}
```
