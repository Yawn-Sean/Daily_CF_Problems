**提示 1：** 先找到第 $m$ 大的差。

**提示 2：** 我们相当于要先找到更大的差的和，怎么计算？

首先，我们先找到第 $m$ 大的差值。

我们可以使用二分，因为我们可以统计插值大于等于某个数 $x$ 的对数（这件事可以对排序后的数组使用双指针得到）。

找到了这个数 $val$ 之后，比这个数大的差值自然都是要选的，我们要对这些数进行求和。

我们仍然对排序后的数组使用双指针，对于某个左端点 $l$ ，右端点的范围是 $[r,n-1]$ ，于是我们要求这些 $nums[i]-nums[l]$ 的和，也就是 $nums[i]$ 的区间和减去 $(n-r)nums[l]$ ，因此多维护一个前缀和用来求区间和即可。

我们把大于 $val$ 的统计了，等于的呢？我们只需要确认需要算几个等于 $val$ 的位置即可，为此我们只需再统计一下大于 $val$ 的差值对有多少个，这件事在前面双指针的过程中可以计算。

时间复杂度为 $\mathcal{O}(n(\log n+\log M))$ 。 

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        nums = LII()
        nums.sort()
        
        l, r = 1, 10 ** 7
        
        while l <= r:
            mid = (l + r) // 2
            
            pt = 0
            cnt = 0
            
            for i in range(n):
                while pt < n and nums[pt] - nums[i] < mid:
                    pt += 1
                cnt += (n - pt) * 2
            
            if cnt <= m: r = mid - 1
            else: l = mid + 1
        
        acc = list(accumulate(nums, initial=0))
        
        bound = l
        pt = 0
        cnt = 0
        ans = 0
        
        for i in range(n):
            while pt < n and nums[pt] - nums[i] < bound:
                pt += 1
            cnt += (n - pt) * 2
            ans += (acc[n] - acc[pt] - (n - pt) * nums[i]) * 2
        
        ans += (m - cnt) * (bound - 1)
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		long long m;
		cin >> n >> m;

		vector<int> nums(n);
		for (auto &v: nums) cin >> v;
		sort(nums.begin(), nums.end());

		int l = 1, r = 1e7;

		while (l <= r) {
			int mid = (l + r) / 2;
			int pt = 0;
			long long cnt = 0;

			for (int i = 0; i < n; i ++) {
				while (pt < n && nums[pt] - nums[i] < mid) pt ++;
				cnt += (n - pt) * 2;
			}

			if (cnt <= m) r = mid - 1;
			else l = mid + 1;
		}

		vector<long long> acc(n + 1, 0);
		for (int i = 0; i < n; i ++)
			acc[i + 1] = acc[i] + nums[i];
		
		{
			int bound = l, pt = 0;
			long long cnt = 0, ans = 0;

			for (int i = 0; i < n; i ++) {
				while (pt < n && nums[pt] - nums[i] < bound) pt ++;
				cnt += (n - pt) * 2;
				ans += (acc[n] - acc[pt] - (n - pt) * nums[i]) * 2;
			}

			ans += (m - cnt) * (bound - 1);

			cout << ans << '\n';
		}
	}

	return 0;
}
```
