**提示 1：** 有些位置一定得分割。

**提示 2：** 这样，我们选择的 $k$ 需要满足什么条件？

考虑数组被分割的位置。

那些相邻的减少的位置一定得分割的。而我们分割的位置只能是 $k$ 的倍数，所以假设我们必须分割的位置是 $v_1,v_2,\dots$ ，则因为这些数都是 $k$ 的倍数，因此 $k$ 只能是这些数最大公约数的因子。

于是我们可以预处理每个数的因子个数。而每次修改一个位置相当于可能改变前后两个位置的增减情况，因此相当于修改了必须分割的 $\mathcal{O}(1)$ 个位置。

而怎么在增减数字的过程中求最大公约数呢？我们用一个长度为 $n$ 的数组，第 $i$ 位在必须分割的情况下取 $i$ ，否则取 $0$ ，则这个数组的最大公约数就是答案，而新增分割位置和删去分割位置在这个数组中就相当于修改元素，修改元素求最大公约数显然可以通过线段树解决。

最后，如果没有必须分割的数字，直接输出 $n$ 即可。

时间复杂度为 $\mathcal{O}(n+q\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    M = 2 * 10 ** 5
    f = [0] * (M + 1)
    for i in range(1, M + 1):
        for j in range(i, M + 1, i):
            f[j] += 1
    
    for _ in range(t):
        n, q = MII()
        nums = LII()
        
        f[0] = n
        
        vals = [0] * n
        for i in range(1, n):
            if nums[i] < nums[i - 1]:
                vals[i] = i
        
        seg = SegTree(math.gcd, 0, vals)
        
        outs.append(f[seg.all_prod()])
        
        for _ in range(q):
            idx, val = MII()
            idx -= 1
            
            nums[idx] = val
            
            if idx:
                vals[idx] = idx if nums[idx] < nums[idx - 1] else 0
                seg.set(idx, vals[idx])
            if idx + 1 < n:
                vals[idx + 1] = idx + 1 if nums[idx + 1] < nums[idx] else 0
                seg.set(idx + 1, vals[idx + 1])
            
            outs.append(f[seg.all_prod()])
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int op(int x, int y) {return gcd(x, y);}

int e() {return 0;}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t, M = 2e5;
	cin >> t;

	vector<int> f(M + 1, 0);
	for (int i = 1; i <= M; i ++) {
		for (int j = i; j <= M; j += i) {
			f[j] ++;
		}
	}

	while (t --) {
		int n, q;
		cin >> n >> q;

		f[0] = n;

		vector<int> nums(n), vals(n, 0);

		for (auto &x: nums) cin >> x;
		for (int i = 1; i < n; i ++) {
			if (nums[i] < nums[i - 1]) {
				vals[i] = i;
			}
		}

		atcoder::segtree<int, op, e> seg(vals);

		cout << f[seg.all_prod()] << '\n';

		while (q --) {
			int idx, val;
			cin >> idx >> val;
			idx --;

			nums[idx] = val;

			if (idx) {
				vals[idx] = (nums[idx] < nums[idx - 1] ? idx : 0);
				seg.set(idx, vals[idx]);
			}
			if (idx + 1 < n) {
				vals[idx + 1] = (nums[idx + 1] < nums[idx] ? idx + 1 : 0);
				seg.set(idx + 1, vals[idx + 1]);
			}
			cout << f[seg.all_prod()] << '\n';
		}
	}

	return 0;
}
```
