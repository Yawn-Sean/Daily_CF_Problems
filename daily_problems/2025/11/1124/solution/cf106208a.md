**提示 1：** 每一次取数的期望如何？

**提示 2：** 其实操作总次数的期望只和区间长度有关。

首先，对于每一个操作序列，我们可以将选取的数和选取的区间都关于 $(l+r)/2$ 这个点对称，两件事发生的概率一定是相等的。所以计算期望的时候相当于我们找到了等概率的配对关系，所以每一轮的期望取的数的值的期望总是 $(l+r)/2$ 。

因此我们只需要计算期望操作轮数就好。

事实上，轮次数量只和区间长度有关，因为区间平移不影响结果。

所以考虑区间长度是 $len$ 的期望轮次是 $dp[len]$ ，于是：

$dp[len]=\frac{2dp[len-1]+3dp[len-2]+\dots+len\times dp[1]}{len(len+1)/2-1}+1$

这个求和式可以理解为：

$2dp[len-1]+3dp[len-2]+\dots+len\times dp[1]=(len+1-(len-1))dp[len-1]+(len+1-(len-2))dp[len-2]+\dots+(len+1-1)dp[1]$ 

也就是将前面的系数拆成里面下标的线性组合，这样进一步的就可以拆分为两个序列的前缀和，即：

$(len+1)(dp[len-1]+dp[len-2]+\dots+dp[1])-((len-1)dp[len-1]+(len-2)dp[len-2]+\dots+dp[1])$

这两个前缀和就好维护很多了。

当然，这里还涉及到逆元，如果用 Python 的话直接分开求大数逆元很可能被卡，但这里有个小优化。

因为 $len(len+1)/2-1=(len^2+len-2)/2=(len+2)(len-1)/2$ ，所以实际上你只需要求的是 $len-1,len+2$ 的逆元，这个东西是可以 $\mathcal{O}(M)$ 预处理得到的。所以维护前缀和 + 逆元的转移复杂度还是线性的。

查询时，用期望轮次乘以每一轮的期望数值即可。

时间复杂度是 $\mathcal{O}(M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    M = 2 * 10 ** 6
    mod = 998244353
    
    f = Factorial(M + 5, mod)
    
    dp = [0] * M
    dp[0] = 1
    
    v0 = 1
    v1 = 0
    
    for i in range(1, M):
        dp[i] = ((i + 1) * v0 - v1) % mod * 2 % mod * f.inv(i) % mod * f.inv(i + 3) % mod
        dp[i] = (dp[i] + 1) % mod
        v0 = (v0 + dp[i]) % mod
        v1 = (v1 + dp[i] * i) % mod
    
    t = II()
    rev2 = (mod + 1) // 2
    
    outs = []
    
    for _ in range(t):
        l, r = MII()
        outs.append((l + r) * rev2 % mod * dp[r - l] % mod)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
long long quickPow(long long base, long long power, long long mod) {
    if (power == 0) return 1 % mod;
    long long cur = quickPow(base, power / 2, mod);
    return power & 1 ? base * cur % mod * cur % mod : cur * cur % mod; 
}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int M = 2e6, mod = 998244353;

	vector<int> f(M + 5, 1), g(M + 5);

	for (int i = 1; i < M + 5; i ++)
		f[i] = 1ll * i * f[i - 1] % mod;

	g[M + 4] = quickPow(f[M + 4], mod - 2, mod);

	for (int i = M + 4; i > 0; i --)
		g[i - 1] = 1ll * i * g[i] % mod;

	auto inv = [&] (int x) -> int {
		return 1ll * g[x] * f[x - 1] % mod;
	};

	auto add = [&] (int &x, int y) -> void {
		x += y;
		if (x >= mod) x -= mod;
	};

	vector<int> dp(M, 0);
	dp[0] = 1;

	int v0 = 1, v1 = 0;

	for (int i = 1; i < M; i ++) {
		dp[i] = (1ll * (i + 1) * v0 % mod + mod - v1) % mod * 2 % mod * inv(i) % mod * inv(i + 3) % mod;
		add(dp[i], 1);
		add(v0, dp[i]);
		add(v1, 1ll * dp[i] * i % mod);
	}

	int t, rev2 = (mod + 1) / 2;
	cin >> t;

	while (t --) {
		int l, r;
		cin >> l >> r;
		cout << 1ll * (l + r) * rev2 % mod * dp[r - l] % mod << '\n';
	}

	return 0;
}
```
