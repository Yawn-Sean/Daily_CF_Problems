**提示 1：** 其实只需要考虑在边为无向边的情况下度数为 $1$ 的点。

**提示 2：** 如果 $S=T$ ？

如果 $S=T$ 相当于求 $S$ 对应点形成的子图的边数。

所以如果要求某个点的度数，只需查整个图的边数，再查去掉该点形成的点集形成的子集的边数，两者相减就是答案。

但这样我们需要查 $n+1$ 次，一个全部点和 $n$ 次扣掉一个点。但我们发现最后一次查询是不必要的，因为所有点的度数和等于边数的两倍。

最后根据度数数组找到 $1$ 的位置即可。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    def query(S, T):
        print('?', len(S), *(i + 1 for i in S), len(T), *(i + 1 for i in T), flush=True)
        return II()
    
    def answer(x):
        print('!', x, flush=True)
    
    t = II()
    outs = []
    
    for _ in range(t):
        n = II()
        
        total_edge = query(list(range(n)), list(range(n)))
        degs = [0] * n
        
        for i in range(n - 1):
            tmp = [j for j in range(n) if j != i]
            degs[i] = total_edge - query(tmp, tmp)
        
        degs[-1] = total_edge * 2 - sum(degs)
        
        if 1 in degs: answer(degs.index(1) + 1)
        else: answer(-1)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n;
		cin >> n;

		cout << '?';
		for (int i = 0; i < 2; i ++) {
			cout << ' ' << n;
			for (int j = 1; j <= n; j ++) cout << ' ' << j;
		}
		cout << endl;

		int total_edge;
		cin >> total_edge;

		int resid_deg = total_edge * 2;
		vector<int> degs(n);

		for (int i = 0; i < n - 1; i ++) {
			cout << '?';
			for (int t = 0; t < 2; t ++) {
				cout << ' ' << n - 1;
				for (int j = 0; j < n; j ++) {
					if (i != j) cout << ' ' << j + 1;
				}
			}
			cout << endl;

			int x;
			cin >> x;
			degs[i] = total_edge - x;
			resid_deg -= degs[i];
		}

		degs[n - 1] = resid_deg;

		int ans = -2;
		for (int i = 0; i < n; i ++) {
			if (degs[i] == 1) {
				ans = i;
			}
		}

		cout << "! " << ans + 1 << endl;
	}

	return 0;
}
```
