**提示 1：** 要构成多边形需要满足的条件是什么？

**提示 2：** 最重要的边是啥？枚举它。

要构成多边形，需要每条边长度都小于其他边和。

其中最不容易满足的就是最长边，因此考虑枚举最长边，此时其他的可选边就是长度不超过它的边。

我们要选取更短的 $k-1$ 条边，并统计凑出来的总长度大于当前最长边的方案数量，这件事显然可以背包 DP 解决。

而我们从小到大遍历最长边，显然可以将之前对更短的边做的 DP 结果给后面用。

时间复杂度为 $\mathcal{O}(nkM)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    mod = 10 ** 9 + 7
    
    n, k = MII()
    nums = LII()
    
    nums.sort()
    
    M = 10 ** 4 + 2
    dp = [[0] * M for _ in range(k)]
    dp[0][0] = 1
    
    ans = 0
    
    for i in range(n):
        for j in range(nums[i] + 1, M):
            ans += dp[k - 1][j]
            if ans >= mod:
                ans -= mod
        
        for x in range(k - 2, -1, -1):
            for y in range(M):
                ny = fmin(y + nums[i], M - 1)
                dp[x + 1][ny] += dp[x][y]
                if dp[x + 1][ny] >= mod:
                    dp[x + 1][ny] -= mod
    
    print(ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, k, mod = 1e9 + 7;
	cin >> n >> k;

	vector<int> nums(n);
	for (auto &x: nums) cin >> x;

	sort(nums.begin(), nums.end());
	
	int M = 1e4 + 2;
	vector<vector<int>> dp(k, vector<int>(M, 0));
	dp[0][0] = 1;

	int ans = 0;

	auto add = [&] (int &x, int y) -> void {
		x += y;
		if (x >= mod) x -= mod;
	};

	for (int i = 0; i < n; i ++) {
		for (int j = nums[i] + 1; j < M; j ++) {
			add(ans, dp[k - 1][j]);
		}

		for (int x = k - 2; x >= 0; x --) {
			for (int y = 0; y < M; y ++) {
				add(dp[x + 1][min(y + nums[i], M - 1)], dp[x][y]);
			}
		}
	}

	cout << ans;

	return 0;
}
```
