**提示 1：** 覆盖区域其实是正多边形和圆的闵可夫斯基和。

**提示 2：** 可以使用交换律。

先看提示，又因为闵可夫斯基和可交换，所以可以理解为一个正多边形区域的基础上，有一个圆在动。（也可以用相对运动来理解这件事）

那么面积覆盖情况就很好求了。先算出这个多边形的面积，接下来圆会沿着周长有一段段长方形的覆盖，这部分面积就是周长乘半径。最后，剩余的扇形部分可以拼成一个圆（可以算下角度和，等于外角和），加上它就行。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        r, n, l = MII()
        ans = math.pi * r * r
        ans += l * r * n
        ans += l * l / 2 / math.tan(math.pi / n) * n / 2
        outs.append(ans)
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	cout << fixed << setprecision(10);
	long double pi = acosl(-1);

	while (t --) {
		int r, n, l;
		cin >> r >> n >> l;

		long double ans = 0;
		ans += (long double)l * r * n;
		ans += pi * r * r;
		ans += (long double)l * l / 2 / tanl(pi / n) * n / 2;

		cout << ans << '\n';
	}

	return 0;
}
```
