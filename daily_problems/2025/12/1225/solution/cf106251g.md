**提示 1：** 先看看 $X$ 需要满足啥条件。

**提示 2：** $Y,Z$ 自由度很高，找个好凑的办法。

$Y$ 和 $Z$ 的数码构成完全一致，为此可以推出什么呢？

事实上 $x_iB^v\bmod (B-1)=x_i$ ，所以 $Y,Z$ 关于 $B-1$ 取模的余数一定等于各个数位的和关于 $B-1$ 取模的余数。

而 $X+Y=Z$ ，因此 $X$ 只能是 $B-1$ 的倍数。

此时，令 $Y=X/(B-1), Z=BY$ ，则足够使得两者除了 $0$ 之外各个数码出现的次数完全一致了，接下来怎么调整 $0$ 的问题呢？

我们直接加上一个足够大的 $B^k$ 在前面补齐位数就行了。

注意这里需要模拟一下除法的过程。

时间复杂度为 $\mathcal{O}(n)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        n, b = MII()
        nums = LII()
        
        if sum(nums) % (b - 1): outs.append('NO')
        else:
            outs.append('YES')
            
            res = []
            
            cur = 0
            for x in nums:
                cur = cur * b + x
                res.append(cur // (b - 1))
                cur %= b - 1
            
            outs.append(str(n + 2))
            outs.append('1 0 ' + ' '.join(map(str, res)))
            outs.append('1 ' + ' '.join(map(str, res)) + ' 0')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int n, b;
		cin >> n >> b;

		vector<int> nums(n);
		int tot = 0;

		for (auto &v: nums) 
			cin >> v, tot = (tot + v) % (b - 1);

		if (tot) cout << "NO\n";
		else {
			cout << "YES\n";

			vector<int> res;
			int cur = 0;

			for (auto &x: nums) {
				res.emplace_back((1ll * cur * b + x) / (b - 1));
				cur = (cur + x) % (b - 1);
			}

			cout << n + 2 << '\n';
			cout << "1 0";
			for (auto &v: res) cout << ' ' << v; cout << '\n';
			cout << '1';
			for (auto &v: res) cout << ' ' << v; cout << " 0\n";
		}
	}

	return 0;
}
```
