**提示 1：** 如果没有传送门，显然最短路跑一下就好。

**提示 2：** 为了处理传送门，把传送门加入上面的图中。怎么加？

没有传送门就是无向带权图最短路，直接求即可。

而加了传送门之后，如果能把传送门加入最短路的图中，也就解决问题了。

传送门是进入的时候需要支付成本。所以从 $u$ 到 $v$ 如果用传送门 $x$ ，我们可以理解为 $u$ 先到 $x$ ，并支付 $c$ ，再从 $x$ 到 $v$ ，支付 $0$ 。而这样，传送门上建立的边数就等于有该传送门的点数，因此建立的新图边数仍然在可控范围内。

时间复杂度为 $\mathcal{O}((n+k)+(m+M)\log(n+k))$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    n, m, k = MII()
    path = [[] for _ in range(n + k)]
    
    for _ in range(m):
        u, v, c = MII()
        u -= 1
        v -= 1
        path[u].append(c * (n + k) + v)
        path[v].append(c * (n + k) + u)
    
    for u in range(n):
        t = II()
        for _ in range(t):
            v, c = MII()
            v = v - 1 + n
            path[u].append(c * (n + k) + v)
            path[v].append(u)
    
    inf = 10 ** 16
    dis = [inf] * (n + k)
    dis[0] = 0
    
    pq = [0]
    while pq:
        d, u = divmod(heappop(pq), n + k)
        if u == n - 1:
            print(d)
            break
        
        if dis[u] == d:
            for msk in path[u]:
                nd, v = divmod(msk, n + k)
                if dis[v] > nd + d:
                    dis[v] = nd + d
                    heappush(pq, dis[v] * (n + k) + v)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int n, m, k;
	cin >> n >> m >> k;

	vector<vector<pair<int, int>>> path(n + k);

	while (m --) {
		int u, v, c;
		cin >> u >> v >> c;
		u --, v --;
		path[u].emplace_back(v, c);
		path[v].emplace_back(u, c);
	}

	for (int u = 0; u < n; u ++) {
		int t;
		cin >> t;

		while (t --) {
			int v, c;
			cin >> v >> c;
			v = v - 1 + n;
			path[u].emplace_back(v, c);
			path[v].emplace_back(u, 0);
		}
	}

	long long inf = 1e16;
	vector<long long> dis(n + k, inf);

	priority_queue<pair<long long, int>, vector<pair<long long, int>>, greater<pair<long long, int>>> pq;

	dis[0] = 0;
	pq.push({0, 0});

	while (!pq.empty()) {
		auto [d, u] = pq.top(); pq.pop();
		if (dis[u] == d) {
			for (auto &[v, w]: path[u]) {
				if (d + w < dis[v]) {
					dis[v] = d + w;
					pq.push({dis[v], v});
				}
			}
		}
	}

	cout << dis[n - 1];

	return 0;
}
```
