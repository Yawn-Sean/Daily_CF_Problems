**提示 2：** 勾股定理。

表面上图很复杂，但是你只需要考虑一根弦就够了。

一根弦的长度可以拆解为两部分——垂直于圆的部分，平行于圆的部分，这两部分和弦本身构成直角三角形，所以可以用勾股定理。

平行于圆的部分的长度等于 $2R\mathrm{sin}(\theta/2)$ ，而弦长是 $L$ ，所以就可以直接得到答案了。

### 具体代码如下——

Python 做法如下——

```Python []
def main(): 
    t = II()
    outs = []
    
    for _ in range(t):
        r, l, theta = LII()
        theta = theta / 180 * math.pi
        w = 2 * r * math.sin(theta / 2)
        outs.append(math.sqrt(l * l - w * w))
    
    print('\n'.join(map(str, outs)))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;
	long double pi = acosl(-1);

	cout << fixed << setprecision(15);
	while (t --) {
		long double r, l, theta;
		cin >> r >> l >> theta;
		long double w = 2 * r * sinl(theta / 180 * pi / 2);
		cout << sqrtl(l * l - w * w) << '\n';
	}

	return 0;
}
```
