**提示 2：** 对数运算可以让乘法变成加法。

我们的矩阵快速幂求通项需要是线性组合，即加法，这里乘法咋办呢？

我们两侧取对数，则 $\log F(n)=\log F(n-1)+\log F(n-2)$ 。这里我们要是以 $2$ 为底，则直接变成第一项是 $0$ ，第二项是 $1$ ，求第 $n$ 项了。

这里取模应该关于什么取模呢？因为 $2^{p-1}=2^0\ (\bmod p)$ ，所以应该关于 $p-1$ 取模。得到上侧次数后，再计算 $2^x\bmod p$ 就行。

这里要是前两项给的没这么漂亮怎么办呢？

我们可以拆解为第一部分是 $a^1b^0$ ，第二项是 $a^0b^1$ ，我们递推第 $n$ 项 $a,b$ 分别的次数如何即可。

时间复杂度为 $\mathcal{O}(\log M)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():  
    n = II()
    mod = 998244353
    print(pow(2, matrix_pow([[1, 1], [1, 0]], n, mod - 1)[0][1], 998244353))
```
